제1조 (총칙)

본 방침은 파이스(이하 "회사")가 운영하는 인생포트폴리오 서비스의 기업·단체 계약(이하 "B2B 그룹 계약")과 관련하여 회사가 처리하는 개인정보에 관한 사항을, 일반 개인정보 처리방침의 보충으로 정합니다. 본 방침에서 정하지 않은 사항은 일반 처리방침을 적용합니다.

회사는 고객사로부터 견적·결제·세금계산서 발행에 필요한 담당자 정보 및 사업자 정보를 제공받아 처리합니다. 임직원의 진단 데이터는 임직원 본인이 회사 서비스에 직접 가입하여 입력합니다.

제2조 (처리하는 개인정보 항목)

2.1 고객사 담당자 정보 (견적·계약 단계)

  • 필수: 회사명/단체명, 담당자명, 담당자 이메일
  • 선택: 사업자등록번호, 부서/직책, 연락처(휴대전화)

2.2 이용자(참여자) 정보 (진단 단계)

  • 필수: 이메일, 비밀번호(암호화 저장), 진단 응답 데이터(76문항)
  • 자동수집: 접속 IP, 접속 일시, 브라우저 종류, 디바이스 정보
  • 생성: 진단 결과 리포트, 사용 Access Code 식별값

2.3 결제 정보

  • 입금자명, 입금 일시, 주문번호 매칭 정보
  • (세금계산서 발행 시) 사업자등록번호, 대표자명, 사업장 주소, 업태·종목, 세금계산서용 이메일

제3조 (처리 목적 및 법적 근거)

항목목적법적 근거
고객사 담당자 정보 견적·계약 체결 및 이행, 코드 발급 안내, 세금계산서 발행, 고객 응대 계약 이행 (개인정보보호법 제15조 제1항 제4호)
이용자 진단 데이터 76문항 진단 처리, 개인 리포트 생성, 이용자 본인 열람 제공 정보주체 동의 (이용자 가입 시 직접 동의)
결제·세금계산서 정보 대금 청구·정산, 국세청 전자세금계산서 발행 의무 이행 법령 의무 (부가가치세법, 국세기본법)

제4조 (보유 및 이용기간)

항목보유 기간근거
계약·결제·세금계산서 관련 기록5년전자상거래법 · 부가가치세법
이용자 진단 응답 및 리포트계약 종료 후 3년 또는 이용자 탈퇴 시까지(이용자 본인 의사 우선)리포트 재열람 편의 + 정보주체 권리
Access Code 사용 이력계약 종료 후 3년분쟁 대응 및 환불·재발급 처리
접속 로그3개월통신비밀보호법

위 보유 기간이 경과한 정보는 지체 없이 파기합니다. 다만 정보주체(이용자)가 본인 데이터의 즉시 삭제를 요청하는 경우, 회사는 법령상 보존 의무가 있는 경우를 제외하고 지체 없이 처리합니다.

제5조 (제3자 제공 및 처리위탁)

5.1 제3자 제공

회사는 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만 다음의 경우는 예외로 합니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 특별한 규정이 있거나 수사기관의 적법한 절차에 따른 요청이 있는 경우

5.2 처리위탁 (회사가 외부에 위탁하는 부분)

수탁자위탁 업무국가
Google LLC (Firebase)인증, 데이터베이스, 호스팅, 파일 저장미국(asia-northeast3 / asia-southeast1 리전)
Resend, Inc.이메일 발송 (견적서·코드·알림)미국
국세청 홈택스전자세금계산서 발행대한민국

회사는 위 수탁자들과 표준 데이터 처리 위탁계약(또는 동등한 약정)을 체결하고 있으며, 수탁자는 위탁 목적 외로 개인정보를 처리할 수 없습니다.

제6조 (안전성 확보 조치)

회사는 개인정보의 안전성 확보를 위해 다음 조치를 취하고 있습니다:

  • 관리적 조치: 개인정보 취급자 최소화, 정기적인 내부 점검
  • 기술적 조치:
    • 전송 시 TLS 암호화 (HTTPS 강제)
    • 비밀번호 일방향 해시 저장 (Firebase Authentication 표준)
    • Firestore/RTDB 규칙 기반 접근 제어, 모든 쓰기는 서버 함수 경유
    • 관리자 권한은 Firebase Auth Custom Claim으로 분리, 다단계 인증 적용
    • App Check + reCAPTCHA Enterprise로 자동화 공격 차단
  • 물리적 조치: Google Cloud 데이터센터의 국제 보안 인증(ISO 27001, SOC 2 등) 활용

제7조 (정보주체의 권리와 행사 방법)

이용자(정보주체)는 언제든지 다음 권리를 행사할 수 있습니다:

  • 개인정보 열람 요구
  • 오류 등이 있을 경우 정정·삭제 요구
  • 처리정지 요구
  • 본인 진단 데이터의 영구 삭제 요구

권리 행사는 회사 마이페이지(/mypage)의 회원 탈퇴 기능을 이용하거나, 이메일(faise@lifeportfolio.co.kr)로 요청하실 수 있습니다. 회사는 요청 접수 후 지체 없이(통상 영업일 기준 3일 이내) 조치합니다.
