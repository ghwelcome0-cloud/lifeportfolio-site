# 숏츠 전략 설계서 — 1편(자산이란) 기반 풀링·에버그린 숏츠

## A. 벤치마킹 요약 (유튜브 숏츠 / 인스타 릴스 / 틱톡 공통 원리, 2025~2026)

| 원리 | 데이터/근거 | 우리 적용 |
|---|---|---|
| **첫 2~3초가 분배를 결정** | VVSA 70~90%면 광범위 분배, 60% 미만이면 매몰 (Paddy Galloway 33억뷰 연구). 첫 2초 즉시 훅이면 +19% 유지 | 0.5초 내 첫 단어/텍스트, 인트로·로고·BGM 인트로 **금지** |
| **훅 5유형** | Bold Claim / Curiosity Gap / Micro-Story / Visual Shock / Direct Question | 교육 콘텐츠엔 **Bold Claim + Direct Question**이 최적 |
| **무음 시청 60%+** | 모바일 다수가 소리 끄고 시청 | 첫 단어부터 **번인 자막**(큰 고대비), 키워드 강조 |
| **길이: 15~35초 황금구간** | 30초·85% 완주 > 60초·50%. "아이디어에 맞는 최단 길이" | 우리 숏츠 **목표 30~45초** (지식 콘텐츠라 약간 길게, 단 완결성 유지) |
| **루프 설계** | 끝과 시작이 이어지면 재생 반복 → 추천 가산 | 마지막 문장이 첫 훅으로 되돌아가는 **순환 구조** |
| **2~4초마다 시각 변화** | 정적이면 알고리즘이 이탈로 읽음 | 컷·줌·B-roll 교차, 자막 모션 |
| **숏츠 = 퍼널** | Hook → Value → Next step. 숏츠로 발견 → 롱폼으로 심화 → 전환 | **숏츠 → 1편 롱폼 → 홈페이지** 2단 퍼널 |
| **시리즈/반복 포맷** | 동일 토픽 30일 강화가 알고리즘 학습에 유리 | 통일된 비주얼·오프닝·CTA의 **시리즈 포맷** |
| **9:16 세로, 중앙 1/3 안전구역** | 상단(채널명)·하단(버튼) 가림 주의 | 핵심 텍스트는 중앙, 상하단 여백 확보 |

### 우리만의 차별화·개선점 (하나님 중심·차분한 다큐 톤)
- 자극적 과장·어그로 훅 **금지**(가치 정합). 대신 **"진실의 반전"**형 훅 — 통념을 조용히 뒤집는 지적 호기심 자극.
- 빠른 컷의 피로감 대신, **차분하지만 한 컷도 멈추지 않는** 리듬(미세 줌+자막 모션으로 정적 방지).
- CTA는 판매 압박이 아니라 **"더 깊은 이야기로의 초대"**(롱폼 시청 유도가 1차, 구매는 2차).

## B. 퍼널 구조 (풀링 + 에버그린)

```
[숏츠 3~5개]  ──풀링(scroll-stopping 훅)──▶  [1편 롱폼 5:42]  ──전환──▶  [홈페이지/진단 ₩9,900]
   에버그린                                    심화·신뢰 구축              구매
```

- **풀링 역할**: 각 숏츠는 독립적으로 완결되며 강한 훅으로 신규 시청자 유입.
- **에버그린 역할**: 트렌드·시의성에 의존하지 않는 보편 주제(자산의 본질, 자기이해)라 1년 내내 유효.
- **연결 장치**: 숏츠 끝 CTA = "전체 이야기는 채널의 '자산이란' 영상에서" + 고정 댓글 = 홈페이지 링크.

---

## C. 숏츠 라인업 (3개 제안 — 각기 다른 훅 유형, 1편의 다른 앵글)

### 숏츠 #1 — "통장보다 먼저 쌓아야 할 자산" (Bold Claim + Curiosity Gap)
- **훅(0~3초)**: "당신이 '자산'이라 부르는 것, 사실 절반은 틀렸습니다."
- **밸류(3~33초)**: 자산의 3조건(미래가치·통제가능·축적가능) → 돈·부동산만이 아니다 → 자기이해(인적자본, 노벨상 이론)도 자산
- **루프/CTA(33~40초)**: "통장에 적히지 않는 그 자산이 뭔지, 전체 이야기는 채널에 있습니다" → 첫 문장으로 순환
- **길이 목표**: ~40초

### 숏츠 #2 — "목적이 또렷한 사람의 통장" (Visual Shock·숫자 + Direct Question)
- **훅(0~3초)**: "삶의 목적이 또렷한 사람은, 통장 잔고까지 달랐습니다."
- **밸류**: Hill 2016 연구(4,660명 추적) → 소득 +$4,400, 순자산 +$20,000 → "더 열심히 일해서가 아니라, 목적이 또렷했을 뿐"
- **CTA**: "왜 그런지, 어떻게 내 것으로 만드는지는 채널 '자산이란' 영상에서"
- **길이 목표**: ~35초

### 숏츠 #3 — "머릿속 생각은 자산이 아니다" (Direct Question + 반전)
- **훅(0~3초)**: "자기 자신을 잘 안다고 생각하세요? 그건 아직 자산이 아닙니다."
- **밸류**: 흩어진 생각 = 조각 → 통제·축적 가능한 형태(한 권의 리포트)로 바꿔야 자산 → 인생 포트폴리오의 역할
- **CTA**: "흩어진 나를 한 권으로 정리하는 법, 전체 이야기는 채널에 / 진단은 고정 댓글"
- **길이 목표**: ~40초

> 우선 **숏츠 #1**을 먼저 완성해 기준점(템플릿)을 만들고, 확정되면 #2·#3을 동일 포맷으로 양산하는 것을 권장합니다. (크레딧 효율)

## D. 제작 방식 (크레딧 효율 우선)

1. **음성**: 클론 음성(Ti6X6rs2eexqEt3ExCTr) 신규 녹음 — 숏츠용 짧은 대본(40초 분량). 1편 톤 유지.
2. **비주얼**: 9:16 세로. 1편 슬라이드/디자인 시스템(design.py) 재사용 + 세로용 재구성. B-roll은 **1편/3·4편 기존 자산 재활용 우선**(신규 생성 최소화).
3. **자막**: 번인(burned-in), 큰 고대비, 키워드 골드 강조, phrase-by-phrase 모션.
4. **BGM**: 1편 BGM 재사용(시리즈 일관성), 볼륨 낮게.
5. **루프**: 마지막 프레임 ≈ 첫 프레임으로 자연 재생 유도.
6. **썸네일/커버**: 9:16 커버 1장(채널 페이지·검색 노출용).

## E. 확인 요청 사항 (제작 착수 전)

1. **라인업**: 숏츠 3개 주제·훅 유형이 적절한가요? (먼저 #1만 완성 후 확장 방식 OK?)
2. **길이**: 30~45초 목표 적절한가요?
3. **훅 톤**: "절반은 틀렸습니다"류 — 자극적이지 않으면서 통념을 뒤집는 선, 가치 정합 OK인가요?
4. **CTA 2단**: 1차=롱폼 시청 유도("채널의 자산이란 영상"), 2차=홈페이지(고정 댓글) — 이 우선순위 OK인가요?
5. **비주얼**: 1편 디자인 재사용 + 세로 재구성 + 기존 B-roll 재활용(크레딧 절약) 방향 OK인가요?
