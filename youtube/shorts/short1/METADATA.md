# 🔴 [A] 유튜브 숏츠 (YouTube Shorts)

## 제목 (택1, 한글 기준 40자 내외 권장)
- **통장보다 먼저 쌓아야 할 '단 하나'의 자산 #shorts** ← 권장
- (대안) 당신이 '자산'이라 부르는 것, 절반은 빠져 있습니다 #shorts
- (대안) 노벨 경제학상이 증명한, 통장에 없는 자산 #shorts

## 설명란
```
당신이 '자산'이라 부르는 것 — 사실 절반은 빠져 있습니다.

자산에는 세 가지 조건이 있습니다.
① 미래에 가치를 만들고
② 내가 통제할 수 있고
③ 시간이 갈수록 쌓이는 것.

그런데 이 조건을 만족하는 건 돈과 부동산만이 아닙니다.
사람이 가진 지식·기술·자기이해 — 경제학은 이것을 '인적자본'이라 부릅니다.
노벨 경제학상이 증명한, 가장 강력한 자산이죠.

▶ 전체 이야기(5분)는 채널의 '자산이란' 영상에서 확인하세요. (고정 댓글 링크)
👉 나의 인생 자산을 한 권의 리포트로 정리하기 (고정 댓글 링크)

#인생자산화 #자기이해 #인적자본
```
> 유튜브 숏츠 해시태그는 3~5개가 적정 (과다 시 역효과). #shorts 는 제목에.

## 고정 댓글 (퍼널 핵심 — 2단 CTA, 클릭 가능)
```
📌 통장보다 먼저 쌓아야 할 '단 하나의 자산', 전체 이야기(5분)는 여기에서:
   ▶ [1편 '자산이란' 롱폼 URL]

🧭 흩어진 나를 '한 권의 리포트'로 정리하고 싶다면:
   👉 [홈페이지 / 인생 포트폴리오 진단 URL]
```

## 유튜브 업로드 체크리스트
- [ ] 제목에 `#Shorts` 포함 (세로+60초 이내면 자동 인식되나 명시 권장)
- [ ] 고정 댓글 작성 후 '고정' 설정
- [ ] 재생목록: "인생 자산화 · 숏츠"에 추가
- [ ] 카테고리: 교육 / 인물·블로그
- [ ] 시청자층: '아동용 아님' 설정
- [ ] 최종 화면/카드는 숏츠 미지원 → 고정 댓글로 대체

# 🟣 [B] 인스타그램 릴스 (Instagram Reels)

## 캡션 (첫 줄 = 훅. 첫 줄이 피드에서 가장 크게 보임)
```
당신이 '자산'이라 부르는 것, 사실 절반은 빠져 있습니다.

자산의 세 가지 조건 —
① 미래에 가치를 만들고
② 내가 통제할 수 있고
③ 시간이 갈수록 쌓이는 것.

돈·부동산만이 아닙니다.
당신의 지식·기술·자기이해, 즉 '인적자본'.
노벨 경제학상이 증명한 가장 강력한 자산이죠.

통장에 적히지 않는, 먼저 쌓아야 할 단 하나의 자산.
전체 이야기와 '나의 인생 리포트'는 👉 프로필 링크에서.

·
·
#인생자산화 #자기이해 #인적자본 #자산관리 #자기계발 #자기성장
#마인드셋 #동기부여 #인생설계 #자기개발 #성장마인드 #삶의지혜
#경제공부 #돈공부 #자존감 #리추얼 #일상기록 #자기반성 #릴스 #reels
```
> 인스타 해시태그는 10~20개 권장(발견 탭 핵심). 캡션 본문과 분리(·줄바꿈)하면 깔끔.
> 클릭 링크 불가 → 본문에서 "프로필 링크" 명시 + 프로필 bio에 링크(링크인바이오) 세팅.

## 인스타 업로드 체크리스트
- [ ] 커버(cover.png)를 릴스 커버로 지정 (피드 그리드 통일감)
- [ ] 첫 줄 훅 캡션 확인 (더보기 전에 노출되는 ~30자)
- [ ] 프로필 bio 링크 = 홈페이지(또는 링크트리: 1편 영상 + 진단 페이지)
- [ ] 오디오: 자체 내레이션이므로 '원본 오디오'로 게시 (트렌딩 사운드 미사용)
- [ ] 위치 태그/협업 태그(선택)
- [ ] 페이스북 동시 게시 옵션(선택)
- [ ] '릴스'로 게시(피드 동영상 아님) 확인

## 인스타 ↔ 유튜브 차이 요약
| 항목 | 유튜브 숏츠 | 인스타 릴스 |
|---|---|---|
| 제목 | 있음(검색 핵심) | 없음(캡션 첫 줄이 훅) |
| 링크 | 고정 댓글 클릭 가능 | 캡션 링크 불가 → 프로필 링크 |
| 해시태그 | 3~5개 | 10~20개 |
| 발견 경로 | 검색+추천 피드 | 해시태그+탐색 탭+추천 |
| CTA 문구 | "고정 댓글 링크" | "프로필 링크" |
