## 1️⃣ 한국 SaaS 벤치마킹 항목

### A. 분석 도구 사용 시 동의 패턴 (5가지 모델)

#### 모델 1: **암묵 동의 (Opt-out)** — *권장하지 않음 ❌*
- 처리방침에만 명시, 별도 동의 절차 없음
- 사용자가 거부하려면 직접 도구별 설정 변경 필요
- **위험**: 개보법 §15 명시·구체적 동의 위반 가능
- **예시**: 일부 구식 사이트, 글로벌 사이트 한국어 버전

#### 모델 2: **쿠키 배너 + 카테고리별 동의** — *GDPR 준수형 ✅*
- 사이트 첫 방문 시 배너 노출
- "필수" / "분석" / "마케팅" 카테고리별 토글
- "모두 거부"와 "모두 동의" 버튼이 동등 위계
- **장점**: GDPR/UK-GDPR 동시 준수
- **예시**: 글로벌 SaaS (Notion, Figma 등)의 한국 페이지

#### 모델 3: **가입 시 별도 동의 체크박스** — *국내 표준 ✅*
- 회원가입 폼에 "선택 동의 — 마케팅·분석 활용" 별도 체크박스
- 개보법 §22 (동의 분리) 준수
- 미동의 시에도 서비스 이용 가능해야 함
- **예시**: 토스, 카카오 등 다수 한국 SaaS

#### 모델 4: **이용 중 모달 동의** — *Late-binding 방식*
- 가입 후 첫 페이지 진입 시 모달로 동의 요청
- "동의" / "나중에" / "거부" 선택지
- **장점**: 가입 마찰 감소
- **단점**: 미응답 사용자 처리 정책 필요

#### 모델 5: **마이페이지 설정형 (Opt-in by default OFF)** — *가장 보수적 ✅✅*
- 기본값 OFF, 사용자가 마이페이지에서 명시적으로 ON
- 가입·이용 흐름에 영향 없음
- **장점**: 다크 패턴 위험 최저, 윤리 가드레일 최강
- **단점**: 옵트인율 낮음 (보통 < 5%)

**➡️ 우리 사이트 권장: 모델 3 + 모델 5 혼합**
- 가입 폼에 선택 동의 체크박스 (기본 미체크) — 모델 3
- 마이페이지에서 언제든 ON/OFF 가능 — 모델 5
- 동의자만 Clarity 스크립트 활성화

## 4️⃣ Microsoft Clarity 도입 시 필수 명시 사항

### 4-1. Clarity 자체 정책 분석 (공개 문서 기준)
- **수집 항목**: 클릭, 스크롤, 마우스 이동, 페이지 체류, 세션 영상 (텍스트 입력은 마스킹)
- **개인정보 처리**: 익명화 처리 (IP 마스킹 옵션 제공)
- **데이터 위치**: 미국 (Microsoft Azure)
- **보유 기간**: 기본 1년
- **공식 문서**: https://clarity.microsoft.com/terms

### 4-2. 처리방침 반영 필수 문구 (개보법 §28-8 국외 이전)

> 다음 5가지 사항을 처리방침에 **반드시** 명시해야 합니다:

| 항목 | 명시 내용 |
|---|---|
| 이전받는 자 | Microsoft Corporation |
| 이전 국가 | 미국 (United States) |
| 이전 일시·방법 | 사용자가 사이트 이용 중 실시간, HTTPS 암호화 전송 |
| 이전 항목 | 익명화된 행동 데이터(클릭·스크롤·세션 영상), 비식별 사용자 ID, 페이지 URL |
| 이전 목적 | UX 개선을 위한 행동 분석 |
| 보유 기간 | 최대 1년 |
| 거부 권리 | 마이페이지 또는 가입 시 동의 거부 가능, 거부 시 서비스 이용에 제한 없음 |

### 4-3. 동의 분리 원칙 (개보법 §22)
- ❌ **금지**: "이용약관 + 개인정보 처리방침 + 마케팅 동의 + Clarity 동의"를 **하나의 체크박스**로 묶기
- ✅ **필수**: 각 항목을 **별도 체크박스**로 분리
- ✅ **권장**: "필수 동의" / "선택 동의"를 시각적으로 명확히 구분

## 5️⃣ GDPR / CCPA 추가 고려 (해외 사용자 — Q2: 해외 더 유망)

### GDPR (EU·UK)
| 조항 | 요구사항 | 우리 사이트 대응 |
|---|---|---|
| Art. 6 | 처리 근거 명확화 (동의·계약·정당이익) | 분석 = 동의 기반 (Art. 6.1.a) |
| Art. 7 | 자유·구체·정보·명시적 동의 | 별도 체크박스, 거부와 동의 동등 위계 |
| Art. 7(3) | 철회권 | 마이페이지 1-click 철회 |
| Art. 13 | 정보 제공 | 처리방침에 처리자·기간·권리 명시 |
| Art. 44-49 | 국외 이전 | Microsoft SCCs (Standard Contractual Clauses) 명시 |
| Art. 25 | Privacy by Design | 기본값 OFF, 명시적 ON 시만 활성 |

### CCPA / CPRA (캘리포니아)
- "Do Not Sell or Share My Personal Information" 링크 푸터 배치
- Opt-out 권리 고지
- **참고**: Clarity는 "판매"에 해당하지 않으나, "공유(Share)" 정의에 따라 명시 권장

## 6️⃣ 윤리 가드레일 (기독교 가치관 + 보편 윤리)

벤치마킹 사례 중 **이런 패턴은 피해야 합니다** (우리 사이트는 채택 ❌):

| 안티 패턴 | 설명 | 우리 정책 |
|---|---|---|
| 다크 패턴 동의 | "동의" 버튼만 크고 "거부"는 작게 | ❌ 동등 위계 유지 |
| 강제 묶음 동의 | 마케팅 동의를 안 하면 가입 불가 | ❌ 선택 동의는 미체크여도 가입 가능 |
| 사전 체크 | 동의 체크박스가 기본 체크 상태 | ❌ 기본 미체크 |
| 회피형 표현 | "선택" 명시 없이 모호하게 처리 | ❌ "선택" 명시 |
| 미성년자 무방비 | 만 14세 미만 가입 차단 없음 | ⚠️ **현재 미명시 — 추가 필요** |
| 거짓 희소성·할인 | 정가 19,900원 ↘ 9,900원 (실제 미적용) | ✅ **이미 제거 결정** |
| 자동 갱신 함정 | 결제 후 자동 갱신 미고지 | ❌ 1회 결제만, 자동 갱신 없음 |

**기독교 가치관 추가 검토 사항**:
- ✅ 진리·정직 (요 8:32) — 거짓 광고 금지
- ✅ 약자 보호 (잠 31:8-9) — 미성년자·취약계층 보호
- ✅ 청지기 직분 (마 24:45) — 데이터를 함부로 다루지 않음
- ✅ 이웃 사랑 (마 22:39) — 사용자 권리 적극 존중

## 7️⃣ 권장 액션 (우선순위 순)

### 🔴 즉시 (PR #77-legal에 포함)
1. **만 14세 미만 가입 차단** 신설 (약관 + 처리방침)
2. **거짓 정가 표시 제거** (PR1·PR2·PR4 — 매핑표 v2 §1.3)
3. **처리방침 §5 위탁**에 Microsoft 추가 (dormant 상태이므로 활성 전 사전 고지)
4. **처리방침 §6 국외 이전**에 Microsoft 추가
5. **처리방침 §10 쿠키·자동수집**을 GA4/GTM/Clarity 명시로 보강

### 🟡 중기 (Clarity 실제 활성 전)
6. **가입 폼에 선택 동의 체크박스 추가** (회원가입 페이지)
7. **마이페이지에 분석 동의 ON/OFF 토글 추가**
8. **쿠키 배너 도입 검토** (해외 사용자 비중 증가 시)

### 🟢 장기 (해외 사용자 확장 시)
9. **CCPA Do Not Sell or Share 링크** 푸터 추가
10. **GDPR Data Protection Officer** 정보 추가 (EU 사용자 일정 규모 도달 시)
