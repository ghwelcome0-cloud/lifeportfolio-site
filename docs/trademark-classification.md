- **4개 류** 선택: 제9류 + 제16류 + 제41류 + 제42류

### 🟦 제9류 (Class 9) — 디지털 콘텐츠·소프트웨어 ⭐ 필수

> **사업 연결**: 결제 즉시 자동 생성되는 Only One 리포트(76문항 자기경영 진단 결과지) — 다운로드 가능한 디지털 출판물

**지정상품 명세 (한글)**:
1. 다운로드 가능한 전자 출판물(자기경영·인생설계 리포트)
2. 다운로드 가능한 전자 출판물(자기진단·강점분석·사명발견 가이드)

4. 인터넷에서 다운로드 가능한 전자 정기간행물
5. 다운로드 가능한 모바일용 응용 소프트웨어(자기경영 진단·실행 루틴 관리용)
6. 다운로드 가능한 컴퓨터 소프트웨어(설문 진단 및 결과 리포트 자동 생성용)
7. 다운로드 가능한 전자적 형태의 진단 결과지·차트·도표
8. 다운로드 가능한 자기계발용 전자북

- Downloadable electronic publications in the nature of self-development reports

## ⚠️ 출원 시 주의사항

### 1) 식별력 사전 검토 필수
- "인생포트폴리오"는 일반명사 조합이라 식별력 약함 가능성 → **로고(도형) 결합 출원** 강력 권장
- 권장: **결합상표** (한글 + 영문 + 도형 = 정사각 마크)

### 2) 선출원/유사 상표 검색
출원 전 KIPRIS(특허정보검색)에서 다음 검색 권장:
- "인생포트폴리오", "라이프포트폴리오", "Life Portfolio"
- 제9, 41, 42류 동일/유사 상품 내 선출원 여부

### 3) 지정상품 표현 규칙
- 너무 광범위한 표현(예: "모든 컴퓨터 소프트웨어")은 거절
- 위 명세서는 KIPO 「상품·서비스업의 명칭에 관한 고시」 표준명에 맞춰 작성됨

### 4) 우선권 주장 (해외 진출 대비)
영문 도메인(lifeportfolio.co.kr) 보유 + 영문 콘텐츠(blog/posts-en/) 운영 중이므로, 한국 출원일로부터 **6개월 내** 미국·일본·EU 등 우선권 주장 출원 가능 (조약상 우선권).

## 🛣️ 출원 절차 요약

```
1. KIPRIS 사전 검색 (무료)            ← 1주
2. 변리사 상담 / 명세서 검토           ← 1~2주
3. KIPO 전자출원 (특허로)              ← 1일
4. 출원 공고 / 심사 (6~10개월)         ← 자동 진행
5. 등록 결정 → 등록료 납부             ← 등록료 약 21만원/1류 10년
6. 등록 완료 → 10년 갱신               ← 매 10년
```

총 소요 기간: **출원 → 등록까지 약 8~12개월**
