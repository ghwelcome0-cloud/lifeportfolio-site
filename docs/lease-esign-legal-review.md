## 1. 결론 요약 (한 줄)

> 양 당사자가 **전자서명 방식에 합의**하면, 본 도구로 만든 전자서명 임대차계약서는 **법적으로 유효한 서명·계약**입니다. 다만 **확정일자·대항력(보증금 보호)이 핵심인 주택 임대차**라면 본 도구만으로는 부족하며, **국토부 공식 시스템** 또는 **주민센터 확정일자**를 병행해야 합니다.

## 2. 법적 효력의 근거 (검증 완료)

| 근거 법령 | 조문 | 핵심 내용 |
|---|---|---|
| 전자문서 및 전자거래 기본법 | **제4조 제1항** | 전자문서는 "전자적 형태"라는 이유만으로 법적 효력이 부인되지 않는다 |
| 전자서명법 | **제3조 제2항** | 공인전자서명이 아닌 일반 전자서명도 **당사자 간의 약정에 따라** 서명·서명날인 또는 기명날인의 효력을 가진다 |
| 민법 | (계약 일반) | 계약은 청약과 승낙의 합치로 성립하며, 임대차계약은 별도의 방식(서면·공증)을 효력요건으로 요구하지 않는다 (낙성·불요식 계약) |

**해석:** 임대차계약은 본래 구두로도 성립하는 계약입니다. 전자서명은 "이 사람이 이 내용에 동의했다"는 **의사 표시 + 신원 귀속**을 보강하는 증거입니다. 전자서명법 제3조 제2항에 따라, 당사자가 "우리는 이 전자서명 방식으로 서명하기로 한다"고 **합의**하면 그 서명은 법적 효력을 가집니다.

> 본 도구는 계약서 본문에 **"본 계약은 양 당사자가 전자서명 방식에 합의하여 체결되었다"**는 문구와, 서명 시 **"위 방식으로 서명함에 합의한다"**는 의사 표시 안내를 명시하여 이 요건을 충족하도록 설계했습니다.

## 4. 본 도구가 갖춘 신뢰성 장치 (증거력 보강)

| 장치 | 구현 방식 | 효과 |
|---|---|---|
| **무결성 해시** | 계약내용 + 양측 서명이미지 + 시각을 합쳐 **SHA-256** 산출 | 한 글자라도 위변조되면 해시 불일치 → 위변조 검증 가능 |
| **타임스탬프** | 서명 확정 시각을 ISO 8601로 기록·문서에 인쇄 | 체결 시점 증거 |
| **서명 의사 명시** | 서명 화면에 "이 방식으로 서명함에 합의" 안내 + 계약서 본문에 합의 문구 | 전자서명법 제3조 제2항 "약정" 요건 충족 |
| **서명 기록 보관(JSON)** | 계약내용·서명·해시·시각을 JSON으로 다운로드 | 분쟁 시 원본 대조용 증거 |
| **로컬 처리** | 모든 처리가 브라우저 내에서만 동작, 서버 미전송 | 개인정보 유출 위험 최소화 |

## 5. ⚠️ 본 도구의 한계 (솔직한 고지)

| 한계 | 설명 | 대안 |
|---|---|---|
| **본인확인(身元확인) 없음** | 휴대폰 인증 등 강한 신원확인을 하지 않음. "서명한 사람이 진짜 본인인지"는 당사자 신뢰에 의존 | 강한 본인확인 필요 시 → **모두싸인·이폼사인**(휴대폰 인증 50~100원/건) |
| **확정일자 자동 부여 불가** | 주택임대차보호법상 대항력·우선변제권의 핵심인 확정일자를 자동으로 받을 수 없음 | **주민센터 방문** 또는 **국토부 부동산거래 전자계약시스템**(irts.molit.go.kr — 확정일자 자동) |
| **임대차 신고 미연동** | 임대차계약 신고제(보증금 6천만원 또는 월세 30만원 초과)는 별도 신고 필요 | 관할 주민센터 / 부동산거래관리시스템 신고 |
| **공인전자서명 아님** | 공동인증서 기반 공인전자서명보다 증거력은 약함(단, 법적 효력 자체는 인정) | 최고 수준 증거력 필요 시 공인전자서명 서비스 |
| **타임스탬프 = 로컬 시각** | 신뢰된 제3자(TSA) 타임스탬프가 아닌 사용자 기기 시각 | 공인 타임스탬프 필요 시 상용 서비스 |

## 7. 권장 운영 수칙 (분쟁 예방)

1. **사전 합의 문구 확보** — "본 계약은 전자서명 방식으로 체결하며 양 당사자가 이에 동의한다"를 특약사항에도 한 줄 넣으면 더 안전합니다.
2. **서명 기록(JSON) + PDF 둘 다 보관** — 양 당사자가 각자 사본을 보관하세요.
3. **무결성 해시 별도 기록** — 계약 직후 해시값을 문자/이메일로 상대방과 공유해 두면 후일 위변조 주장에 대응 가능.
4. **신분증 사본 등 본인확인 자료 별도 첨부** (필요 시) — 도구가 본인확인을 하지 않으므로, 중요 계약은 신분 확인 자료를 함께 보관.
5. **주택 임대차는 반드시 확정일자** — 본 도구로 서명했더라도, 보증금 보호를 위해 주민센터 확정일자를 별도로 받으세요.
