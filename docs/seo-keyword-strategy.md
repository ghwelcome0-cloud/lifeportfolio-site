### 1.3 본질 문제 (Rev 1 유지, 명료화)

**우리는 두 가지 검색 게임을 잘못 동시에 시도하고 있었다:**
1. 브랜드 키워드 ("인생포트폴리오") — 검색량 부족, 단어 자체에 경쟁 콘텐츠(찰스 핸디 책 등)
2. 거시 키워드 ("사명", "자기경영") — 검색량 자체가 채널화 불가

→ **둘 다 포기.** "MBTI 다음을 검색하는 30대 직장인"만 노린다.

| 키워드 | 제거 사유 |

1. **`이직을 결정하기 전, 30대가 스스로에게 던져야 할 13가지 질문`** → `이직 결정 방법`
2. **`갤럽 강점검사 27,000원, 30대 직장인에게 정말 그 값을 할까`** → `갤럽 강점 단점`
3. **`MBTI 다음에 뭘 해야 할까 — 30대를 위한 4단계 자기 정렬 방법`** → `MBTI 다음 단계`
4. **`30대 번아웃의 진짜 이유 — 회복은 휴식이 아니다`** → `30대 번아웃`
5. **`16personalities는 정식 MBTI가 아니다 — 무료 검사 너머의 자기 이해`** → `16personalities 신뢰도`

각 글 사양:
- 분량 2,500~3,500자

- FAQ 섹션 + Schema.org FAQPage JSON-LD

- 카카오 CTA + 9,900원 결제 CTA + (출시 후) 다이어리 CTA
