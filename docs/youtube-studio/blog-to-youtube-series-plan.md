# 블로그 → 유튜브 콘텐츠 시리즈 기획 (인생포트폴리오 고유 자산 기반)

> 원칙
> - **MBTI·타 검사와 비교/경쟁하지 않는다.** 인생포트폴리오는 독립된 가치(사명·비전·강점·실행).
> - **모든 주제는 인생포트폴리오 블로그 44편**에 이미 담긴 대표님 가치관·의도에서 가져온다.
> - 새로 지어내지 않고 **검증된 자기 자산을 재활용** → 톤·신학·근거 일관성 보장.
> - 성경 가치관(사랑·진실·섬김), 첫 주 9,900원 진단 전환.
> - 용어: 콘텐츠 완성도 시각화 UI는 **"완성도 게이지"**로 통일(특정 외부 비유 명칭 미사용).

## 0) 검증 결과 — 왜 블로그 재활용이 유튜브 알고리즘에도 유리한가 (내·외부 검증)

**내부 검증(블로그 44편 분석):**
- 글의 80% 이상이 "사명/소명/자기경영/실행 설계" 축 → **채널 정체성 일관**(알고리즘이 좋아하는 '주제 집중도' 확보).
- 학술 근거 풍부(프랭클·Gollwitzer·Bandura·Ryff·Maslach 등) → 영상 신뢰도·체류시간(AVD) 유리.
- 시즌 글(Q1~Q4) 존재 → **발행 캘린더**에 그대로 얹으면 검색 시즌성 공략 가능.

**외부 검증(2026 유튜브 알고리즘 일반 원리):**
- 알고리즘은 ① 클릭(CTR) ② 시청지속(AVD/retention) ③ 만족·참여를 본다.
- **토픽 클러스터**(한 주제를 여러 각도로 묶어 발행)는 "연관 영상 추천"을 강화 → 한 편이 다음 편을 끌어올림.
- 블로그 글은 이미 **검색 키워드가 정리**돼 있어, 영상 제목·태그·설명란 SEO에 그대로 활용 가능.
- → 결론: **"이미 가진 검증된 글을 클러스터로 묶어 발행"하는 것이 알고리즘·전환 양쪽에 가장 효율적.**

**솔직한 한계:** 블로그가 곧 조회수를 보장하진 않는다. 썸네일·제목·도입 30초(후킹)는 **영상용으로 새로 설계**해야 한다. 블로그는 '내용·근거·톤'의 원천이고, '주목도'는 영상 제작에서 만든다.

## 1) 3개 콘텐츠 카테고리 매핑 (블로그 → 유튜브)

| 카테고리 | 역할 | 블로그 출처 후보 |
|---|---|---|
| **키(Key)** | 채널 간판·정체성 | 「의미를 살아낸다는 것」 / 「마24:45 신학적 뿌리」 / 「Founder Case」 |
| **에버그린(Evergreen)** | SEO 검색 영구 유입 | 진로·번아웃·자기계발 실패·미라클모닝 등 '검색 수요 큰' 글 |
| **풀링(Pulling)** | 트렌드·숏츠 신규 유입 | 원포인트업·새해결심·연말회고 등 '시즌/통념 반전' 글 |

## 2) 토픽 클러스터 5개 (시리즈 단위로 발행 → 알고리즘 추천 강화)

### 클러스터 A — "의미·소명" (채널의 심장 / 키)
- 키: 「의미를 살아낸다는 것 — 우리 시대의 가장 조용한 갈증」(빅터 프랭클)
- 보조: 「소명이라는 단어가 일터에서 사라진 이유」 / 「사명을 분별하는 13가지 질문」
- 전환 포인트: "내 사명을 한 줄로" → 76문항 진단

### 클러스터 B — "진로·방향" (검색 강세 / 에버그린)
- 「10·20대, 진로 정하기 전 점검할 한 가지」
- 「30대, 진로가 흔들릴 때」 / 「40대, 인생 2막 전에」
- 「내가 뭘 좋아하는지 모르겠다는 사람들에게 — 좋아함의 해상도」
- 전환: "방향 먼저, 직업은 나중" → 진단

### 클러스터 C — "번아웃·회복" (검색·시즌 강세 / 에버그린+풀링)
- 「40대 번아웃의 정체 — 효능감이 무너져서」
- 「번아웃 후 다시 시작하는 법 — 사명부터」
- 「5월에 자주 오는 번아웃 신호」(시즌 풀링)
- 전환: "다시 시작은 사명부터" → 진단

### 클러스터 D — "왜 안 바뀌는가: 실행의 과학" (자기계발 통념 반전 / 에버그린+풀링)
- 「자기계발서 50권 읽어도 안 바뀌는 이유」
- 「새벽 5시에 일어나도 안 바뀌는 이유 — 미라클모닝이 놓친 것」
- 「If-Then 한 줄의 힘」 / 「검사 결과는 왜 서랍에서 잠드는가」
- 전환: "읽기와 실행 사이의 간극" → 진단+실행 설계도

### 클러스터 E — "측정으로 다시 짜는 한 해" (시즌 풀링)
- Q1 「1월 결심이 사라질 때 — 결심 대신 측정」
- Q3 「휴가에서 돌아오기 싫을 때 — 원포인트업」
- Q4 「연말 회고를 측정 한 장으로」
- 전환: "결심 말고 측정" → 진단

## 3) 발행 캘린더 예시 (클러스터 순환 + 시즌)

| 주차 | 키(월 1편) | 에버그린(주 1편) | 풀링 숏츠(주 2~3편) |
|---|---|---|---|
| 1월 | A 의미·소명 | B 진로(20대) | E Q1 새해결심(반전 숏츠) |
| 2월 | (A 심화) | D 자기계발서 50권 | D 미라클모닝(숏츠) |
| 5월 | C 번아웃 정체 | C 번아웃 회복 | C 5월 번아웃 신호(시즌) |
| 연말 | A Founder Case | E 연말회고 | E 측정 회고(숏츠) |

→ **키는 월 1편**(공들여), 에버그린은 꾸준히, 풀링은 자주(신규 유입). 풀링이 키/에버그린으로 핑.

## 4) 각 영상 공통 제작 규칙
- 제목·썸네일은 **블로그 제목을 그대로 쓰지 말고** 영상 후킹용으로 재설계.
- 대본 5섹션(후킹·공감·가치·증거·CTA). 가치=블로그 핵심 논지, 증거=블로그 속 학술 근거/사례.
- 가치관 점검 통과(공포·조작·분노·결핍 자극 금지).
- CTA는 인생포트폴리오 진단(첫 주 9,900원)으로. 강요 아닌 초청.
- 설명란: 원문 블로그 링크 + 진단 링크. 고정 댓글: 진단 링크 + 다음 영상.
