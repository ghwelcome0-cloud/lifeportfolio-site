# 가치관 점검 — 금지 표현 리스트 (체크리스트 기능용)

> 목적: 대본 작성/붙여넣기 시, **공포·조작·분노·결핍 자극** 표현을 감지해 경고하는
> "가치관 점검 장치"의 판정 사전. 무거운 AI 판정 없이 **키워드/패턴 매칭**으로 동작.
> 기준 성구: 빌립보서 4:8(참된 것) / 빌립보서 2:3-4(섬김) / 사명: 마태복음 24:45.
> 핵심 원칙: **사랑·진실·섬김으로 "초청"한다. 공포·결핍·분노로 "압박"하지 않는다.**
> (막시즘의 분노·헤겔의 대립진보와 정반대 방향)

---

## 동작 방식 (개발자 참고)

1. 대본 텍스트에서 아래 **금지 패턴**이 감지되면 노란색 경고 + 해당 문장 하이라이트.
2. 경고 문구 예: `"이 표현은 사랑·진실 기준(빌4:8)에서 벗어날 수 있습니다. 아래 대안을 참고하세요."`
3. 강제 차단이 아니라 **점검·제안**. 최종 판단은 대표님(사람)이 함.
4. 카테고리별 점수: 금지 표현 개수 → "가치관 안전도" 게이지(완성도 게이지와 별개의 보조 지표).

---

## 카테고리 1. 공포·불안 자극 (FOMO·협박성)

**금지 패턴 (감지 대상):**
- "지금 안 하면 망한다 / 늦는다 / 뒤처진다 / 끝장난다"
- "이거 모르면 평생 후회"
- "당신만 모르는 / 남들 다 하는데 너만"
- "마지막 기회 / 두 번 다시 없는 / 곧 사라집니다"
- "안 사면 손해 / 손해 보고 싶지 않으면"
- "경고합니다 / 충격 / 소름"

**✅ 대안 방향:**
- "지금 안 하면 망한다" → "준비되셨을 때 시작하셔도 늦지 않습니다"
- "이거 모르면 평생 후회" → "알면 분명 도움이 되는 이야기예요"
- "마지막 기회" → "관심 있으시면 편하게 살펴보세요"

---

## 카테고리 2. 결핍·열등감 자극

**금지 패턴:**
- "당신은 부족하다 / 무능하다 / 실패자"
- "남들보다 뒤처진 / 평범한 / 그저 그런 인생"
- "이대로 살면 안 됩니다 / 지금 당신은 틀렸다"
- "한심한 / 게으른 / 핑계 대는"
- "왜 아직도 / 아직도 이러고 있나요"

**✅ 대안 방향:**
- "당신은 부족하다" → "아직 발견하지 못한 강점이 있을 뿐입니다"
- "이대로 살면 안 됩니다" → "조금만 방향을 정리하면 달라질 수 있어요"
- "남들보다 뒤처진" → "당신만의 속도가 있습니다"

---

## 카테고리 3. 분노·적개심·편가르기 (막시즘적 대립)

**금지 패턴:**
- "그들이 당신을 속였다 / 착취 / 음모"
- "기득권 / 시스템이 당신을 망쳤다"
- "분노하라 / 들고일어나 / 맞서 싸워라"
- "적 / 가해자 vs 피해자" 식 이분법
- "세상이 잘못됐다 / 사회가 너를 망쳤다"

**✅ 대안 방향:**
- "그들이 당신을 속였다" → "오해가 있었던 부분을 함께 정리해 봐요"
- "분노하라" → "차분히 내 길을 다시 세워봅시다"
- 편가르기 → 누구도 적으로 돌리지 않고 "당신 자신의 성장"에 집중

---

## 카테고리 4. 과장·거짓·조작된 증거

**금지 패턴:**
- "100% 보장 / 무조건 / 반드시 성공 / 절대"
- "단 하루 만에 / 즉시 인생역전 / 누구나 부자"
- "비밀 / 아무도 모르는 / 단 하나의 비법"
- "검증되지 않은 수치·후기" (출처 없는 "○○% 효과")
- "기적의 / 마법 같은 / 한 방에"

**✅ 대안 방향:**
- "100% 보장" → "도움이 되도록 정직하게 설계했습니다"
- "단 하루 만에 인생역전" → "한 걸음씩 방향을 잡아갑니다"
- "아무도 모르는 비밀" → "꾸준히 적용하면 분명한 원리"
- 수치 인용 시 **출처/근거 표기** 권장

---

## 카테고리 5. 탐욕·시기심 자극

**금지 패턴:**
- "남들보다 앞서가라 / 이겨라 / 밟고 올라서라"
- "돈방석 / 떼돈 / 일확천금 / 부자만 아는"
- "부러움을 사는 / 과시 / 자랑"
- "경쟁자를 따돌리고"

**✅ 대안 방향:**
- "남들보다 앞서가라" → "당신답게 자라가세요"
- "떼돈" → "정직한 성장의 열매"
- 비교·시기 → "자기 자신과의 비교, 어제보다 나은 오늘"

---

## 카테고리 6. 압박·강매성 행동 유도

**금지 패턴:**
- "지금 당장 / 망설이지 말고 / 고민하지 말고 결제"
- "선착순 / 한정 / 오늘만 / 마감 임박" (없는 긴급성 조작)
- "안 사면 바보 / 손해"
- "카드 꺼내세요 / 일단 질러"

**✅ 대안 방향:**
- "지금 당장 결제" → "준비되셨다면 편하게 시작해 보세요"
- "선착순/마감 임박"(가짜 긴급성) → 제거. 진짜 마감이 아니면 쓰지 않음
- "고민하지 말고" → "충분히 생각하고 결정하셔도 됩니다"

## 허용 권장 표현 (긍정 사전 — 이런 톤이면 통과)

- 초청형: "~해 보세요 / 궁금하시면 / 준비되셨다면 / 함께"
- 진실형: "정직하게 / 솔직히 / 있는 그대로 / 검증된"
- 섬김형: "도움이 되도록 / 응원합니다 / 동행 / 곁에서"
- 소망형: "당신답게 / 회복 / 성장 / 한 걸음씩 / 방향"
- 자유의지 존중: "강요하지 않습니다 / 선택은 당신의 몫 / 부담 없이"

## 판정 규칙 요약 (개발자용 의사코드)

```
for each 금지패턴 in (카테고리1~6):
    if 대본.contains(금지패턴):
        문장 하이라이트(노랑)
        경고("사랑·진실 기준에서 벗어날 수 있습니다")
        대안문구 표시
가치관_안전도 = 100 - (감지된 금지표현 수 × 가중치)
# 강제 차단 아님. 사람(대표)이 최종 판단.
```
