# 인생포트폴리오 벤치마킹 리포트 v1.0

> **목적**: 우리 서비스 결(**신뢰 · 품격 · 자기발견 · 진단 리포트 · 하나님 중심의 진중함 · 자산화**)에 맞는
> 검증된 성공 랜딩페이지를 해부하여, **"무엇이 그들을 9점으로 만드는가"**를 정의하고
> 우리 톤앤매너 리디자인의 원칙으로 삼는다.
>
> **작성**: 2026-06-25 · **방법 A** (제가 조사·선정, 객관 후보 + 성공 원리 분석)

---

## 1. 벤치마킹 대상 4선 (우리 결과의 매핑)

| 사이트 | 카테고리 | 우리 결과의 연결고리 | 규모/권위 |
|--------|----------|---------------------|-----------|
| **Gallup CliftonStrengths** | 진단 리포트 권위 | "76문항→Only One 리포트" = 진단의 권위·신뢰 | 3,746만 명 사용 |
| **Crystal Knows** | 성격 리포트 SaaS | "유형 아닌 한 사람" = 다차원 개인 프로파일 | 200만+ 전문가 |
| **16Personalities** | 자기발견 진단 (대중) | "발견하고 살아내고" = 따뜻한 자기발견 초대 | 누적 15.5억 검사 |
| **Wealthfront** | 자산화 핀테크 | "삶이 자산이 된다" = 축적·자산화의 정점 | 140만 고객·$95B |

> 4개 모두 **"개인의 무언가(강점·성격·자아·돈)를 진단/축적하여 더 나은 미래로"** 라는
> 우리와 동일한 서사 구조를 가지면서, 톤앤매너에서 9점대를 유지하는 사례다.

## 2. 각 사이트 해부 — "왜 9점인가"

### 🟦 Gallup CliftonStrengths (진단 권위)
- **히어로**: "You Are Unique. You Are Powerful. Be Confident With Who You Are."
- **신뢰 구축**: 단일 거대 카운터 `37,464,983명` + Gallup 트레이드마크(연구 권위)
- **리포트 시각화**: 모바일 mockup으로 "당신의 리포트"를 **만질 수 있게** 보여줌
- **구조**: Step 1(검사) → 2(결과) → 3(적용) **선형 3단계** = "검증된 방법론" 신호
- **★ 9점 비결**: **정제된 자신감** — 짧은 단언형 헤드라인 + 넓은 여백 + 단 하나의 핵심 숫자

### 🟩 Crystal Knows (성격 리포트 SaaS)
- **히어로**: "Your personality is irreplaceable." + "200만+ 전문가가 신뢰, 2015년 이래"
- **CTA**: 단 2개 (`Try it Free` / `Request a Demo`)
- **신뢰 구축**: 사용자 수 + 업력 + **로고 클라우드**(Adobe·Verizon·Amazon)
- **타이포**: 굵은 헤드라인 + **이탤릭 강조**(irreplaceable, advantage)
- **★ 9점 비결**: **감정적 훅 → 사회적 증명 → 기능 카테고리** 시선 동선이 단일·명확. 여백 과감.

### 🟨 16Personalities (자기발견·따뜻함)
- **히어로**: "It's so incredible to finally be understood." + "단 10분이면…"
- **CTA**: **단 하나** `Take the Test` (인트로+후기 뒤 반복 배치)
- **따뜻함**: 공감형 카피 + 실제 인물 사진 후기 + 부드러운 일러스트
- **마찰 제거**: "무료 · 10분 · 가입 불필요" 를 전면에
- **★ 9점 비결**: **공감 헤드라인 + 단일 CTA + 마찰 최소화**. 진단인데 "초대"처럼 느껴짐.

### 🟫 Wealthfront (자산화 핀테크)
- **히어로**: 핵심 숫자(`4.20% APY`)를 **초대형 타이포**로 → "Turn savings into wealth"
- **신뢰 4기둥**: ①대량 숫자(140만·$95B) ②규제·보안(FDIC) ③업력(2011~) ④제도적 정당성(NASDAQ·Forbes)
- **축적 시각화**: "savings → wealth" 전환을 카피·그래픽으로 명시
- **타이포 위계**: "가장 인상적인 숫자"가 **가장 먼저** 눈에 들어오도록 스케일 차등
- **★ 9점 비결**: **단 하나의 압도적 숫자 + 차분한 프리미엄 + 신뢰 기둥의 레이어드**

## 3. 공통 성공 원리 — "9점 만드는 7원칙"

> 4개 사이트가 **공통으로** 지키는 것. 우리 리디자인의 헌법.

| # | 원칙 | 4사 공통 패턴 | 우리 현재 상태 |
|---|------|--------------|----------------|
| **P1** | **단 하나의 영웅 메시지** | 히어로 헤드라인 1줄 + 서브 1줄. 끝. | ❌ 텍스트 블록 9개 |
| **P2** | **CTA는 1개(최대 2개)** | 주 CTA 1개를 압도적으로 | ⚠️ CTA 2개+가격박스+칩 혼재 |
| **P3** | **하나의 압도적 신뢰 신호** | 큰 숫자 1개(3746만/200만/15.5억/$95B) | ❌ 신뢰 신호 분산·약함 |
| **P4** | **과감한 여백** | 첫 화면 호흡, 요소 간 큰 간격 | ❌ 빽빽함 |
| **P5** | **타이포 위계 = 시선 동선** | 헤드라인 초대형 → 서브 → 본문, 단계 명확 | ❌ 33종 사이즈, 위계 붕괴 |
| **P6** | **리포트/결과를 "보여줌"** | mockup·이미지로 결과물을 만질 수 있게 | ⚠️ 텍스트로만 설명 |
| **P7** | **차분한 프리미엄 컬러** | 절제된 팔레트 + 1~2 액센트 | ✅ 우리 강점(슬레이트+골드+크림) |

## 4. 우리에게 주는 결론 (리디자인 방향)

### ✅ 유지할 강점
- **컬러 팔레트**(딥 슬레이트 블루+골드+크림) — 이미 차분한 프리미엄. P7 충족.
- **콘텐츠·메시지의 깊이** — 카피 자체는 훌륭함(버릴 게 아니라 **순서·노출 절제**의 문제)

### 🎯 핵심 전환 (5대 약점 ↔ 7원칙 매핑)
| 우리 약점 | 적용할 원칙 | 리디자인 액션 |
|-----------|------------|--------------|
| #2 히어로 과포화 | P1·P2 | 첫 화면 = **헤드라인 1 + 서브 1 + 주 CTA 1** + 가격·신뢰는 한 줄로 압축. 나머지는 스크롤 아래로 |
| #1 타이포 붕괴 | P5 | **헤드라인 초대형화**, 6~8단계 스케일로 위계 = 시선 동선 |
| #3 본문 가독성 | P5 | 본문 15~16px, 스캔 가능 라벨/숫자 |
| #4 간격 부재 | P4 | 8px 그리드 + 과감한 첫 화면 여백 |
| 신뢰 분산 | P3 | **하나의 압도적 신뢰 신호** 정립 (예: 실제 검사 완료자 후기·누적 수 — 이미 후기 시스템 가동!) |
| 결과 시각화 | P6 | 리포트 **샘플 이미지/mockup**을 히어로 또는 바로 아래에 노출 |

### 💡 우리만의 차별화 (벤치마크에 없는 우리 결)
- **하나님 중심의 진중함** → 가벼운 "10분 재미 테스트"(16P)가 아니라, **"삶을 자산으로 남기는 진중한 발견"**의 품격
- **자산화 서사** → Wealthfront의 "돈의 축적"을 우리는 **"삶·사명의 축적"**으로 번역
- → 즉, **Gallup의 권위 + Crystal의 정제 + Wealthfront의 자산 서사 + 우리만의 영적 진중함**
