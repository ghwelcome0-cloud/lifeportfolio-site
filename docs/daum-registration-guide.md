# 다음(Daum) / 카카오 검색노출 등록 가이드 (C2)

> 목표: 다음 검색에서 **"인생포트폴리오"** 검색 시 공식 사이트가 상단 노출되도록 한다.
> 카카오톡 검색은 다음(Daum) 검색 결과를 그대로 사용하므로, 이 가이드 하나로 다음 + 카카오 모두 커버한다.
> Last updated: 2026-05-19

---

## 0. 다음/카카오 검색의 구조 (한 줄 요약)

- 다음(Daum)은 자체 크롤러 **Daumoa**로 색인.
- 카카오톡 검색창은 **다음 검색을 그대로** 호출 — 별도 등록 불필요.
- 다음은 **사이트 등록 신청 페이지**가 있으므로 명시적 등록이 필수.

---

## 1. 다음 검색 등록 신청

1. 접속: <https://register.search.daum.net/index.daum>
2. 메뉴 **[사이트 등록]** 클릭
3. 등록 정보 입력:
   - **사이트명**: 인생포트폴리오 / Life Portfolio
   - **URL**: `https://lifeportfolio.co.kr`
   - **카테고리**: 자기계발 / 생활·취미 / 진로·진학 중 가장 가까운 것
   - **사이트 소개 (500자 이내)**:
     ```
     인생포트폴리오는 자기이해 → 자기표현 → 자기설계 → 자기실행 4단계를
     하나의 흐름으로 묶은 맞춤 자기경영 진단 서비스입니다.
     76문항 15분 진단으로 사명·비전·정체성·강점과 추천 진로·직업,
     21일 실행 프로그램까지 두 권의 PDF로 결제 즉시 자동 생성됩니다.
     국내 9,900원 · 해외 USD 8.99. 한국어/영어 동시 지원.
     ```
   - **담당자 이메일**: `faise@lifeportfolio.co.kr`
4. [등록 신청] 클릭
5. 신청 후 영업일 기준 **1~3주 이내** 심사 완료 (이메일 통지)

## 2. 컬렉션별 등록 (선택적)

다음은 **블로그**, **카페**, **사이트** 컬렉션이 분리되어 있다.
공식 사이트는 §1로 충분하지만, 다음 블로그(티스토리 포함)에 동일 콘텐츠를 일부 옮기면 추가 노출 기회가 생긴다.

- **티스토리 블로그 운영** (무료, 다음 우대): <https://www.tistory.com/>
  - 운영 계정: `faise@lifeportfolio.co.kr` 연결
  - 한 달에 1~2건만, 블로그 본문에 본 사이트 백링크 1회 (자연스럽게)

---

## 3. 사이트맵·RSS 제출

다음은 **별도의 사이트맵 직접 제출 UI가 없다.** 대신 다음 봇이 `robots.txt`의 sitemap 선언을 자동으로 읽는다.
이미 `robots.txt`에 다음 두 줄이 포함되어 있어 추가 작업 불필요:

```
Sitemap: https://lifeportfolio.co.kr/sitemap.xml
Sitemap: https://lifeportfolio.co.kr/blog/rss.xml
```

---

## 4. Daumoa 크롤러 정책

- `robots.txt`에 Daumoa 명시적 허용이 포함되어 있어야 한다 (이미 적용됨).
- 방화벽(Cloudflare)에서 Daumoa UA 차단되지 않는지 분기 1회 확인.

확인 방법:
```bash
curl -A "Mozilla/5.0 (compatible; Daum/4.1; +http://cs.daum.net/faq/15/4118.html?faqId=28966)" \
     -I https://lifeportfolio.co.kr/
```
→ 200 OK가 나와야 정상.

## 5. 카카오톡 검색 노출 확인

- 카카오톡 검색창에서 "인생포트폴리오" 검색
- 결과 카드가 다음 검색 결과와 동일하게 노출되는지 확인
- 노출되지 않으면 다음 검색에서 먼저 노출되어야 한다 (다음 우선)

## 6. 카카오 채널 (선택 — 강력 권장)

다음과 별개로 **카카오 채널**을 운영하면 카카오톡 검색 결과에 채널 카드가 우선 노출된다.

1. 접속: <https://center-pf.kakao.com/>
2. 카카오 비즈니스 계정 로그인 (없으면 신규 가입)
3. **[새 채널 만들기]** > 채널명: `인생포트폴리오` / 검색용 ID: `lifeportfolio`
4. 프로필 등록:
   - 프로필 사진: `og-default-ko.png` (1200×630 잘라서 정사각형 또는 카카오 권장 비율)
   - 소개: 본문은 §1의 500자 소개 그대로
   - 홈페이지 URL: `https://lifeportfolio.co.kr`
5. **카카오톡 채널 검색 허용** ON
6. 채널 공개 후 1주일 이내 카카오톡 검색에서 채널 결과 카드 노출

> 채널은 한 번 만들면 카카오톡 사용자가 "인생포트폴리오"를 검색했을 때
> 다음 검색 결과 + 채널 카드가 동시에 노출되므로 SERP 점유율이 늘어난다.

## 8. 트러블슈팅

- [ ] 등록 신청 후 4주가 지나도 등록 안 되면 → 같은 페이지에서 [등록 상태 확인]으로 재확인, 필요시 재신청.
- [ ] 카카오톡 검색에서만 안 보이면 → 다음 검색 노출 먼저 확인. 다음에서도 안 보이면 등록 단계로 되돌아감.
- [ ] 검색 결과의 사이트 제목·설명이 의도와 다르면 → `index.html`의 `<title>`, `<meta name="description">`을 손보고 다음 봇 재크롤 대기 (수동 재제출 UI 없음, 자연 갱신 1~2주).
