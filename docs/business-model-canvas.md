## 0. 한 줄 비즈니스 정의 (Business Definition v1.4.2)

> **"30대 직장인이 이직·번아웃 한가운데서 자기를 다시 정렬하는 데 쓰는, 진단 → 다이어리 → 코칭 → 커뮤니티가 이어지는 자기경영 서비스. 설립자가 직접 운영한다."**

이전 정의(v1.3 이전: "76문항 진단 + PDF 리포트 판매")에서 **명시적으로 폐기된 것**:
- ❌ "MBTI 대안" 포지셔닝 (MBTI와 직접 경쟁 안 함 — 검색량 너무 큼, 16personalities에 잠식됨)
- ❌ "진단 PDF 판매 사업" (보고서: LTV/CAC 구조적 함정)
- ❌ "AI 자동 코칭" (설립자가 줄 수 있는 것이 아님)

## 1. 페르소나 (Customer Segments)

### 1.1 핵심 페르소나 — "이직·번아웃 30대 직장인" (시장의 40%)

| 항목 | 값 |
|---|---|
| 연령 | 만 30~39세 |
| 직업 | 직장 3~10년차, 대리~과장급 |
| 가족 상태 | 결혼 전/직후, 자녀 없음 또는 1명 |
| 연소득 | 4,000만~7,000만 원 |
| 결제 능력 | 9,900원 충동 OK · 49,000원 신중 OK · 149,000원 망설임 |
| 핵심 통증 | "지금 직장 계속해야 하나, 이직? 전직? 그런데 나는 뭘 잘하지?" |
| 시도해본 것 | MBTI(무료) → 결과지 받고 "그래서 뭐?" → 좌절 |
| 검색 패턴 | "30대 진로 고민", "이직 결정", "번아웃 회복", "MBTI 한계" |
| 의사결정 시간 | 첫 진단 결제까지 2~4주 (블로그 여러 편 정독 후) |

**보고서 근거:** §4-1 — "30대 직장인이 핵심 구매층 40%"
**설립자 검증:** 사장님 결정 — "30대 직장인이 가장 적합하다는 거에 놀랐지만 객관적 분석을 받아들임"

### 1.2 보조 페르소나 (현재는 우선순위 아님, 추후)

| 페르소나 | 비중 | 비고 |
|---|---|---|
| 20대 후반 구직자 | 25% | 결제력 낮음, 자소서 시장과 경쟁 |
| 40대 임원·관리자 | 15% | ARPU 높지만 영업 사이클 김, B2B로 가야 효과 |
| 종교/교회 그룹 | 10% | ARPU 매우 낮음, 우선순위 후순위 |

→ **이번 12개월은 30대 직장인 한 점에만 집중.** 보조 페르소나는 자연 유입은 받되, 마케팅·콘텐츠는 30대 직장인에게 100% 정렬.

## 2. 가치 제안 (Value Proposition)

### 2.1 우리가 진짜 주는 것 (Promise)

> **"MBTI 결과지를 받고 '그래서 뭐?'를 경험한 30대에게, '한 점'을 정해서 21일을 살아내게 만든다."**

| 단계 | 가치 |
|---|---|
| 9,900원 진단 | "내가 어떤 사람인지" → 사명 한 줄 + 강점 + 21일 실행 워크북 |
| 49,000원 다이어리 | "그 한 점을 매일 살아내는 도구" (제작 중) |
| 149,000원 3개월 코칭 | 설립자(평생교육사) 직접 코칭 + 외부 전문기관 연계 |
| 커뮤니티 (가격 미정) | 같은 단계의 30대들이 모인 팀 프로젝트형 소그룹 |

### 2.2 우리만 줄 수 있는 것 (Moat — 모트)

보고서가 분석한 64개 디지털 채널 누구도 갖지 못한 것:

1. **설립자 김영식이 직접 운영** — 평생교육사 자격 + 1:1 코칭 가능
2. **오프라인 발영업 + 입소문** — 차가운 디지털 마케팅 대신 따뜻한 신뢰 기반
3. **9,900원이라는 자기선별 장벽** — 진성 사용자만 들어옴 (무료 사용자의 끝없는 진상 회피)
4. **한국 30대 직장인 컨텍스트** — 갤럽의 영문 결과지, 16personalities의 글로벌 일반론과 차별

→ **이 4가지가 모두 다른 자기계발 플랫폼(클래스101·탈잉·트레바리·MKYU)이 못 만든 차별점.**

### 2.3 우리가 안 주는 것 (Anti-Promise)

`docs/promise-and-limits.md` 참조 — 줄 수 없는 것을 명시적으로 박아둠.

## 3. 가격 사다리 (Price Ladder)

| Tier | 상품 | 가격 | 역할 | 상태 |
|---|---|---|---|---|
| **1** | 76문항 진단 + PDF 리포트 + 21일 워크북 | **9,900원** | **미끼 + 자기선별** | ✅ 라이브 |
| **2** | 자기경영 다이어리 (인쇄/디지털) | **49,000원** | **주력 상품** (메인 수익원) | ⏳ 제작 중 |
| **3** | 21일 동행 프로그램 | **39,900원** | 진단 직후 행동화 | ⏳ 검토 중 |
| **4** | 3개월 1:1 코칭 (설립자 직접) | **149,000원** | 고가치 고객 | ⏳ 수동 신청 |
| **5** | 외부 전문기관 연계 | 별도 | B2B 또는 임상 영역 | ⏳ 파트너 발굴 |

### 핵심 결정 사항
- **무료 진단 X** — 자기선별 효과 + 16personalities 대비 신뢰도 + 카카오 채널이 이미 무료 진입 경로 역할
- **9,900원은 미끼**, 다이어리(49,000원)가 주력
- **다이어리 제작 완료 전에는 사이트 UI/카피 재배치 안 함** (PR-B/C는 다이어리 실물 준비 후)

## 4. 채널 (Channels)

### 4.1 채택 채널

| 채널 | 역할 | 비용 | 우선순위 |
|---|---|---|---|
| **블로그 SEO** (Google + Naver) | 30대 직장인 검색 유입 | 거의 0 (시간만) | ★★★★★ |
| **유튜브 롱폼** | 신뢰 형성, 진단 데모 | 시간만 (자체 제작) | ★★★★ |
| **카카오 채널** | 무료 진입 funnel + 결제 망설이는 방문자 락인 | 거의 0 | ★★★★ |
| **오프라인 발영업** | 설립자 직접 미팅, 강의, 모임 참석 | 시간만 | ★★★★★ |
| **입소문** | 만족 고객의 자발적 추천 | 0 | ★★★★★ (목표) |

### 4.2 명시적으로 배제한 채널 (시장검증 보고서 §6 + §8 근거)

| 채널 | 배제 이유 |
|---|---|
| Meta(인스타·페이스북) 페이드 광고 | RPM 1.3원 실측, CPC 상승세, CAC 폭주 위험 (클래스101 패턴) |
| 네이버 검색광고 (파워링크) | 브랜드 키워드 검색량 부족, 비-브랜드 CPC 과대 |
| 인스타 카드뉴스 제작 | 과거 22,776뷰 → 3건 결제 실측 |
| 크몽·탈잉·클래스101 입점 | 수수료 20~60%, 가격 통제권 상실 |
| TikTok·X·Threads | 30대 직장인 결제 페르소나와 정렬 안 됨 |

→ **"하나(블로그+유튜브+카카오+오프라인) 잘되기 전까지는 둘 안 한다"** 원칙.

## 5. 수익 모델 (Revenue Streams)

### 5.1 매출 구조

| 단계 | 단위 매출 | 마진율 추정 | 비고 |
|---|---|---|---|
| 9,900원 진단 | 9,900원 | ~95% (디지털) | 결제 수수료만 |
| 49,000원 다이어리 | 49,000원 | 50~70% (인쇄 시) / 90%+ (디지털) | 제작 후 확정 |
| 149,000원 코칭 | 149,000원 | 80%+ (설립자 시간이 주 비용) | 시간 제약 = 월 5~10건 한계 |

### 5.2 LTV 가설 (검증 전, 90일 후 측정)

| 가설 | 값 |
|---|---|
| 9,900원만 사는 사람 | 60% (LTV = 9,900원) |
| 9,900원 + 다이어리 | 30% (LTV = 58,900원) |
| 9,900원 + 다이어리 + 코칭 | 10% (LTV = 207,900원) |
| **평균 LTV (가설)** | **약 41,000원** |

### 5.3 측정 지표 (KPI)

| 지표 | 90일 목표 | 의미 |
|---|---|---|
| 진단 결제 (월) | 50건 | 채널 트래픽 검증 |
| 다이어리 부가 결제율 | 30%+ | 가치 사다리 검증 |
| 코칭 부가 결제율 | 10%+ | 고가치 funnel 검증 |
| 카카오 채널 친구 누적 | 200+ | 유료 안 가는 사용자 락인 |
| 입소문(추천 결제) 비중 | 20%+ | 모트 검증 (만족도) |

### 5.4 Kill Criteria (시장검증 보고서 §결론C-3 적용)

다음 6개월 (2026-11-26까지) 안에 4개 지표 중 **3개 이상 달성 시 본격 확장, 2개 이하면 BM 재설계 또는 철수**:

1. 누적 진단 결제 **300건 이상**
2. 다이어리 부가 결제율 **20% 이상**
3. LTV/CAC ≥ **3 이상** (오가닉 채널이므로 CAC는 거의 시간 비용)
4. 만족 고객 추천 결제 **10건 이상**

> 김병훈 원칙: "신호가 명백히 부정적인데 1~2년 더 버틴 것"이 클래스101·탈잉·트레바리의 공통 실패 패턴.
