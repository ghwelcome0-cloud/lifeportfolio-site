💡 채널 이름은 나중에 바꿀 수 있으니 너무 고민하지 마세요. 일단 `인생포트폴리오`로 시작하면 됩니다.

<a name="part-a-채널-만들기"></a>
## PART A · 채널 만들기 (한 번만, 약 40분)

### A-1. 구글 계정 로그인 ⏱ 3분

☑ ① 크롬(Chrome) 브라우저를 엽니다.
☑ ② 주소창에 `youtube.com` 입력 → Enter.
☑ ③ 오른쪽 위 파란색 **[로그인]** 버튼 클릭.
☑ ④ 회사용 구글 계정으로 로그인. (계정 없으면 [계정 만들기] → 안내대로 진행)

### A-2. "브랜드 채널" 만들기 ⏱ 7분

⚠️ 그냥 채널이 아니라 **"브랜드 계정 채널"** 을 만들어야 합니다. (나중에 여러 명이 같이 관리 가능)

☑ ① 로그인 상태에서 오른쪽 위 **내 프로필 동그라미** 클릭.
☑ ② 메뉴에서 **[설정]** (톱니바퀴 모양) 클릭.
☑ ③ **[채널 추가 또는 관리]** 클릭.
☑ ④ **[채널 만들기]** 버튼 클릭.
☑ ⑤ 채널 이름에 `인생포트폴리오` 입력 → **[만들기]** 클릭.

🎉 축하합니다. 채널이 생겼습니다.

### A-3. 채널이 잘 만들어졌는지 확인 ⏱ 2분

☑ ① 오른쪽 위 프로필 클릭 → **[채널 보기]** 클릭.
☑ ② 내 채널 페이지가 나오면 성공입니다.

<a name="part-b-채널-꾸미기"></a>
## PART B · 채널 꾸미기 (브랜딩, 약 60분)

> 사람은 **"이 채널 믿을 만한가?"** 를 3초 만에 판단합니다. 그래서 첫인상(프로필·배너·소개)이 중요합니다.
> 이 단계의 모든 이미지/문구는 우리 브랜드 자료(`C.브랜드_메시지북`)에 맞춰 만듭니다.

### B-1. 프로필 사진(로고) 만들기 ⏱ 15분

☑ ① 프로필 사진은 **로고**를 씁니다. 로고가 있으면 그걸 쓰고, 없으면 만듭니다.
☑ ② 로고가 없다면 → 저(AI)에게 "인생포트폴리오 유튜브 로고 만들어줘"라고 요청하시면 만들어 드립니다.
   - 추천 스타일: 깔끔한 원형, 브랜드 색(차분한 네이비/그린), "사명·나침반" 느낌
☑ ③ 만든 이미지를 PC에 저장.

**업로드 방법:**
☑ ④ 내 채널 → **[채널 맞춤설정]** 버튼 클릭.
☑ ⑤ **[브랜딩]** 탭 클릭 → **사진** 항목에서 **[업로드]** → 저장한 로고 선택 → **[게시]**.

💡 권장 크기: 800×800 픽셀, 정사각형. 2MB 이하.

### B-2. 배너(채널 아트) 만들기 ⏱ 15분

배너 = 채널 맨 위에 길게 깔리는 큰 그림입니다.

☑ ① 배너에는 **딱 한 문장**만 넣습니다. 예:
   - `방향은 아는데, 실행이 안 되는 당신에게` (← JTBD 카피, O.제안서 권고)
   - `Only One 인생 설계도 · 인생포트폴리오`
☑ ② 배너 이미지가 없으면 저에게 "유튜브 배너 만들어줘"라고 하시면 위 문구로 만들어 드립니다.
☑ ③ **[채널 맞춤설정] → [브랜딩] → 배너 이미지 → [업로드]** → **[게시]**.

💡 권장 크기: **2560×1440 픽셀.** 중요한 글자는 가운데 1235×338 안에 넣어야 모든 기기에서 잘립니다.

### B-3. 채널 소개 글 쓰기 ⏱ 10분

☑ ① **[채널 맞춤설정] → [기본 정보]** 탭 클릭.
☑ ② **설명** 칸에 아래를 복사해서 붙여넣기 (자유 수정 가능):

```
방향은 아는데, 실행이 안 되는 당신을 위한 채널.

인생포트폴리오는 당신 안에 숨겨진 사명과 강점을 발견해
'살아있는 인생 설계도'로 만드는 자기경영 시스템입니다.

번아웃, 이직 고민, "MBTI 다음은 뭐지?" 막막함 —
이 채널이 당신의 첫 한 걸음을 함께 설계합니다.

👉 나의 사명·강점 진단 (3주 실행 설계도): lifeportfolio.co.kr
📩 문의: (이메일 주소)
```

☑ ③ **링크** 항목에 홈페이지 주소 추가:
   - 표시 이름: `인생포트폴리오 진단`
   - URL: (UTM 링크 — 아래 B-4 참조)
☑ ④ **[게시]** 클릭.

### B-4. ⭐ 가장 중요 — 링크에 UTM 붙이기 ⏱ 5분

⚠️ **그냥 `lifeportfolio.co.kr`을 붙이면 안 됩니다.** 유튜브에서 몇 명이 왔는지 측정이 안 됩니다.

☑ ① 우리 홈페이지의 `/utm-builder.html` 페이지에 접속.
☑ ② 아래처럼 입력:
   | 칸 | 입력값 |
   |---|---|
   | source | `youtube` |
   | medium | `social` |
   | campaign | `yt_channel_link` |
   | content | `channel_about` |
☑ ③ 생성된 긴 주소를 복사해서 B-3의 링크에 붙여넣기.

💡 영상마다 다른 UTM을 쓰면 "어느 영상이 고객을 데려왔는지"까지 알 수 있습니다. (PART E에서 다시 설명)

<a name="part-d-영상-만들기"></a>
## PART D · 영상 만들기 (대본 → 촬영/AI → 편집)

> 두 가지 길이 있습니다. **길 1(직접 촬영)** 또는 **길 2(AI로 제작)**. 편한 것을 고르세요.

### D-1. 대본 쓰기 (모든 영상의 시작) ⏱ 숏츠 10분 / 롱폼 30분

**숏츠 대본 공식 (4단계, 이대로):**
```
① 후킹 (0~3초)   : "30대 번아웃, 사실 게으름이 아닙니다."   ← 충격/공감
② 문제 (3~15초)  : "왜냐하면 진짜 원인은 '방향 상실'이거든요."
③ 해결 (15~50초) : "이 3가지만 확인하면 방향이 보입니다. 첫째…"
④ CTA (50~60초)  : "전체 내용은 채널에 있어요. 구독하고 보세요."
```

**롱폼 대본 공식 (5단계):**
```
① 인트로 (0~30초)  : 시청자 문제 공감 + "오늘 이거 알려드려요"
② 본론1 (1~4분)    : 문제의 진짜 원인
③ 본론2 (4~8분)    : 구체적 해결 방법 3가지
④ 사례/증거 (8~10분): 통계·후기 ("번아웃 75.3%")
⑤ CTA (마지막 30초): "더 깊이 알고 싶다면 → 설명란 링크"
```

⚠️ **CTA 철칙**(시장조사 3-2): 숏츠에서는 "구독", 롱폼에서만 "링크 클릭"을 유도하세요.
⚠️ **메시지 순서**(O.제안서): 철학(Why)을 먼저 들이대지 마세요. **결과 → 방법 → 그다음 철학.**

### D-2. 길 1 — 직접 촬영 ⏱ 영상당 20~40분

☑ ① 스마트폰을 **세로(숏츠)** 또는 **가로(롱폼)** 로 고정 (삼각대 추천, 1만 원대).
☑ ② 창문 빛을 얼굴 앞쪽으로 (역광 금지).
☑ ③ 대본을 보면서 또박또박 말하기. 틀리면 그 문장만 다시.
☑ ④ 소리가 제일 중요 → 조용한 방에서, 핸드폰 마이크 가까이.

### D-3. 길 2 — AI로 제작 (얼굴 안 나와도 됨) ⏱ 영상당 15~30분

대표님이 직접 안 나와도 됩니다. 저(AI)에게 요청하면 만들어 드립니다.

☑ ① **AI 음성(나레이션)**: "○번 대본으로 차분한 30대 남성 목소리 나레이션 만들어줘" → 음성 파일 제공
☑ ② **썸네일/이미지**: "○번 영상 썸네일 만들어줘" → 이미지 제공
☑ ③ **짧은 영상 클립**: "번아웃 직장인 분위기 5초 영상 만들어줘" → 영상 클립 제공
☑ ④ 이 재료들을 편집 앱에서 합치기 (D-4).

💡 자막 + AI음성 + 이미지 슬라이드만으로도 훌륭한 숏츠가 됩니다. (요즘 가장 흔한 형식)

### D-4. 편집하기 (무료 앱) ⏱ 숏츠 20분 / 롱폼 60분

**추천 무료 편집 도구:**
| 도구 | 용도 | 난이도 |
|---|---|---|
| **CapCut(캡컷)** | 숏츠·롱폼 다 됨, 자막 자동 | ⭐ 쉬움 (강력 추천) |
| **VLLO(블로)** | 모바일 편집 | ⭐ 쉬움 |
| **유튜브 자체 편집** | 간단 컷편집 | ⭐ 쉬움 |

**CapCut 기본 순서 (숏츠 기준):**
☑ ① CapCut 앱 설치 → **[새 프로젝트]**.
☑ ② 찍은 영상 또는 AI 이미지/클립 불러오기.
☑ ③ 불필요한 부분 자르기 (영상 클릭 → 분할 → 삭제).
☑ ④ **[텍스트] → [자동 캡션]** → 말소리가 자동으로 자막이 됩니다. (필수! 무음으로 보는 사람 많음)
☑ ⑤ 첫 3초에 큰 글씨로 후킹 문장 넣기.
☑ ⑥ 배경음악 작게 (음성 안 묻히게).
☑ ⑦ **[내보내기]** → 1080p → PC/폰에 저장.

⚠️ **숏츠는 세로 9:16, 롱폼은 가로 16:9** 로 만드세요.

<a name="part-e-업로드"></a>
## PART E · 업로드 (제목·설명·태그·썸네일)

> 영상은 다 만들었습니다. 이제 올립니다. **여기서 제목·썸네일이 조회수의 80%를 결정합니다.**

### E-1. 영상 올리기 ⏱ 10분

☑ ① 유튜브 오른쪽 위 **[+ 만들기]** → **[동영상 업로드]** 클릭.
☑ ② 만든 영상 파일 선택.

### E-2. 제목 쓰기 (검색 + 호기심) ⏱ 5분

☑ ① 제목 앞쪽에 **검색되는 단어**를 넣습니다.
   - ❌ 나쁜 예: "오늘의 이야기"
   - ✅ 좋은 예: "30대 번아웃, 게으름이 아닙니다 (진짜 원인 3가지)"
☑ ② 40자 이내, 숫자/괄호 활용.

### E-3. 설명란 쓰기 ⭐ 링크는 첫 줄에! ⏱ 5분

⚠️ **링크는 반드시 설명란 맨 첫 줄**에 넣으세요. (외부 데이터: 최적화 시 CTR +40%)

```
👉 나의 사명·강점 진단하기 (방황을 끝내는 첫 3주 설계도):
(여기에 UTM 링크)

────────────
이 영상에서 다루는 내용:
00:00 30대 번아웃의 진짜 원인
02:30 게으름과 방향상실의 차이
05:00 회복을 위한 3단계 설계

번아웃, 이직, 자기경영이 고민이라면 구독해 주세요.
매주 '인생 설계' 콘텐츠가 올라옵니다.

#번아웃 #30대 #자기계발 #인생포트폴리오 #자기경영
```

### E-4. 영상마다 다른 UTM 링크 만들기 ⏱ 3분

☑ ① `/utm-builder.html` 접속.
☑ ② 입력:
   | 칸 | 값 |
   |---|---|
   | source | `youtube` |
   | medium | `video` (롱폼) 또는 `social` (숏츠) |
   | campaign | `yt_evergreen_2026q3` |
   | content | 영상 고유이름 (예: `burnout_direction_01`) |
   | term | 타겟 (예: `p1_burnout`) |
☑ ③ 생성된 링크를 설명란 첫 줄 + 고정 댓글에 붙여넣기.

💡 영상마다 `content`만 바꾸면 "어느 영상이 사람을 데려왔는지" 정확히 알 수 있습니다.

### E-5. 썸네일 올리기 (롱폼 필수) ⏱ 10분

☑ ① 썸네일 = 영상의 표지. **큰 글씨 6글자 이하 + 얼굴/강한 이미지.**
☑ ② 저에게 "○번 영상 썸네일 만들어줘"라고 하면 만들어 드립니다.
☑ ③ 업로드 화면에서 **[썸네일 업로드]** → 선택.

💡 좋은 썸네일 문구 예: `번아웃 ≠ 게으름`, `MBTI 다음은?`

### E-6. 고정 댓글 + 게시 ⏱ 3분

☑ ① **[게시]** 또는 **[예약]** (특정 시간 발행).
☑ ② 발행 후 → 내 영상에 댓글로 UTM 링크 작성 → 내 댓글 오른쪽 **⋮ → [고정]**.

💡 **발행 추천 시간**: 평일 저녁 7~9시, 일요일 저녁 (번아웃·이직 감정이 올라오는 시간 — 노출전략 3단계).

<a name="part-f-운영"></a>
## PART F · 운영 (매일·매주 루틴)

> 2026 알고리즘은 **"꾸준함"** 을 보상합니다(시장조사). 완벽한 1개보다 평범한 꾸준함이 이깁니다.

### F-1. 매일 할 일 (10분) ☑

☑ ① 어제 올린 영상 **댓글 확인 + 답글** (참여율↑ = 노출↑).
☑ ② 다른 자기계발 영상 1~2개 보고 댓글 (우리 분야 알고리즘 학습 + 노출).

### F-2. 매주 할 일 (반나절, 배치 제작) ☑

> ⚠️ 매일 찍으면 지칩니다(시장조사 리스크 #3). **하루에 몰아서** 1주일치를 만드세요.

☑ ① **월요일**: 이번 주 영상 5~6개 **대본 한꺼번에** 작성 (저에게 요청).
☑ ② **화요일**: 촬영 또는 AI 음성/이미지 한꺼번에 생성.
☑ ③ **수요일**: 편집 몰아서 (숏츠 5개 ≈ 2~3시간).
☑ ④ **목~일**: 하루 1개씩 **예약 발행**.

### F-3. 매월 할 일 (2시간) ☑

☑ ① 한 달 숫자 리뷰 (PART G).
☑ ② 잘된 영상 TOP3 → 비슷한 주제로 더 만들기.
☑ ③ 키 콘텐츠(브랜드 영상) 1개 제작.
☑ ④ 잘된 영상을 블로그·블라인드·스레드에 재배치 (자산 재활용).

### F-4. 콘텐츠 자산화 루틴 (1영상 = 5채널) ☑

영상 1개를 만들면 5곳에 씁니다 (시장조사 2-3):
☑ 유튜브(원본) → ☑ 네이버 블로그(임베드+글) → ☑ 블라인드(캡처+후기형) → ☑ 스레드/인스타(릴스) → ☑ 카카오 알림톡(구매자에게 VOD)

<a name="part-g-측정과-개선"></a>
## PART G · 측정과 개선 (숫자 보는 법)

> ⚠️ **첫 90일은 "매출"이 아니라 "자산"을 봅니다.** (시장조사 5장)

### G-1. 매주 볼 숫자 4개 (유튜브 스튜디오에서) ☑

☑ ① 유튜브 → 오른쪽 위 프로필 → **[YouTube 스튜디오]** → **[분석]**.

| 숫자 | 어디서 보나 | 좋은 기준 |
|---|---|---|
| **조회수** | 분석 첫 화면 | 늘고 있으면 OK |
| **구독자 증가** | 분석 → 개요 | 주 단위로 +면 OK |
| **시청 지속률** | 분석 → 참여도 | **30% 이상**이면 좋음 |
| **노출 클릭률(CTR)** | 분석 → 콘텐츠 | **4~10%**면 좋음 (썸네일·제목 효과) |

### G-2. 진짜 중요한 숫자 — 홈페이지 방문 (GA에서) ☑

☑ ① 구글 애널리틱스(GA4) → 획득 → `utm_source = youtube` 필터.
☑ ② "유튜브에서 홈페이지로 몇 명이 왔나" 확인.
☑ ③ 이 숫자가 **0이 아니면** 유튜브가 일하기 시작한 것입니다.

### G-3. 개선 규칙 (숫자가 나쁠 때) ☑

| 문제 | 원인 | 고칠 것 |
|---|---|---|
| 조회수 낮음 | 썸네일·제목 약함 | 제목 앞에 검색어, 썸네일 6글자 |
| 조회수는 높은데 시청 지속률 낮음 | 첫 3초 후킹 약함 | 인트로 다시 (충격/공감 문장) |
| 시청은 잘 되는데 클릭(링크) 없음 | CTA 약함 | 설명 첫 줄 링크 + 고정 댓글 |
| 클릭은 있는데 결제 없음 | 홈페이지 문제 | 랜딩 카피를 JTBD로 (O.제안서) |

### G-4. 90일 목표 (현실적, 검증된 수치) ☑

| 시점 | 목표 |
|---|---|
| D+14 | 채널 완성 + 영상 5개 |
| D+45 | 숏츠 누적 조회 1만+, 구독 100+ |
| D+75 | 검색 유입 20%+, 구독 500+ |
| D+90 | 유튜브→홈 방문 월 300+ 측정 |

| UTM 링크 | `/utm-builder.html`에서 직접 (PART E-4) |

### 부록 2. ⚠️ 솔직하게 — 제가 못 하는 것

- **유튜브에 자동 업로드는 제가 직접 못 합니다.** (보안상 대표님 계정 접근 불가)
- 대신 ① **재료(대본·음성·썸네일·영상)는 전부 만들어 드리고**, ② 원하시면 **YouTube Data API 자동 업로드 스크립트**를 만들어 드릴 수 있습니다 (대표님이 1회 인증 후 명령어로 업로드).
- 가장 현실적인 길: **재료는 AI가 만들고, 업로드 클릭만 대표님이** 하시는 것입니다 (영상당 5분).

### 부록 3. 절대 하지 말 것 (계정 정지 위험)

❌ 조회수·구독자 사는 것 (정지)
❌ 남의 영상 그대로 올리기 (저작권)
❌ 제목·썸네일과 다른 내용 (어그로 → 알고리즘 페널티)
❌ 숏츠에서 "지금 결제하세요" 직접 판매 (시청자 이탈 + 전환율 0.1~0.5%, 효과 없음)

### 부록 4. 우선순위 다시 (잊지 마세요)

> 유튜브는 **메인이 아니라 보조 + 복리 자산**입니다.
> 메인 매출: **메타 광고(60%)** · 최우선 무료: **블라인드(0원)** · 복리 자산: **유튜브.**
> 셋을 **함께** 굴리되, 유튜브는 "조급해하지 않고 90일 쌓는다"는 마음으로 가세요.
