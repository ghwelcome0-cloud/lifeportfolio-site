## 30초 핵심

- **어디(길목)**: 숏츠(노출) + 유튜브 검색(고의도 전환). 우리 고객은 "퇴사·번아웃·MBTI 다음"을 검색한다.
- **어느 결(전환)**: 숏츠 → 롱폼(신뢰) → 설명란 첫 줄 링크 → lifeportfolio.co.kr 진단(₩9,900) → 21일 → 다이어리.
- **솔직한 진실**: 유튜브는 메인 엔진이 아니라 **24시간 일하는 무료 영업사원(복리 자산)**. 첫 90일 KPI는 매출이 아니라 **노출·구독·방문**.
