## 🎓 핵심 원칙 (Non-Negotiables)

1. **모든 트래픽 URL에 UTM** — `/utm-builder.html` 통과한 것만 사용
2. **모든 카피에 "절대가치 + 근거"** — ₩9,900은 가격이 아니라 "점심값으로 10년 진단"
3. **모든 진단 완료자에 디폴트 옵션 노출** — 21일 코칭이 "다음 단계"로 자연 노출
4. **모든 광고에 결과 측정 약속** — 24h ROAS / CTR / CVR 매일 기록
5. **카피는 감정 후킹 + 결제는 이성적 근거** — 후기·통계·가격·실행성 4종 LP 노출
