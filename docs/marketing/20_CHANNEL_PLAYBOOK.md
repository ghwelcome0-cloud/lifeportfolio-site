## 2. 5대 노출 포인트 (NP) × 페르소나 매트릭스

|  | NP1 LLM SEO | NP2 유튜브 PPL | NP3 블라인드 시드 | NP4 트렌드 코리아 타겟 | NP5 메타 감정 후킹 |
|---|---|---|---|---|---|
| **P1 번아웃 30대** | ✅ 핵심 | ✅ 핵심 | ✅ **최우선** | ✅ | ✅ 핵심 |
| **P2 이직 준비생** | ✅ | 🟡 보조 | ✅ **최우선** | 🟡 | ✅ |
| **P3 갓생 추구자** | 🟡 | ✅ | ✗ | 🟡 | ✅ 핵심 (인스타/스레드) |
| **P4 MBTI 졸업자** | ✅ | ✅ **쇼츠** | ✗ | 🟡 | ✅ |

**우선순위 결론**: NP3(블라인드 시드) + NP5(메타 감정 후킹)이 P1·P2 합산 도달 최대. **D+30까지 이 두 채널에 광고비/리소스의 70% 집중**.

## 5. UTM 컨벤션 (모든 카피에 적용)

```
utm_source=    {blind|naver|youtube|instagram|threads|kakao|meta}
utm_medium=    {seed|seo|ppl|shorts|organic|paid|alert}
utm_campaign=  s1_w{1-12}_{persona}    예: s1_w3_p1
utm_content=   {creative_id}_{angle}   예: cr01_aifear
utm_term=      {keyword_or_creator}
```

**예시 URL**:
```
https://lifeportfolio.co.kr/lead?
  utm_source=meta&
  utm_medium=paid&
  utm_campaign=s1_w3_p1&
  utm_content=cr01_aifear&
  utm_term=ai_career
```

`/utm-builder.html` 통해 생성. 모든 광고/시드 URL은 **반드시 utm-builder로 생성한 것만 사용**.

## 7. 광고 카피 윤리 체크리스트

모든 광고/시드 게시 전 다음 확인:

[ ] 절대 단정형 금지: "최고", "유일한", "단 하나" 미사용
[ ] 비교 폄훼 금지: MBTI/갤럽을 "쓸모없다" 표현 X
[ ] 페르소나 고지: 사용된 사례는 "가상의 페르소나" 또는 동의받은 실제 사례만
[ ] 통계 출처 표기: 70%/69.5%/83.2% 등은 본문에 출처 표시 (블로그 글로 연결)
[ ] 의료/심리 진단 금지: "병명/치료" 표현 X — "자기경영/방향성" 톤
[ ] 9,900원 명시: 회피·혼동 카피 금지 (가격 정확히 노출)
