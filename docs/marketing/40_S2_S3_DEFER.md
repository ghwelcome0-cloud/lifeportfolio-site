## 4. D+60 활성화 체크리스트

S2/S3는 다음 조건 모두 충족 시 활성화:

[ ] S1 월 결제 800건+ 도달 (D+60)
[ ] S1 21일 코칭 CVR 8%+ 검증
[ ] S1 LTV/CAC ≥ 3.0
[ ] 운영 인력 가용성 확보 (S2 영업 + S3 인프라 동시 진행 가능?)
[ ] 광고비 / 운영비 추가 예산 확보 (S1 매출의 30% 재투자 가능?)

위 5개 모두 ✅이면 D+61부터 S2 본격, S3 인프라 셋업 시작.
