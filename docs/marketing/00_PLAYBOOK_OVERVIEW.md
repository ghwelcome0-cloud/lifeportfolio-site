> **버전**: v1.0 · **작성일**: 2026-05-28 · **분류**: Internal — Execution
>
> **출처 문서**:
> - `시장조사 3단계 — 고객 노출 전략 보고서` (넛지/절대가치 5대 포인트)
> - `시장조사 4단계(최종) — 3대 신규 수요 시나리오 시장 진입 전략` (S1/S2/S3 P1·P2·P3)

## 0. North Star — 한 문장 KPI

```
12M 누적 매출 ₩7억 ~ ₩10억
= S1(₩9,900 ARPU × 월 5,000건 × 12개월) + S2(교회 30개 × 50~300만) + S3(베타 1,000건 × $8.99)

= 디지털 캐시카우(P1) + B2B 고LTV(P2) + 글로벌 옵션가치(P3)
```

**90일 안에 검증해야 하는 것 단 하나**:
> "9,900원 진단 LP의 월 5,000건 전환은 가능한가?"
> (= 월 ARPU 1,000명 × 4.95 = 약 ₩495만 → 12M 환산 ₩5.94억의 P1 가설)

이 한 가지 가설이 깨지면 S2/S3 전체 로드맵이 재설계됩니다. **따라서 90일은 100% S1에 집중**.

## 1. 두 보고서를 합치면 보이는 그림

3단계 (HOW) + 4단계 (WHERE) = **단일 실행 좌표계**

```
                  ┌─────────────────────────────────────────────┐
                  │   4단계: WHERE — 3대 시장 (S1/S2/S3)             │
                  │   = 어느 시장에서 매출을 만드는가                       │
                  └──────────────────┬──────────────────────────┘
                                     │
                                     │ 90일은 S1만
                                     ▼
                  ┌─────────────────────────────────────────────┐
                  │   3단계: HOW — 5대 노출 포인트 + 4 페르소나 + 5단계 퍼널 │
                  │   = S1에서 어떻게 ₩9,900 결제를 만드는가                │
                  └─────────────────────────────────────────────┘
```

## 3. 90일 의사결정 트리 (한 페이지)

```
START
  │
  ├─ Phase 1 (Day 1~30) — S1 집중 런칭
  │   목표: 유입 1만 / 결제 300건 / ROAS 검증
  │   ↓
  │   ✅ ROAS ≥ 350% & 결제 CVR ≥ 3% ?
  │       ├─ YES → Phase 2 (광고 스케일업)
  │       └─ NO  → STOP. Phase 2 보류, 카피·LP 재설계 (60일 가용)
  │
  ├─ Phase 2 (Day 31~60) — S1 스케일 + S2 영업
  │   목표: 월 결제 500건+ / B2B 시범 5건
  │   ↓
  │   ✅ 월 결제 500건+ & 21일 코칭 전환율 10%+ ?
  │       ├─ YES → Phase 3 (S2 본격 / S3 인프라)
  │       └─ NO  → 21일 코칭 디폴트 옵션 UX 재설계
  │
  └─ Phase 3 (Day 61~90) — S2 본격 / S3 셋업
      목표: B2B 계약 5건 / S3 베타 1,000건 결제
      ↓
      D+90 — Go/No-Go 결산
      ├─ S1 ARR ≥ ₩2억 환산 → 12M 시뮬레이션 GO
      └─ S1 ARR <  ₩1억 환산 → 모델 재검토
```

## 7. 절대 원칙 (Non-Negotiables)

1. **모든 트래픽 URL에 UTM**: `/utm-builder.html`로 생성한 URL만 광고/시드에 사용
2. **모든 카피에 "절대가치 + 근거"**: ₩9,900은 가격이 아니라 "점심값으로 10년 커리어 진단"
3. **모든 진단 완료자에 디폴트 옵션 노출**: 21일 코칭이 "다음 단계"로 자연스럽게 보여야 함
4. **모든 광고에 결과 측정 약속**: 24h ROAS / CTR / 결제 CVR을 매일 기록
5. **카피는 감정 후킹 + 결제는 이성적 근거**: 후기·통계·가격·실행성 4종 모두 LP에 노출
