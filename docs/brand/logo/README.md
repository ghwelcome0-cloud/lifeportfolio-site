## 디자인 의미 (출원서 "상표의 설명"에 활용 가능)

- **L (슬랩세리프)**: Life / Lifeportfolio. 건축적이고 견고한 형태 = "삶을 설계한다"는 구조적 신뢰감.
- **골드 점(별·이정표)**: 인생의 목표·기준점. 흩어진 한 점.
- **골드 궤도(arc)**: 그 점을 향해 잇는 길 = 76문항 진단으로 흩어진 요소를 하나의 방향(포트폴리오)으로 연결.
- **딥 슬레이트 블루(#4A6984)**: 신뢰·깊이·사색.
- **골드(#C49E45)**: 가치·완성·프리미엄.
- **크림(#F5EEDC)**: 따뜻함·정직·사람 중심.
- **태그라인 "삶을 설계하다"**: 서비스 본질을 한 문장으로.

## 상표 출원 시 권장 사항 (참고)

1. **출원 형태**: "도형 + 문자 결합상표"로 출원하면 식별력이 가장 강함(업체 피드백과 일치).
2. **흑백본 함께 제출**: 색채를 한정하지 않으려면 흑백본이 권리범위에 유리할 수 있음 — 변리사와 상의.
3. **지정상품/서비스(예상)**: 35류(광고·사업), 41류(교육·코칭), 42류(소프트웨어/SaaS), 44류(상담) 등 — 변리사와 최종 확정.
4. **태그라인 "삶을 설계하다"**: 식별력이 약한 표어성 문구이므로, 핵심 식별력은 "인생포트폴리오 + L 도형"에 두는 것이 안전.
5. PNG는 래스터다. 출원·인쇄 고품질이 필요하면 **벡터(AI/SVG) 변환**을 별도로 진행 권장(요청 시 제작).
