## 운영 팁 (릴스 전용)

1. **릴스는 도달(신규 노출)에 가장 강력** — 캐러셀(저장·신뢰)과 역할 분담. 둘 다 운영하세요.
2. **첫 3초가 생명**: 후킹 장면(0~3.4초)에서 멈추게 만드는 게 핵심. 이미 강한 질문으로 시작.
3. **무음 시청 대응**: 자막이 화면에 번인되어 있어 소리 꺼도 전달됨(인스타 사용자 대다수 무음 시청).
4. **음악 교체 권장**: 인스타 앱 내에서 업로드 시 **인기 오디오/트렌딩 음원**으로 바꾸면 도달이 더 늘어납니다. (지금 BGM은 무음 대체용 안전장치)
5. **커버 지정**: 업로드 시 1번 후킹 장면을 커버로 지정 → 프로필 그리드에서도 후킹 유지.
6. **재게시 주기**: 같은 영상도 2~3주 후 다른 캡션/음원으로 재활용 가능.
