## 📝 개정 초안 본문 (변경 부분만)

### §1. 처리 목적 (변경 후)

> 인생포트폴리오 검사 응답 수집·리포트 제공, 결제 및 환불 처리, 고객문의 응대, 서비스 품질 개선 및 통계 분석, **그리고 이용자가 명시적으로 동의한 경우에 한해 행동 분석을 통한 UX 개선**.

**변경 근거**: 개보법 §15-1 — 수집·이용 목적을 명확히 고지.

### §2. 처리하는 개인정보 항목 (변경 후)

**필수**: 이름(또는 닉네임), 이메일, 결제정보(승인내역, 주문/거래기록)
**선택**: 검사 응답 데이터(문항별 선택·기술)
**자동 수집**: 접속 로그, IP, 브라우저/기기 정보, Firebase 인증 과정에서 생성·검증되는 인증 토큰 및 로그인 기록
**선택 동의 시 자동 수집** (신규 추가):
- **마우스 클릭·이동·스크롤 패턴** (행동 분석용)
- **페이지 체류 시간 및 이동 경로**
- **세션 영상 기록** (텍스트 입력 영역은 자동 마스킹 처리되어 입력 내용은 수집되지 않음)
- **비식별 사용자 ID** (이름·이메일과 연결되지 않는 익명 식별자)

**수집 경로**: 결제 페이지, Google Forms, 고객센터, Firebase Authentication(구글 계정 로그인), **Microsoft Clarity 스크립트(동의자에 한함)**

**변경 근거**: 개보법 §15-2 — 처리 항목을 구체적으로 고지. 마스킹 처리 명시로 다크 패턴 회피.

### §10. 쿠키 및 자동 수집 도구 (변경 후 — 전면 개정)

> **기존**: "필수 보안/세션 쿠키를 사용할 수 있으며, 선택적 광고 쿠키는 사용하지 않습니다(변경 시 고지)."
>
> **개정 후**:

회사는 다음과 같이 쿠키 및 자동 수집 도구를 사용합니다.

#### 10-1. 필수 도구 (동의 불요)
| 도구 | 제공자 | 목적 | 거부 방법 |
|---|---|---|---|
| 세션 쿠키 | 회사 | 로그인 상태 유지, 보안 | 브라우저 설정에서 거부 가능(거부 시 로그인 불가) |
| Firebase Authentication | Google LLC | 회원 인증 | 회원 탈퇴 시 즉시 삭제 |

#### 10-2. 분석 도구 (필수 — 익명 통계)
| 도구 | 제공자 | 목적 | 거부 방법 |
|---|---|---|---|
| Google Analytics 4 (GA4) | Google LLC | 익명 통계 분석 (페이지뷰, 전환 등) | 브라우저의 GA Opt-out 확장 프로그램 |
| Google Tag Manager (GTM) | Google LLC | 태그 관리 | 브라우저 설정 |

> ⚠️ **IP 익명화 처리됨** (Google IP Anonymization 적용)

#### 10-3. 행동 분석 도구 (선택 동의 필수 — Opt-in)
| 도구 | 제공자 | 목적 | 거부 방법 |
|---|---|---|---|
| **Microsoft Clarity** | Microsoft Corporation | 마우스 동작·스크롤·세션 영상 분석 (UX 개선) | **기본 OFF** — 가입 시 또는 마이페이지에서 명시적 동의 시에만 활성. 언제든 1-click 철회 가능. |

**중요 안내**:
- Microsoft Clarity는 **이용자가 명시적으로 동의한 경우에만 활성화**됩니다.
- 동의하지 않으셔도 서비스 이용에 **어떠한 제한도 없습니다**.
- 동의 후에도 마이페이지에서 **언제든지 1-click으로 철회**할 수 있습니다.
- 수집되는 데이터는 **익명화 처리**되며, 텍스트 입력 영역(비밀번호·결제 정보 등)은 **자동 마스킹** 처리됩니다.
- 수집된 데이터는 **미국 Microsoft 서버**에 저장되며, 자세한 내용은 §6 국외 이전을 참조하세요.

**변경 근거**: 개보법 §22 (동의 분리), §28-8 (국외 이전), 개인정보보호위원회 가이드라인.

### §14. 만 14세 미만 아동의 개인정보 보호 (신설)

> ⚠️ **신설 조항 — 개보법 §22-2 준수**

1. 회사는 만 14세 미만 아동의 개인정보를 원칙적으로 수집하지 않습니다.
2. 회원 가입 시 출생 연도 입력을 통해 만 14세 미만 여부를 확인하며, 만 14세 미만으로 확인된 경우 가입이 제한됩니다.
3. 만 14세 미만 아동이 부득이하게 서비스를 이용해야 하는 경우, 법정대리인의 동의를 받아 가입할 수 있도록 별도 절차를 마련합니다(현재는 미제공).
4. 만 14세 미만 아동의 개인정보가 수집된 사실이 사후 확인되는 경우, 즉시 해당 정보를 파기하고 법정대리인에게 통지합니다.

**변경 근거**: 개보법 §22-2 (만 14세 미만 아동의 개인정보 처리), §39-3 (아동 보호).
