# 홈페이지(index.html) 페르소나 최적화 — 진단 및 개선 설계서

> **목적**: 4개 핵심 페르소나(P1~P4)에 맞춘 랜딩 페이지 최적화.
> **원칙**: "유지할 것"과 "개선할 것"을 객관적 기준으로 분리.
> **작성일**: 2026-05-19
> **상태**: 초안 (사용자 승인 대기)

---

## 0. Executive Summary (3줄 요약)

1. **현 index.html은 "9,900원 진단 단품 판매"에 최적화되어 있다.** 17개 섹션 모두 'Only One 리포트' 1회 결제 전환을 목적으로 한다.
2. **그러나 진짜 시장은 P1~P3 (5~15만 원 시장)이다.** P4 한정판 다이어리(₩89,000~199,000)는 더 윗선이다. 현 페이지는 P2 ("자기성장 마니아", ₩49,000~89,000 시장의 일부 진단 수요자)와 강하게 맞고, P1·P3·P4와는 **부분적**으로만 맞는다.
3. **결론**: 7개 섹션은 유지, 5개 섹션은 부분 수정, 3개 섹션은 페르소나 맞춤 강화, **2개 신규 섹션을 추가**한다. Hero는 페르소나 4분기 자동 호명 구조로 재설계한다.

## 4. 7가지 객관적 판단 기준 (KEEP / MODIFY / ADD / REMOVE)

### KEEP (유지) 기준
다음 중 **2개 이상** 해당하면 유지:
1. 4개 페르소나 모두에게 🟡2 이상 (중립적 가치 제공)
2. 사용자가 명시적으로 유지 요청 (예: Self-check)
3. SEO/구조화 데이터 기반 (JSON-LD, hreflang 등 — 변경 시 리스크)
4. A/B 테스트나 실측 데이터로 효과 검증된 영역

### MODIFY (수정) 기준
다음 중 **1개라도** 해당하면 수정:
1. 특정 페르소나(P2)에게만 🟢3, 다른 페르소나는 🟠1 (편중)
2. 본 페이지의 단일 목적("9,900원 진단")을 강화하지만 **"5~15만 원 다이어리 진입"** 신호를 막고 있음
3. 페르소나 호명이 추상적("당신")에 머물러 P1/P3가 자기 이야기로 인식하지 못함

### ADD (신규) 기준
1. 현재 어디에도 없는 페르소나 진입로 (예: P1 신앙 진입로 / P4 수집가 진입로)
2. 다이어리 사전 신청 의향 캡처 장치 (현재는 Journey 섹션의 안내만 있음)
3. 4분기 자동 호명 (Hero 1뷰에서 P1~P4가 자기 자리 알아봄)

### REMOVE (삭제) 기준 — *현재 적용 없음*
- 객관적으로 페르소나 진입을 **방해**하는 요소가 발견되지 않음 (현 페이지는 잘 만들어져 있다)
- 따라서 본 1차 개선에서는 **REMOVE 0건**. "덜어내기"보다 "정확히 호명하기"가 핵심.

## 6. Hero 재설계안 (Before/After)

### Before (현재)
```
[Eyebrow] ★ Only One Report · 사람마다 단 하나의 결과
[H1]      당신 안에 이미 답이 있습니다.
          그 답을 당신의 길로, 또 누군가의 길로 잇습니다.
[Sub]     — 발견하고, 살아내고, 남기는 한 권의 인생 설계도
[Lead]    15분, 76문항. ...
[Chips]   ✨ Only One / ⏱ 15분 / ⚡ 자동 / 🎯 첫 행동
[Price]   ₩9,900 (정가 19,900 강조)
[CTA]     9,900원으로 첫 인생 설계도 받기
[Card]    오른쪽 3단 카드 (유형 아닌 한 사람 / 첫 걸음 / 양식)
```

**문제점**:
- ₩9,900 가격이 Hero에 너무 일찍 등장 → **P3·P4가 "내 이야기 아니구나" 빠르게 이탈**
- 4페르소나 호명이 "당신"에 머묾 (추상)
- 진단(9,900) ↔ 다이어리(89,000~199,000)의 **이중 진입로**가 안 보임

### After (제안)
```
[Eyebrow] ✦ 인생포트폴리오 · 발견하고, 살아내고, 남기는 한 권
[H1]      방향은 아는데, 살아내지지 않는 분께
[Sub]     사명·강점·실행이 한 줄로 정리되는 곳
[Lead]    검사 결과가 책상 서랍에서 잠들지 않게 —
          76문항 15분 진단부터, 매일의 다이어리까지.
[Chips]   ✨ 76문항 / ⏱ 15분 / 🎯 첫 행동 / 📔 다이어리 (Coming)
[Dual-CTA] [₩9,900 진단 시작]    [다이어리 출시 알림 받기]
           ↑ 즉시 행동              ↑ P3·P4 캡처
[Card]    오른쪽: "당신은 어디에 가까우신가요?" — 4 분기 미리보기
          (Persona Routes 섹션으로의 anchor 링크)
```

**변경 핵심**:
1. **₩9,900을 Hero에서 빼고**, Dual-CTA로 두 갈래(₩9,900 진단 / 다이어리 알림)를 동시 제시
2. H1을 "당신 안에 답이…" → "방향은 아는데, 살아내지지 않는…"으로 — **명시적 갈증 호명**
3. 우측 카드를 "3가지 가치 설명" → **"당신은 어디에 가까우신가요?" 4분기 미리보기**로 교체
4. 가격은 Deliverables/Final CTA 섹션에서 충분히 노출되므로 손해 없음
