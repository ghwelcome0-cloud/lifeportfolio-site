## [핵심 — 빈칸 양식의 실제 항목명(직전 세션이 운영 PDF에서 추출 완료)]

### A. 리포트 PDF에서 확인된 실제 구조 (5페이지)

**[페이지 1] 사명·비전 제안 페이지**
- 상단 배지: 발행일 / 인생포트폴리오 / 유형 한 줄 설명
- 카드 1 "사명 (Mission)": [핵심 문장 1줄] + [보조 문장 1줄] + 소라벨 "다이어리 본문" + 소라벨 "한 줄 통합본"
- 카드 2 "비전 (Vision)": [핵심 문장 1줄] + [보조 문장 1줄] + 소라벨 "10년 뒤 회상 (Diary)" + 소라벨 "한 줄 통합본"

**[페이지 2] 실행 프로파일 카드 + 성장 가이드맵 + 진로 큐레이션**
- 실행 프로파일 표 6필드: 실행 유형 / 실행 스타일 / 추진력 요인 / 몰입 환경 / 잘 맞는 활동 / 추천 도구·전략
- 성장 가이드맵 박스 2개: "TOP3 강점" / "TOP2 성장 포인트"
- 진로 큐레이션 3카드: "추천 진로 · 직업" / "추천 교육 · 훈련" / "추천 확장 방향"

**[페이지 3] 활용 예시 + 자기이해**
- 활용 박스 3개: 직무 적용 / 학습 적용 / 실행 과제
- 체크박스 영역: "바로 실행할 첫 행동 3가지"
- 자기이해 (Self-Understanding) 카드: 응답 강도 %, 4라벨 (본질 / 한 마디 / 키워드 / 결합)

**[페이지 4] 자기표현 + 자기설계**
- 자기표현 (Self-Expression) 카드: 응답 강도 %, 4라벨 (본질 / 한 마디 / 키워드 / 결합)
- 자기설계 (Self-Design) 카드: 응답 강도 %, 4라벨 (본질 / 한 마디 / 키워드 / 결합)

**[페이지 5] 자기실행 + 요약**
- 자기실행 (Self-Execution) 카드: 응답 강도 %, 4라벨 (본질 / 한 마디 / 키워드 / 결합)
- "요약하자면" 박스: 종합 서술 2~3줄
- 하단 3액션 카드: "사명·비전 정교화" / "실행 전략 설계" / "진로·교육 설계"

→ **4SE 구조** (Self-Understanding / Self-Expression / Self-Design / Self-Execution), 각 **4라벨** (본질 / 한 마디 / 키워드 / 결합) 공통.

### B. 실행프로그램 PDF에서 확인된 실제 구조 (6페이지)

**[페이지 1] 표지 + 분기 테마 + 전체 요약**
- 인용 박스: [사명 한 줄] 강조
- "분기 테마" 카드 1개
- "전체 요약" 카드 4개
- "이 사명이 여는 길" 카드: 직업 후보 4개
- 강조 바: 이번 분기 루틴 한 줄

**[페이지 2] 맞춤 실행 프로그램 — 3주 실행 루틴**
- 상단 구조 표기: "3주 / 3개월 / 1년"  ※ 보고서의 "21일 점검"은 실제 3주 = 21일과 동일
- 1주차 카드 / 2주차 카드 / 3주차 카드 — 각 "실행 안내" / "실행 방법(4 bullet)" / "실행 효과"

**[페이지 3] 3개월 구조화 목표 + 1년 확장 방향**
- 3개월 카드 3개: 각각 [목표] + "확인 기준"
- "1년 확장 방향": VISION 박스 + bullet 4개 + 실행 효과 바

**[페이지 4] 실행 모듈 카드 TOP3 + 성과 추적 보드**
- TOP3 카드: 각각 [제목] + [설명] + bullet 2개 + "추천 도구"
- **성과 추적 보드 표 — 4컬럼: 주차 / 실행 과제 / 완료(Y/N) / 성찰 메모**
- 하단 안내문 + "월간 점검 항목" 3개

**[페이지 5] 기대 효과 + 다음 단계 + 리스크**
- "기대 효과" 4줄: 직무 적합성 / 직업 확장성 / 경력 성장 / 인생 설계 비전
- "다음 단계 제안" 3시점: 1개월 후 / 3개월 후 / 1년 후
- "리스크 및 보완 전략" 박스 3개

### C. 13영역의 정확한 매핑
- 운영 리포트 5페이지에는 "13영역"의 명시적 한글 리스트가 나오지 않음.

  - **Part 3** 주간 펼침면(52주 × 6p = 312p): 왼쪽 — "이번 주 사명 한 줄", "If-Then 3개", "실행 방법 bullet", "실행 효과" / 오른쪽 — 일정 + "주간 회고", "성찰 메모"
  - **Part 4** 영역별 분기 회고(4p × 13영역 = 52p): 영역명 인쇄 + Pennebaker 표현적 글쓰기 4문항 빈칸
