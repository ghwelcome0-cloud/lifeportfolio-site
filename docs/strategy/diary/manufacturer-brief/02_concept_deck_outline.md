# 02 — 시안 PPT 슬라이드 아웃라인 (16 슬라이드)

> **목적**: 제조사 3사(우일·북토리·이든프린팅)에 다이어리 컨셉·디자인 방향·제작 사양을 시각적으로 설명하기 위한 발표 자료.
> **톤**: Apple Keynote 톤. 미니멀 텍스트 중심. 흰 배경 + 짙은 네이비(#1A2B4A) 포인트 + 골드(#C9A04F) 보조.
> **노출 범위 원칙**: 가격·마진·CAC·LTV·텀블벅·매출 예측 일체 비공개. 제품 사양·디자인 컨셉·수량 견적 요청만.
> **3 No / 3 Yes**: 1인칭/자사·경쟁사 강조/책 제목 강조 금지. 독자 주어/원천 명시/추상화 마무리.

---

## 슬라이드 구성

| # | 슬라이드 제목 | 핵심 메시지 |
|---|---|---|
| 1 | **표지** — 인생포트폴리오 커스텀 다이어리 | 제품명 + 시안 v1.0 + 500부 견적 요청 |
| 2 | **목차** | 1) 제품 개요 → 2) 외관 → 3) 속지 → 4) 인쇄 구조 → 5) 견적 요청 |
| 3 | **제품 한 줄 정의** | "자기 검사 리포트가 그대로 옮겨 적히는 1년치 자기설계 다이어리" |
| 4 | **타겟 사용자 & 사용 시나리오** | 30~50대 자기설계 의지자 / 검사→리포트→다이어리 흐름 |
| 5 | **외관 사양** | A5 / PU양장 + 금박 / 양장사철 / 가름끈 2개 / 박스+인삿말 카드 |
| 6 | **컬러 & 마감** | 클래식 네이비(기본) / 버건디 / 그린 / 한정판 + 형압 로고 |
| 7 | **종이 사양 — 양자 견적** | 미색모조 80g / 만년필친화 70g(도모에리버 대체) 분리 견적 |
| 8 | **속지 구조 개요 — Part 0~7** | 7부 구조 + 합계 360p/432p 두 시나리오 |
| 9 | **Part 0 — VDP 영역(8p)** | 100% 개인화 / 사명·비전·4SE·강점·실행프로파일 인쇄 |
| 10 | **Part 1·2 — 13영역 인생지도 + 연간비전(28p)** | 라벨만 인쇄, 빈칸 채움 영역 |
| 11 | **Part 3 — 주간 펼침면 52주 × 6p(312p)** | 왼쪽 미션·실행의도 / 오른쪽 일정·회고 |
| 12 | **Part 4·5 — 분기회고 + 감사일기(64p)** | Pennebaker 표현적 글쓰기 4문항 + 월별 감사 3가지 |
| 13 | **Part 6·7 — 코칭기록 + 부록(20p)** | 성과 추적 보드 표 4컬럼 + 성경구절/명언/13영역 참고 |
| 14 | **인쇄 구조 — 2층(VDP + 일반 인쇄)** | Layer 1: VDP / Layer 2: 빈칸 양식 일반 인쇄 |
| 15 | **타이포·여백·페이지 설계 원칙** | 미니멀, 만년필 친화, 가독성 우선, 빈 공간 = 사용자 공간 |
| 16 | **견적 요청 항목 17개 + 회신 양식** | 사양서 PDF와 동일 항목 / 회신 기한 / 담당자 |
