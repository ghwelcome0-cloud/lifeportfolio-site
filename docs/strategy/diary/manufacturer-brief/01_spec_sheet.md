### 1.1 견적 요청 배경

본 제품은 **자기설계 다이어리(Identity Diary)** 카테고리의 신제품으로, 사용자가 자기 검사 리포트를 보면서 다이어리 빈칸 양식을 **직접 손글씨로 채워 넣는** 1년치 통합 다이어리입니다. 본 사양서는 500부 1차 발주 견적을 위한 표준 양식이며, 견적 비교 후 1개 제조사를 선정하여 샘플 제작을 의뢰합니다.

> 본 다이어리는 **사용자별 가변 데이터 인쇄(VDP)를 사용하지 않습니다.** 모든 페이지는 동일한 빈칸 양식의 일반 인쇄로 진행되며, 라벨·항목명은 인쇄되고 실제 내용은 사용자가 직접 손으로 적습니다. 이는 손글씨의 자기인지 효과를 극대화하기 위한 의도된 설계입니다.

### 1.2 견적 회신 시점

본 사양서 수령 후 7영업일 이내 회신 부탁드립니다.

## 2. 제품 정의 · Product Definition

### 2.1 한 줄 정의

> **"검사 리포트가 그대로 옮겨 적히는 1년치 자기설계 다이어리 — Only One, 세상에 단 하나"**

### 2.2 시장 차별점

| 일반 다이어리 (몰스킨/프랭클린/호보니치) | 인생포트폴리오 맞춤형 다이어리 |
|---|---|
| 표준 다이어리 양식 + 자유 메모 | 표준 양식 + **검사 리포트 7항목 손글씨 옮겨 적기** |
| 사용자가 콘텐츠를 만들어 채움 | **검사 리포트 콘텐츠 + 다이어리 양식 통합** |
| 익명의 도구 | **사용자별 1권의 "Only One" 정체성 다이어리** |
| 평균 240~300p | **256p (16절판 최적)** |

### 2.3 타겟 사용자

- 30~50대 자기설계 의지자 (직장인 · 프리랜서 · 사업자)
- 검사·진단 도구 결과를 일상에 통합하려는 사용자
- 1년 단위 목표 설계 + 주간 실행 점검을 병행하려는 사용자
- 만년필·종이 다이어리를 선호하는 아날로그 친화 사용자

### 2.4 사용 시나리오

```
STEP 1 — 검사 응답 (온라인 검사 도구)
        ↓
STEP 2 — PDF 리포트 5p + 실행 프로그램 6p 발급
        ↓
STEP 3 — 다이어리 Part 0 7항목에 리포트 결과를 손글씨로 옮겨 적기 (90분 리추얼)
        ↓
STEP 4 — 1년간 Part 1~7 일상 활용
```

## 4. 인쇄 구조 · Print Structure

### 4.1 단일 일반 인쇄 (v1.2)

**모든 페이지가 동일한 일반 인쇄 방식**. VDP 없음.
- Layer 1 / Layer 2 구분 없음 — 단일 인쇄 처리
- 사용자별 가변 인쇄 없음 — 전수량 100% 동일 양식
- 라벨·항목명·표 헤더·인용 박스는 인쇄됨
- 사용자가 검사 리포트를 보며 빈칸을 **직접 손글씨로** 채움

### 4.2 인쇄 사양

| 항목 | 사양 |
|---|---|
| 인쇄 방식 | 오프셋 / 디지털 (제조사 추천) |
| 컬러 | **2도 인쇄** (Navy #1A2B4A + Gold #C9A04F) 권장 / 흑백 단도 협의 가능 |
| 본문 컬러 | 검정 단도 / 라벨·헤더만 골드 |
| 후가공 | 금박 박표지 + 형압 'L' 로고 + 형압 'Only One' |

### 4.3 권장 사유

- **단일 인쇄** → 제작 공정 단순화 → 단가 안정성
- **70g 종이** → 만년필 친화 → 손글씨 사용자 경험 극대화
- **2도 인쇄** → 브랜드 일관성 + 시각적 격조
