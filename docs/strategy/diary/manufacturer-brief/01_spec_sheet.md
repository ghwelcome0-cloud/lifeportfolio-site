### 1.1 견적 요청 배경

본 제품은 자기설계 다이어리(Identity Diary) 카테고리의 신제품으로, 사용자별 검사 리포트 데이터를 다이어리 인쇄 시점에 박아 넣는 **개인화 인쇄(VDP)** 영역과 사용자가 직접 채워 넣는 **빈칸 양식** 영역의 2층 구조를 가집니다. 본 사양서는 500부 1차 발주 견적을 위한 표준 양식이며, 견적 비교 후 1개 제조사를 선정하여 샘플 제작을 의뢰합니다.

### 1.2 견적 회신 시점

본 사양서 수령 후 7영업일 이내 회신 부탁드립니다.

## 4. 빈칸 양식의 2층 구조 · Two-Layer Print Structure

본 다이어리의 가장 큰 차별점은 **인쇄 고정 영역(층 1, VDP)** 과 **빈칸 양식 영역(층 2, 일반 인쇄)** 의 구분입니다.

### 4.1 층 1 — Part 0 (VDP, 가변 데이터 인쇄 8p)

구매자별 검사 결과를 **다이어리 인쇄 시점에 박아 넣는** 영역입니다. 구매자 1명당 1권에 고유 데이터가 인쇄됩니다.

| 인쇄 데이터 | 비고 |
|---|---|
| 사명 1줄 (Mission) | 핵심 문장 + 보조 문장 |
| 비전 1줄 (Vision) | 핵심 문장 + 보조 문장 |
| 4SE 점수(%) 4개 | 자기이해 / 자기표현 / 자기설계 / 자기실행 |
| 4SE 4라벨 (각 영역당) | 본질 / 한 마디 / 키워드 / 결합 |
| TOP3 강점 | 성장 가이드맵 |
| TOP2 성장 포인트 | 성장 가이드맵 |
| 실행 프로파일 6필드 | 실행 유형 / 실행 스타일 / 추진력 요인 / 몰입 환경 / 잘 맞는 활동 / 추천 도구·전략 |

### 4.2 층 2 — Part 1~7 (일반 인쇄, 빈칸 양식)

라벨·항목명은 인쇄되어 있고 빈칸만 남깁니다. 사용자가 자기 리포트를 보고 직접 채워 넣습니다.

| Part | 빈칸 양식 라벨 |
|---|---|
| Part 1 (16p) | 13영역 명칭만 인쇄, 점수/메모 칸 빈칸 |
| Part 2 (12p) | "VISION" 박스, "분기 마일스톤 3개" 라벨만 인쇄 |
| Part 3 (312p) | 좌: "이번 주 사명 한 줄" / "If-Then 3개" / "실행 방법 bullet" / "실행 효과" — 우: 일정 + "주간 회고" / "성찰 메모" |
| Part 4 (52p) | 영역명 인쇄 + Pennebaker 표현적 글쓰기 4문항 빈칸 |
| Part 5 (12p) | "이번 달 감사 3가지" 라벨만 인쇄 |
| Part 6 (8p) | **성과 추적 보드 표 — 4컬럼: 주차 / 실행 과제 / 완료(Y/N) / 성찰 메모** |
| Part 7 (12p) | 성경구절/명언/13영역 참고 — 라벨만 인쇄 |

> **요청 포인트**: 층 1(VDP) 단가와 층 2(일반 인쇄) 단가를 **별도 분리하여 회신**해주십시오. 향후 운영에서 VDP 비율이 변동될 수 있어 분리 단가 확인이 핵심입니다.

## 5. 외관 사양 · Physical Specifications

### 5.1 판형

- A5 (148 × 210 mm)
- 호보니치 Cousin과 동일 규격 → 시중 보호 커버 호환

### 5.2 표지

- 재질: PU 양장 (인조가죽)
- 마감: 금박 박표지 (1박 기본 + 추가 박 가능)
- 표지 인쇄: 형압 로고 (음각/양각 협의)
- 색감: 클래식 네이비(기본) · 버건디 · 그린

### 5.3 종이 (양자 견적 분리 요청)

| 시나리오 | 평량 | 특징 | 용도 |
|---|---|---|---|
| ① 미색 모조 80g | 80gsm | 일반 다이어리 표준 | 안정적 단가·뒷비침 적음 |
| ② 만년필 친화 70g | 70gsm | 도모에리버 대체재 | 만년필·잉크펜 사용자 대응 |

### 5.4 제본

- 양장 사철 (Smyth-sewn binding)
- 펼침성: 180도 완전 펼침 가능해야 함
- 두께 검증: 360p 시나리오 약 18~22mm / 432p 시나리오 약 22~26mm 예상 (실제 종이 평량에 따라 변동)

### 5.5 부속

- 가름끈 2개 (네이비/골드 또는 협의)
- 모서리 코너 4개 (금속 마감, 모서리 보호)
- 인덱스 박 (Part별 색인용 박)
- 뒷주머니 1개 (영수증·메모지 보관용)

### 5.6 컬러 옵션 (3색 동시 제작)

| 컬러 | 적용 | 단가 영향 회신 요청 |
|---|---|---|
| 클래식 네이비 | 기본 (전체 부수 중 60% 예상) | 기준 단가 |
| 버건디 | 옵션 (전체 부수 중 25% 예상) | 단가 변동 여부 |
| 그린 | 옵션 (전체 부수 중 15% 예상) | 단가 변동 여부 |

> 3색 동시 제작 시(인쇄판 변경·소량 분할) 부당 단가 변동을 별도 명시해주십시오.

### 5.7 포장

- 외함 박스 (네이비 또는 화이트 매트 — 협의)
- 인삿말 카드 1매 (90 × 130mm 추정, 별도 양식)
- 사용 가이드 8p 별책 (A5 또는 A6, 무선 또는 중철 — 협의)

## 6. 견적 요청 사항 · Quote Request

### 6.1 회신 요청 항목

각 제조사는 아래 항목을 **동일 양식**으로 회신해주십시오.

1. **부당 원가** (단위: 원/부, 부가세 별도 명시)
   - 시나리오 A (360p, 80g) / 시나리오 A (360p, 70g)
   - 시나리오 B (432p, 80g) / 시나리오 B (432p, 70g)
2. **총견적** (500부 기준, 위 4개 시나리오 각각)
3. **VDP 가변 데이터 인쇄 (Part 0 8p)**
   - 가능 여부 (Y/N)
   - 가능 시 부당 추가 단가
   - 입고 데이터 포맷 (CSV / Excel / API 연동 등)
4. **샘플 제작비** (1부, 디자인 미확정 단계의 가제본 샘플)
5. **납기**
   - 디자인 입고일 기준 D+? 일 (영업일 기준)
   - 시안 확정 후 D+? 일
6. **결제 조건**
   - 계약금 / 중도금 / 잔금 비율
   - 결제 수단 (계좌 이체 / 카드 / 세금계산서)
7. **3색 동시 제작 시** 단가 변동 (네이비/버건디/그린)
8. **재고 보관·재발주** 시 단가 변동 (2차 발주 시 기준)

## 7. 부록 · Appendix

### 7.1 학술 근거

본 다이어리의 콘텐츠 설계는 다음 4편의 학술 연구를 근거로 합니다. 견적과는 무관하나 제조사가 제품 가치를 이해하는 데 도움이 됩니다.

| 연구 | 출처 | 효과 크기 |
|---|---|---|
| Matthews (2015) — 글로 쓰면 +78% | Dominican Univ. 5그룹 RCT | 목표 달성률 +78% (Group 5) |

| Cregg (2024) — 감사 일기 | PNAS Meta-Analysis | 우울 감소 d = 0.31 |

### 7.2 향후 확장 (참고 — 이번 견적 범위 외)

본 사양서는 **다이어리 본책 1종 500부**만을 견적 대상으로 합니다. 다음은 참고용 향후 로드맵이며 이번 견적과 무관합니다.

- Year 2 이후 분기 미션북·월간 보충제 등 Refill 제품 검토

### 7.3 비밀유지
