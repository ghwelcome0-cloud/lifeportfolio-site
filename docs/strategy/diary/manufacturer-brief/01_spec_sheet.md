### 1.1 견적 요청 배경

본 제품은 **자기설계 다이어리(Identity Diary)** 카테고리의 신제품으로, 사용자가 자기 검사 리포트를 보면서 다이어리 빈칸 양식을 **직접 손글씨로 채워 넣는** 1년치 통합 다이어리입니다. 본 사양서는 500부 1차 발주 견적을 위한 표준 양식이며, 견적 비교 후 1개 제조사를 선정하여 샘플 제작을 의뢰합니다.

> 본 다이어리는 **사용자별 가변 데이터 인쇄(VDP)를 사용하지 않습니다.** 모든 페이지는 동일한 빈칸 양식의 일반 인쇄로 진행되며, 라벨·항목명은 인쇄되고 실제 내용은 사용자가 직접 손으로 적습니다. 이는 손글씨의 자기인지 효과를 극대화하기 위한 의도된 설계입니다.

### 1.2 견적 회신 시점

본 사양서 수령 후 7영업일 이내 회신 부탁드립니다.

## 2. 제품 개요 · Product Overview

### 2.1 한 문장 컨셉

> **"검사 리포트를 손글씨로 옮겨 적는 365일 자기설계 다이어리."**

### 2.2 제품 정체성

- **카테고리**: 자기설계 다이어리 (Identity Diary)
- **유형**: 양장 본책 1종, 연 1회 갱신 모델
- **차별점**: 라벨·항목명만 인쇄되어 있고, 검사 리포트의 핵심 데이터를 사용자가 **직접 손으로 옮겨 적는** 빈칸 양식 다이어리. 자기 리포트와 다이어리의 라벨·항목 순서가 100% 일치하여, 옮겨 적는 행위 자체가 자기화(自己化) 의식이 되는 구조.
- **참고 다이어리 IP** (저작권 범위 외 방법론 참고): 몰스킨 위클리(자유 메모 우측 배치) · 프랭클린 플래너(ABC 우선순위) · 호바니치(빈칸 양식 위주)
- **인생포트폴리오 자체 IP**: 13영역 인생지도 / 4SE 4라벨 / 이번 주 사명 한 줄 / 영역별 분기 회고 / 성과 추적 보드 4컬럼

## 4. Part 0 손글씨 7항목 (8p) — 핵심 자기화 구간

본 다이어리의 첫 8페이지는 사용자가 자기 리포트를 보면서 **직접 손으로 옮겨 적는** 의식(ritual) 공간입니다.

| p | 인쇄 라벨 | 빈칸 양식 |
|---|---|---|
| 1 | 인트로 — "내 리포트를 손으로 옮겨 적기" + 손글씨의 학술 효과 1단락 | 본문 인쇄 |
| 2 | **사명 (Mission)** | 핵심 문장 1줄 + 보조 문장 1줄 |
| 3 | **비전 (Vision)** | 핵심 문장 1줄 + 보조 문장 1줄 |
| 4 | **4SE 응답 강도(%)** | 자기이해 ___% / 자기표현 ___% / 자기설계 ___% / 자기실행 ___% |
| 5 | **TOP3 강점** | 3줄 (성장 가이드맵에서 발췌) |
| 6 | **TOP2 성장 포인트** | 2줄 (성장 가이드맵에서 발췌) |
| 7 | **실행 프로파일 6필드** | 유형 / 스타일 / 추진력 / 몰입 환경 / 활동 / 도구 — 각 1줄 |
| 8 | **추천 진로 3카드** | 추천 진로 · 추천 교육 · 확장 방향 — 각 1줄 |

> **이 영역은 VDP 가변 데이터 인쇄가 아닙니다.** 모든 사용자에게 동일한 빈칸 양식이 인쇄되며, 사용자는 자기 리포트(웹 또는 PDF)를 보면서 직접 손글씨로 옮겨 적습니다. 이 의도된 옮겨 적기 행위가 "리포트 데이터의 자기화"입니다.

> **요청 포인트**: 70g 만년필 친화 종이가 손글씨의 다양한 필기구(만년필·잉크펜·중성펜·연필)를 잘 받아내는지 샘플 확인이 견적의 핵심입니다. 견적서에 **종이 샘플 1매 동봉** 부탁드립니다.

## 6. 견적 요청 사항 · Quote Request

### 6.1 회신 요청 항목

각 제조사는 아래 항목을 **동일 양식**으로 회신해주십시오.

1. **부당 원가** (단위: 원/부, 부가세 별도 명시)
   - 232p × 70g × A5 양장사철 단일 시나리오
2. **총견적** (500부 기준)
3. **샘플 제작비** (1부, 디자인 미확정 단계의 가제본 샘플)
4. **종이 샘플 동봉** (70g 만년필 친화 종이 1매 — A5 또는 A4)
5. **납기**
   - 디자인 입고일 기준 D+? 일 (영업일 기준)
   - 시안 확정 후 D+? 일
6. **결제 조건**
   - 계약금 / 중도금 / 잔금 비율
   - 결제 수단 (계좌 이체 / 카드 / 세금계산서)
7. **3색 동시 제작 시** 단가 변동 (네이비/버건디/그린)
