# Instagram Coming Soon Carousel — 기획서 (Final v5)

> 인생포트폴리오 맞춤형 다이어리 v1.4.2 출시 준비 — 에버그린 캐러셀 1세트

## 1. 목적 & 위치

- **목적**: "잠자는 동안에도 알고리즘으로 트래픽이 발생하는" 에버그린 콘텐츠 1세트 구축
- **포지셔닝**: 트렌드 의존 ❌ / 진정성 있는 창업자 1인칭 여정 서사 ✅
- **타깃**: 다이어리 검색 트래픽 + 자기이해/적성검사 검색 트래픽 양쪽
- **운영**: 프로필 상단 고정(Pin), 6개월 유지, 분기마다 캡션 미세 조정

## 3. 스토리텔링 4단계 공식 — 포지티브 빌딩 매핑

기존 공식(문제→스토리→고난→해결)을 네거티브 프레이밍 없이 **사명(Why) → 여정(Journey) → 현재(Now) → 비전(Vision)** 으로 전환.

| 단계 | 슬라이드 | 실제 활동 |
|---|---|---|
| 시작 — 고유함을 찾는 도구 | 2 | 검사지 직접 설계 |
| 공간 — 만남의 자리 | 3 | 홈페이지 제작 |
| 확장 — 글로벌 접근성 | 4 | 결제-검사-결과 즉시 전달 체계 |
| 현재 — 다이어리 (이 게시물의 주제) | 5 | 검사 결과를 매일 살아낼 수 있는 도구 |
| 비전 — 인생포트폴리오 | 6 | "발견 → 살아낸 삶 → 누군가의 양식" |

## 4. 캐러셀 6장 — 카피

| # | 슬라이드 | 메인 카피 | 보조 |
|---|---|---|---|
| 1 | 표지 | 한 사람의 인생이, / 하나의 포트폴리오가 / 되기까지. | 인생포트폴리오 커스텀 다이어리 (Coming Soon · 2026) |
| 2 | 시작 | 먼저, / 고유함을 찾아가는 / 도구를 만들었습니다. | 사명 · 비전 · 4SE 응답 · 추천 진로/직업 |
| 3 | 공간 | 그 도구를 / 만나는 자리를 / 만들었습니다. | 온라인에서, 언제든. |
| 4 | 확장 | 국내외 누구나, / 결제 즉시 검사와 / 결과를 받습니다. | Anywhere · Anytime |
| 5 | 현재 (다이어리) | 이제, 검사 결과지가 / 매일의 자리에서 함께 / 살아낼 수 있도록, / 다이어리를 만들고 있습니다. | (보조 문구 없음) |
| 6 | 비전 (CTA) | 발견이 살아낸 삶이 되고, / 그 삶이 누군가의 / 양식으로 남도록. | 이 여정에 함께 머물러주세요. |

## 5. 디자인 시스템

- **컬러**: BRG `#0A3D2A` / Gold `#C9A04F` / Off-white `#F5F1E8`
- **타이포**: 영문 = Cormorant Garamond / 한글 = Noto Serif CJK KR (다이어리 본문과 100% 동일)
- **레이아웃**: 1:1 정사각형 1080×1080
- **공통 요소**: 상단 챕터 표기 + 금색 가로선 + 메인 카피 + 보조 문구 + 하단 footer (Life Portfolio · Coming Soon · NN/06)

## 6. 운영 전략

| 항목 | 전략 |
|---|---|
| **포스팅 시점** | 화 또는 목 오후 8~10시 |
| **고정(Pin)** | 프로필 상단 3장 슬롯 중 1개로 6개월 고정 |
| **CTA 진화** | 출시 준비 시 캡션 마지막 1줄만 업데이트 → 에버그린 유지 |
| **재활용** | 슬라이드 1, 5, 6 → 스토리/릴스 / 슬라이드 2~4 → 단독 피드 |
| **A/B 테스트** | 표지(1번) 2가지 버전 → 1주씩 교차, 저장률 비교 |
| **분기별 점검** | 3개월마다 캡션 마지막 1~2줄 미세 조정 (단계 진척 반영) |

## 7. 제작 방식

- HTML/CSS 직접 제작 → WeasyPrint PDF → PyMuPDF PNG 변환 (한글 오타 0% 보장)
- 슬라이드 2~5 배경: nano-banana-pro 생성 (텍스트 없는 분위기 일러스트만)
- 슬라이드 1, 6: 순수 타이포그래피 (BRG 단색 + Gold)
- 빌더 스크립트: `scripts/build_carousel.py` — 카피·디자인 수정 시 1줄 변경 후 즉시 재빌드 가능
