## 📝 캡션 (Final v6)

```
한 사람의 인생이,
하나의 포트폴리오가 되기까지 —

지금, 다이어리를 만들고 있습니다.

검사 결과지가 책상 서랍에 머무르지 않고,
매일의 자리에서 함께 살아낼 수 있도록.

페르소나 분석을 토대로,
페이지마다 당신의 고유함을 적어 내려갈 자리를 두었습니다.

하루의 컨디션, 한 주의 회고, 한 달의 방향,
사명, 비전, 4SE 응답, 추천 진로·직업까지.

당신의 고유함이 머무는 자리.

인생포트폴리오는
당신을 멋지게 설명하기 위한 것이 아닙니다.

발견이 살아낸 삶이 되고,
그 삶이 누군가의 양식으로 남도록.

이 여정에 함께 머물러주세요.

— — —
🔖 이 글이 마음에 닿았다면, 저장해두세요.
   다이어리가 준비되면 가장 먼저 보여드리고 싶으니까요.

#인생포트폴리오 #포트폴리오다이어리 #맞춤형다이어리 #1년계획 #ComingSoon
```

### 캡션 설계 의도 (마케팅 매핑)

| 구간 | 카피 | 마케팅 역할 |
|---|---|---|
| **1행(Hook)** | "한 사람의 인생이, / 하나의 포트폴리오가 되기까지 —" | 슬라이드 1과 동일 운율로 **시각·청각 연속성** 확보. em대시(—)로 호기심 트리거 → "더보기" 클릭 유도 |
| **2행** | "지금, 다이어리를 만들고 있습니다." | "**다이어리 출시 준비 중**"이라는 캠페인 핵심 메시지 즉시 노출 (피드 미리보기 3줄 안에 들어감) |
| **본문 1단** | "검사 결과지가 책상 서랍에…" | **공감 포인트**: 검사받고 잊혀진 경험을 환기 → 본 다이어리의 존재 이유 자연 도출 |
| **본문 2단** | "페르소나 분석을 토대로…" | **차별화 선언**: 일반 다이어리와 다른 점(개인화) 명시. SEO 키워드 '페르소나·고유함' 포함 |
| **본문 3단** | "하루의 컨디션, 한 주의 회고…" | **기능 구체화**: 추상적 가치 → 실제 콘텐츠로 변환. 잠재 구매자가 "내가 무엇을 받게 되는지" 즉시 파악 |
| **본문 4단** | "당신을 멋지게 설명하기 위한 것이 아닙니다." | **포지셔닝 차별화**: 자기과시·자기계발 시장과 선 긋기 → 우리 브랜드만의 '깊이 있는' 포지션 확립 |
| **클로징** | "발견이 살아낸 삶이 되고… / 이 여정에 함께 머물러주세요." | 비전 CTA. 강매가 아닌 **'동행 초대'** — 친밀감 + 장기 관계 신호 |
| **저장 트리거** | "🔖 이 글이 마음에 닿았다면…" | **Save Rate 부스팅** (Later 2025: 명시 시 저장률 2~3배↑). 이모지 1개로만 시각 신호. "준비되면 가장 먼저"로 **선점 심리** 발동 |

## 💬 고정 댓글 (Pinned Comment)

### 마케팅 목적
1. **댓글 섹션 활성화** — 빈 댓글 < 활발한 댓글, 인스타 알고리즘이 명확히 우대
2. **첫 댓글 유도 질문** — 답변하기 쉬운 형태로 진입장벽 낮춤
3. **외부 정보 노출** — 캡션 안에 넣기엔 무거운 정보(검사·홈페이지)를 자연스럽게 안내
4. **저장(Save) 한 번 더 트리거** — 캡션과 다른 표현으로 재강조

### 본문

```
🌿 이 다이어리가 만들어지기까지의 여정.

  ① 한 사람의 고유함을 발견하는 '인생포트폴리오 검사'를 만들었고
  ② 누구나 만날 수 있도록 온라인 홈페이지를 열었고
  ③ 국내외 어디서든 결제 즉시 받아보실 수 있게 했고
  ④ 이제, 매일의 자리에서 함께 살아낼 다이어리를 만들고 있습니다.

검사·결과지·다이어리가 한 흐름으로 연결되는 — 
국내 유일의 '발견에서 일상까지' 라이프 포트폴리오 시스템입니다.

💬 당신의 한 해는, 어떤 한 권으로 남기고 싶으신가요?
   댓글로 들려주시면, 다이어리 설계에 곱게 새기겠습니다.

🔖 출시 알림을 놓치고 싶지 않다면 이 게시물을 저장해두세요.
🔗 검사 미리 받아보기 → 프로필 링크
```

### 고정 댓글 설계 의도 (마케팅 매핑)

| 요소 | 카피 | 마케팅 역할 |
|---|---|---|
| **여정 요약 ①~④** | 검사 → 홈페이지 → 글로벌 → 다이어리 | **신뢰 구축**: 브랜드가 '하루아침에 등장한 다이어리'가 아니라 **체계적 시스템의 일부**임을 증명. 캐러셀의 스토리를 텍스트로 압축 재강조 |
| **"국내 유일" 선언** | "발견에서 일상까지" 시스템 | **USP(Unique Selling Proposition)**: 명확한 차별화 포지셔닝 한 줄 선언. 신규 방문자가 즉시 가치 파악 |
| **참여 질문** | "어떤 한 권으로 남기고 싶으신가요?" | **댓글 트리거** — 답하기 쉬움 + 자기 투영적 → 자연스러운 댓글 유발. 답변은 그대로 **고객 인사이트**로 수집 가능 |
| **"다이어리 설계에 곱게 새기겠습니다"** | — | **참여 보상**: 단순 질문이 아니라 "당신의 답이 제품에 반영된다"는 약속 → UGC 동기 강화 |
| **저장 트리거 재강조** | "이 게시물을 저장해두세요" | 캡션과 다른 워딩으로 한 번 더 발동. 알림 신청 CTA는 evergreen 원칙상 사용 안 함 |
| **프로필 링크 안내** | "🔗 검사 미리 받아보기 → 프로필 링크" | **트래픽 전환**: 인스타 → 홈페이지 이동 동선. "출시 후" 가 아니라 "지금 바로" 경험 가능한 검사로 **즉시 가치 제공** |

## 🏷️ 해시태그 (5개) — 알고리즘 전략

```
#인생포트폴리오 #포트폴리오다이어리 #맞춤형다이어리 #1년계획 #ComingSoon
```

| # | 해시태그 | 역할 | 검색 의도 매핑 |
|---|---|---|---|
| 1 | `#인생포트폴리오` | **브랜드 자산** (Owned) | 검색 시 100% 우리 콘텐츠 노출 → 브랜드 검색 트래픽 흡수 |
| 2 | `#포트폴리오다이어리` | **카테고리 창출** (Branded Category) | '인생포트폴리오 + 다이어리' 신규 카테고리 점유 선점 |
| 3 | `#맞춤형다이어리` | **카테고리 키워드** (실질 유입) | 다이어리 검색 사용자가 실제로 검색하는 어휘 — 메인 트래픽 |
| 4 | `#1년계획` | **에버그린 일반 키워드** | 시즈널('2026다이어리') 회피 → 1년 내내 검색 유입 가능 |
| 5 | `#ComingSoon` | **단계 표시** | 출시 후 `#NowAvailable` 또는 `#출시완료` 로 교체 예정 |

### 왜 5개만? — 시장조사 기반 의사결정
- 인스타 공식 가이드 (2024 업데이트): "**3~5개의 정확한 해시태그가 30개의 광범위 해시태그보다 효과적**"
- 럭셔리 브랜드 사례 분석: 평균 3~7개 사용, 키워드 정밀도가 양보다 중요
- 우리는 **브랜드 자산(2개) + 카테고리(2개) + 단계(1개)** 의 3계층 구조로 5개 운용

## 📅 운영 가이드

### 게시 시간 권장
- **주중 화·수·목 오후 7~9시** — 인스타 한국 사용자 피크 + 다이어리 카테고리는 '하루를 마무리하는 시간대' 정서와 일치

### 첫 1시간 운영 (Golden Hour)
1. 게시 직후 **고정 댓글 즉시 등록 → 본인 계정으로 고정(Pin)**
2. 본인이 단 첫 댓글에 1시간 내 추가 답글 1~2개 (대화 활성화 신호)
3. 스토리에 캐러셀 1번 슬라이드 공유 + "swipe up 대신 캐러셀로 →" 유도

### A/B 테스트 후보 (다음 게시물)
- 캡션 첫 줄을 **질문형**으로 전환: "한 사람의 인생이, 한 권에 담긴다면?"
- 저장 트리거 위치를 **캡션 상단**으로 이동 후 비교
