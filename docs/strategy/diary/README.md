## 🎯 핵심 결론 (Top 5)

1. **다이어리 = 굿즈 X, Identity Product O** — 호보니치·프랭클린·드로우앤드류 모두 검증
2. **가격 사다리로 9,900원 가치 재인식** — 단독 노출 = 저가치, 사다리 노출 = 입문가치
3. **다이어리 + 1:1 코칭 = 학술적으로 입증된 +78%** (Matthews 2015 RCT)
4. **페르소나 재정의**: "실행 정체자" → "의미를 살아내려는 자" (Meaning-Liver)
5. **실물 우선** — 손글씨 뇌효과(NTNU 2024) + 365일 정체성 매개체
