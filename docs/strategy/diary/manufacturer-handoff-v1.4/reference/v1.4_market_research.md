### 1-3. 인생포트폴리오 서비스 적합성 분석

**✅ 매우 높은 적합성 — 4가지 이유**

| 근거 | 설명 |
|---|---|
| ① **검사 → 1년 살아내기 흐름과 일치** | 리포트 수령일이 곧 다이어리 시작일. 매년 1월에 종속되면 7~12월 구매 고객은 "내년까지 대기" 또는 "남은 달만 사용" 손실 발생 |
| ② **타깃 페르소나의 라이프스타일** | LP 고객 = 자기성찰 코칭 수강자 = 이벤트 트리거 발생 시점(이직/리셋/번아웃 회복) 구매. 1/1 시작 강제는 비현실적 |
| ③ **재고/SKU 관리 단순화** | 2026 / 2027 / 2028 ... 매년 새 SKU 발주 불필요. **단일 SKU 영구 판매 가능** (오롤리데이 11년째 같은 SKU 유지) |
| ④ **"Only One" 컨셉과 정합** | 이미 "Only One 인증서" 컨셉으로 단순화 강조. 시즌 종속성 제거는 그 연장선 |

### 1-4. 인생포트폴리오 언데이티드 설계 권장안

**자기화(自己化) 의식 + 1년 살아내기 흐름과 결합**

```
[Day 0]    : 리포트 수령 + 다이어리 첫 페이지에 "시작일" 직접 기입
[Week 1]   : Part 0 (검사 리포트 옮겨 적기 7항목) — 자기화 의식
[Week 1-2] : Part 1 (13영역 현재 점수) — 출발점 측정
[Week 1-2] : Part 2 (90일 비전 + 마일스톤) — 목표 설정
[월 1~12]  : 월간 → 주간 반복 (52주 × 2p, 연속 흐름)
[분기]     : Part 4 (분기 회고) × 4회
[월말]     : Part 5 (감사 일기) × 12회
[필요시]   : Part 6 (1:1 코칭) + Part 7 (자유 메모)
[Day 365]  : 종료일 자동 도달 → 다음 1년 시작 (재구매 트리거)
```

### 1-5. 달력의 처리 방식 (사용자 요청: "달력은 반영")

**3가지 옵션 비교**:

| 옵션 | 방식 | 장점 | 단점 |
|---|---|---|---|
| **A** | 미니 연간 캘린더만 (2년치) 부록에 배치 | 가장 보편적 | 월간이 데이트프리라 약점 |
| **B** ⭐ | **"올해 / 다음해" 미니 연간 캘린더 2장 부록 + 월간 페이지는 1~31일 그리드만(요일/날짜 직접 기입)** | 언데이티드 본연 + 연도 무관 영구 사용 | 첫 사용 시 직접 기입 부담 |
| **C** | 사용자가 직접 시작일 기재 → "1개월차 / 2개월차" 표기 (몰스킨 식) | 가장 자유로움 | "달력" 요구와 어긋남 |

**권장**: **옵션 B** — 미니 연간 캘린더(부록 4p, 2년치 × 2년 = 4년 커버)를 두되, **월간 펼침면은 1~31일 숫자 그리드 + 요일은 사용자가 첫 사용 시 직접 기입**. 이렇게 하면 7월에 사도 8월에 사도 그 시점부터 즉시 작성 가능.

## ④ 실용신안등록출원 필요성 검토

### 4-1. 한국 IP 4종 비교

| 종류 | 보호 대상 | 다이어리 적용 가능성 |
|---|---|---|
| **특허** | 자연법칙 이용한 **고도한** 기술적 발명 | ❌ 다이어리에 부적합 (기술적 발명 없음) |
| **실용신안** | **물품의 형상·구조·조합**에 관한 (소발명) 고안 | ⚠️ 제본·표지 구조 등 **물리적 구조 혁신**이 있을 때만 적용. 단순 종이 다이어리는 거절 가능성 높음 |
| **디자인** | **물품의 외관**(형상·모양·색채) | ✅ **표지·속지 레이아웃·전체 디자인** 보호 가능 |
| **상표** | 상품 명칭·로고 | ✅ "LIFE · PORTFOLIO" / "Only One" 로고 보호 |

### 4-2. 실용신안 — 다이어리 적합성 분석

**결론: 권장하지 않음.** 이유:

1. **실용신안은 "물품의 형상·구조·조합"** 이 대상 — 즉 **물리적 구조에 신규성이 있어야 함**
2. v1.3 다이어리는 일반적인 사철 제본 + A5 + 양장 — **구조적 혁신 없음**
3. 다이어리 **내용·페이지 구성·레이아웃**은 실용신안 대상 아님 → 거절 가능성 높음
4. 실용신안으로 강제로 출원해도 등록 후 **신규성 결여로 무효심판 위험**

### 4-3. 다이어리 IP 보호 최적 전략 (3중 보호)

**✅ 권장: 디자인등록 + 저작권 + 상표 3중 보호**

#### 보호 ①: 디자인등록 (가장 중요)

**KIPRIS 검색 결과**: 다이어리 디자인등록은 일반화된 영역. **부분디자인**(2025-11-28 시행 개정으로 절차 간소화)으로 보호 가능:

| 보호 가능 요소 | 출원 방식 |
|---|---|
| 전체 다이어리 외관 (표지+크기+책등 형상) | 일반 디자인등록 |
| **표지의 "Only One" 텍스트 배치** | 부분디자인 |
| **속지 핵심 페이지(13영역 그리드, 4SE 차트, 주간 펼침면)** | 부분디자인 (각각 별도 또는 한벌물품) |

**비용 (참고)**:
- 직접 출원: 관납료 약 9.4만원 + 등록료 7.5만원 = **약 17만원/건**
- 변리사 대행: 출원 30~40만 + 등록 30~40만 = **약 70~90만원/건**
- **3건 일괄 (전체 + 표지 부분 + 핵심 속지 부분) 대행 시 약 200~270만원**

**소요 기간**:
- 일반심사: 약 6~9개월
- 우선심사: 추가비용 발생, 약 3개월

**권장 출원 건수**:
| # | 대상 | 출원 유형 | 우선순위 |
|---|---|---|---|
| 1 | 다이어리 전체 외관 (커버+책등+가름끈) | 일반 디자인 | 필수 ★★★ |
| 2 | 표지 형압 "L" 로고 + "Only One" 텍스트 배치 | 부분디자인 | 필수 ★★★ |
| 3 | Part 0 4SE 응답 강도 차트 페이지 | 부분디자인 | 권장 ★★ |
| 4 | Part 1 13영역 인생 지도 페이지 | 부분디자인 | 권장 ★★ |
| 5 | 주간 펼침면 (Part 3) 레이아웃 | 부분디자인 | 선택 ★ |

→ **최소 1~2번 필수, 3~4번 권장 = 약 60~140만원 (직접 출원 기준)**

#### 보호 ②: 저작권 등록

**저작권법 제6조 (편집저작물)**: 다이어리 속지 = **편집저작물로서 독자적 보호** 대상.

- **자동 발생**: 창작 시점부터 권리 발생 (등록 불필요)
- **등록 시 이점**: 침해 시 입증 부담 경감, 손해배상 액수 산정 유리
- **등록 비용**: 한국저작권위원회 등록 — **저작물 1건당 약 2.3만원** (디자인 대비 매우 저렴)
- **권장**: 다이어리 전체를 **편집저작물 1건으로 등록** → 모든 페이지 일괄 보호

**근거 판례**: 2012 저작권 백서에 *"다이어리 속지 디자인 2종과의 실질적 유사성 여부"* 분쟁 사례 — 다이어리 속지가 저작권 보호 대상임을 법원이 인정한 선례 존재.

#### 보호 ③: 상표등록

- **"LIFE · PORTFOLIO" 워드마크** + **"Only One" 슬로건** → 상표등록 (1상품류 ~6만원 + 변리사 수수료)
- **이미 LP 서비스 본체에 상표가 있다면 다이어리 출시 시 16류(종이/문구류) 추가 출원** 필요

### 4-4. 출원 타이밍 — 매우 중요

**디자인등록은 "신규성"이 필수.** 출원 전 공개되면 등록 거절됩니다.

⚠️ **현재 위험 신호**:
- v1.3 시안이 **인터넷에 공개**되어 있음 (보안 게이트 적용했지만 패스워드 노출 가능성 존재)
- 제조사에 시안 전달 시 **공지(공연한 양도) 발생** 가능

**KIPO 신규성 상실 예외**: 공개 후 **12개월 이내** 출원하면 예외 적용 가능. 하지만 **공개 전 출원이 가장 안전**.

### 4-5. 권장 출원 일정

```
[ NOW ]     변리사 사전 상담 (무료 상담 활용 — 서울지식재산센터 등 1~2주 내)
[ 2주 내 ]  필수 디자인등록 출원 2건 (전체 외관 + 표지 부분디자인)

[ 6~9개월 ] 디자인등록 결정 → 다이어리 정식 출시
```

### 4-6. 비용/효익 종합 (1년차 기준)

| 항목 | 직접 출원 | 변리사 대행 |
|---|---|---|
| 디자인등록 2건 (전체 + 표지) | 약 34만원 | 약 140~180만원 |
| 디자인등록 3건 (속지 부분) 추가 | +51만원 | +210~270만원 |
| 저작권 등록 1건 | 약 2.3만원 | 약 30~50만원 |
| 상표등록 1건 (16류) | 약 11만원 | 약 50~70만원 |
| **합계 (최소)** | **약 47만원** | **약 220~300만원** |
| **합계 (전체)** | **약 98만원** | **약 430~570만원** |

**권장**: **1차 핵심 2건은 변리사 대행, 나머지는 직접 출원** = **약 200~250만원** 

이유: 권리범위 설정(claim)이 등록보다 더 중요하며, 변리사 없이 셀프 출원 시 권리범위가 너무 좁아 모방 방어가 어려움 (Instagram 변리사 경고 다수).
