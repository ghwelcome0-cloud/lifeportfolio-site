# 종이 사양 — 70g 만년필 친화지 (v1.4)

## 핵심 사양

| 항목 | 사양 | 비고 |
|---|---|---|
| **평량(Basis Weight)** | **70 g/m²** | 단일 시나리오 (v1.3에서 확정) |
| **재질** | 만년필 친화지 | 도모에리버(Tomoe River) 대체재 권장 |
| **불투명도** | 90% 이상 | 만년필 잉크의 뒤비침(Ghosting) 최소화 |
| **색상** | Ivory/Cream 톤 | 본문 잉크 가독성 + 고급감 |
| **표면 처리** | Smooth (글로시 아님) | 만년필·중성펜 친화 |
| **결(Grain)** | 책등과 평행 (Long Grain) | 사철 제본 안정성 |

## 권장 대체재 (한국 제조사 발주 기준)

### 1차 후보 (만년필 친화 최고급)
- **모리사와(Morisawa) Atelier 70g** — 일본산, 도모에리버급 품질
- **두성종이 르베르크림 70g** — 한국산, 고급 다이어리/노트 전용
- **삼화제지 만년필지 70g** — 한국산, 가격대 합리적

### 2차 후보 (실용 우수)
- **한솔제지 매끄러운 책지 80g (감량 검토)** — 70g 비중 시 결 안정성 우수
- **무림제지 미색지 70g** — 비용 효율, 만년필 사용 가능 수준

## 검증 요청 항목 (제조사 회신 시)

견적 회신 시 다음 항목을 함께 명시해 주십시오:

1. **제조사 권장 종이명** (모델·평량·색상 코드)
2. **샘플 제공 가능 여부** (계약 전 손글씨 테스트용 1매)
3. **블리딩(Bleed-through) 테스트 결과** — 만년필(EF-F 닙) + 중성펜 0.38mm 기준
4. **256p 총 두께 추정** — 책등 폭 확정용 (현재 추정 ~20mm)
5. **단가 영향** — 70g vs 80g vs 100g 비교 (옵션)

## 사용자 측 사용 시나리오

본 다이어리의 주요 필기구 시나리오:

| 필기구 | 사용 비중 | 종이 요구 사항 |
|---|---|---|
| 만년필 (EF-F 닙) | 40% | 잉크 흡수 + 마름 시간 단축 + 뒤비침 최소 |
| 중성펜 (0.38-0.5mm) | 50% | 균일한 잉크 흐름, 번짐 방지 |
| 연필·색연필 | 10% | 그립감, 지우개 사용 가능 |

→ 위 시나리오 우선순위로 종이 선정

## 환경 요인

- **습도 변화 대응**: 70g지의 한계 — 한국 여름 고습도(70%+) 환경에서 표면 마찰 증가 가능
- **권장 보관**: 직사광선 회피, 상온 18-25℃, 습도 50-60%
- **다이어리 사용자 안내문 포함 권장** (Owner Profile 페이지 또는 부록에 1줄)

## 책등 두께 계산 (참고)

```
종이 두께 (g/m² → mm 환산 추정)
─────────────────────────────────
70g (만년필 친화) ≈ 0.075~0.085 mm/장
256p ÷ 2 = 128장 (양면 인쇄)
128장 × 0.080 mm = 10.24 mm (본문)
+ 면지 4장 (양쪽) × 0.15 mm = 0.6 mm
+ PU 양장 커버 두께 = 약 3~4 mm
+ 사철 풀제본 두께 = 약 2~3 mm
─────────────────────────────────
총 책등 폭 추정 ≈ 16~20 mm
```

**실제 확정**: 제조사 견본 작업 후 ±2mm 보정.
