### 1.2 채널 1 (★★★★★): 기존 결제고객 리마케팅 — "0원 CAC의 보물"

**왜 이게 1순위인가?**
- 9,900원이라도 **이미 결제 행동을 한 사람**의 LTV는 신규 방문자 대비 **10배**
- 평균적으로 1차 결제 고객의 30~50%가 업셀 제안에 반응 (이커머스 표준)
- 광고비 ₩0, 이메일·카카오톡 발송 비용만 ₩100/통

**실행안**:

**Step 1. 자료 정리 (Day 1)**
- 기존 9,900원 결제자 명단 추출 (이름·이메일·휴대폰·검사일·검사결과)
- 카카오톡 채널 친구 명단 추출
- 뉴스레터 구독자 명단 추출

**Step 2. 세그멘트 (Day 2)**
- A 그룹 (Hot): 최근 30일 이내 결제자 → 다이어리 즉시 제안
- B 그룹 (Warm): 30~90일 전 결제자 → "다이어리로 진짜 살아내기" 스토리 우선
- C 그룹 (Cold): 90일+ 전 결제자 → 재참여 캠페인 후 다이어리 제안

**Step 3. 메시지 (Day 3~7)**
- 카카오톡 알림톡 #1: "다이어리 사전 예약 — 검사 받으신 분께 먼저 알립니다 (200명 한정)"
- 이메일 #1: "당신의 인생포트폴리오 검사 결과가 — 단 1권의 다이어리가 됩니다"
- 마이페이지 배너: "내 검사 결과로 만든 다이어리 사전 예약하기"

**예상 결과**:
- 명단 3,000명 기준
- 카카오톡 열람율 60% × 클릭 10% × 전환 20% = **36명 즉시 결제 (₩89,000 × 36 = ₩320만)**
- 이메일 열람율 30% × 클릭 5% × 전환 15% = **7명 결제**
- 사전 예약 (결제 X, 이메일만) = 추가 200명+

**📍 CAC = ₩2,000 미만 / LTV = ₩89,000~189,000 / ROAS = 30x+**

### 1.3 채널 2 (★★★★★): 텀블벅 펀딩 — "시장 검증 + 제작비 회수 동시"

**왜 이게 2순위인가?**
- 한국 다이어리 펀딩 평균 달성률 = **626%** (재테크 다이어리 사례)
- 드로우앤드류 키트 = **2.5억원 누적** (자기계발 다이어리)
- BEP 135부, 목표 500부 = 안정적 흑자 + 마케팅 효과

**실행안**:

**1) 펀딩 페이지 전략**
- 헤드라인: "65만 유튜버도, 베스트셀러 100만부도 못 풀었던 문제: 의미를 살아내는 다이어리"
- 서브: "검사 → 리포트 → 다이어리. 당신의 인생 검사 결과로 만든 단 한 권."
- 영상 90초 (스토리: 김영식 창업가 6년 프랭클린 → 인생포트폴리오 시작 → 이 다이어리까지)
- 사진 30+ (제품·라이프스타일·상세)

**2) 리워드 구조** (앵커링 최적화)

| Tier | 가격 | 구성 | 수량 | 목표 |
|---|---|---|---|---|
| 1. Super Early Bird | ₩59,000 | 다이어리 단독 (개인화 X) | 100개 한정 | 빠른 펀딩 추진력 |
| 2. Early Bird | ₩69,000 | 다이어리 단독 (개인화 O) | 200개 | 핵심 매출 |
| 3. Standard | ₩89,000 | 다이어리 + 분기 미션북 | 무제한 | 객단가 상승 |
| 4. Premium | ₩159,000 | 다이어리 + 1:1 코칭 1회 | 50개 한정 | 코칭 진입자 확보 |
| 5. VIP 연간 | ₩390,000 | 다이어리 + 코칭 3회 + VIP 카톡방 | 20개 한정 | 최고 LTV 고객 |

**3) 펀딩 진행 시뮬레이션 (보수적)**

| Tier | 단가 | 수량 | 매출 |
|---|---|---|---|
| 1. Super EB | ₩59,000 | 100개 (완판) | ₩590만 |
| 2. EB | ₩69,000 | 200개 (완판) | ₩1,380만 |
| 3. Std | ₩89,000 | 150개 | ₩1,335만 |
| 4. Premium | ₩159,000 | 30개 | ₩477만 |
| 5. VIP | ₩390,000 | 10개 | ₩390만 |
| **합계** | — | **490개** | **₩4,172만** |

**📍 CAC = 펀딩 수수료 5% + 광고 ₩5,000/명 = 약 ₩9,000~10,000 / ROAS = 9x**

**4) 펀딩 사전 준비 체크리스트**
- [ ] D-30: 텀블벅 계정 + 메이커 인증
- [ ] D-21: 페이지 제작 (디자인 외주 ₩200만 or 자체 제작)
- [ ] D-14: 사전 알림 신청 모집 (목표 500명 — 1순위 채널 활용)
- [ ] D-7: 인플루언서 5명 사전 시딩 (샘플 제공 + 펀딩 첫날 동시 게시)
- [ ] D-1: 런칭 메시지 작성 + 카운트다운 콘텐츠

**📌 텀블벅 성공 공식** (텀블벅 자체 데이터):
- 첫 24시간 펀딩률 30%+ 달성 시 → 최종 200% 달성 확률 80%
- 첫 24시간을 위해 **사전 알림 500명 + 인플루언서 5명**이 결정적

### 1.4 채널 3 (★★★★☆): 유튜브 인플루언서 시딩

**타깃 채널 리스트** (자기계발·라이프스타일):

| 인플루언서 | 구독자 | 페르소나 | 추정 단가 | 예상 효과 |
|---|---|---|---|---|
| **드로우앤드류** | 65만 | P2 직격 | ₩500~1,000만 | 최강. 본인이 자기계발 다이어리 만든 사람 → 협업 가능 |
| **자청** (역행자) | 130만 | P2·P3 | ₩1,000~2,000만 | 4세대 수첩 철학과 정확히 일치 |
| **신사임당** | 180만 | P2·P3 | ₩2,000만+ | 사업가 트랙. 가능 시 폭발적 |
| **김작가TV** | 90만 | P3 | ₩500~1,000만 | "인생 2막" 정확한 페르소나 |
| **너진똑** | 100만 | P2 | ₩800~1,500만 | 책 요약 → 다이어리 자연 연결 |
| **체인지그라운드** (고영성) | 60만 | P2 | ₩500~800만 | "메타인지·자기관리" 핵심 메시지 |
| **러닝메이트** | 35만 | P2 | ₩300~500만 | 자기계발 핵심층 |

**시딩 전략 — 3단계**:

**1단계 (저비용 시작)**: 마이크로 인플루언서 (구독자 1~10만) 10명
- 각 ₩30만~100만 = 총 ₩500~1,000만
- 다이어리 무상 제공 + 코드 ("크리에이터명-다이어리 결제 시 ₩20,000 할인")
- 예상 클릭 5,000명 × 전환 3% = 150명 결제 (₩89,000 × 150 = ₩1,335만)
- **ROI = 1.3x~2.7x (손익분기 이상)**

**2단계 (검증 후 확대)**: 중간 인플루언서 (10~50만) 5명
- 각 ₩200~500만 = 총 ₩1,500~2,500만
- 예상 전환 500명 = ₩4,450만 매출
- **ROI = 1.8x~3x**

**3단계 (스케일)**: 메가 인플루언서 1~2명
- ₩500~2,000만 = 1명
- 예상 전환 1,000~3,000명 = ₩9천만~2.7억 매출
- **ROI = 4x~18x** (히트 시)

**📍 핵심 KSF**: 인플루언서가 **"진짜 사용 후기"**를 찍어야 함. 단순 PPL은 효과 절반. 사용 1~2주 후기 영상이 가장 강력.

### 4.2 카피 핵심 원칙

1. **"₩9,900에서 ₩89,000"이 아니라 "₩177,000에서 ₩79,000"** → 앵커 변경
2. **"오늘만"** = 행동 압박 (urgency)
3. **옵션 C 제공** = 거절 대신 "나중에" 선택지 → 이메일 리타깃팅 자산화
4. **추가하기 버튼** = "결제하기" 아닌 "추가" → 심리적 부담 감소

### 4.3 예상 효과 (O.제안서 가정 적용)
