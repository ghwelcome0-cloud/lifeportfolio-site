우리는 무엇을 파는 회사인가

한 줄 정의

“인생포트폴리오는 각 사람 안에 숨겨진 사명과 비전을 발견하게 하고, 그것을 살아낸 삶이 또 다른 누군가의 양식이 되는 구조를 만드는 자기경영 시스템입니다.”

검사 결과지가 아닙니다. ‘나를 설명하는 글’이 아닙니다. 당신만의 방향을 발견하고, 첫 행동까지 연결되는 살아있는 인생 설계도입니다.

CORE PROMISE

Only One 인생

유형으로 분류하지 않습니다.
당신의 답변 그대로, 당신만을 위한 리포트로 설계됩니다.

76문항이 만들어내는 답변 조합은 약 10⁵³가지 이상으로, 통계적으로도 동일한 결과가 반복되기 어려운 구조입니다. 여기에 5가지 어조 매칭이 더해져, 고객은 ‘유형’이 아닌 ‘고유한 한 사람’으로서 자신의 답변이 그대로 반영된 리포트를 받습니다 — 그것이 인생포트폴리오의 약속입니다.

한눈에 보는 핵심 성과

마케팅 핵심 메시지 5가지 (즉시 광고 카피화 가능)

  1. Only One 리포트 – 76문항이 만드는, 세상에 단 하나뿐인 인생 설계도.
  2. ‘설명’이 아니라 ‘방향’ – 나를 묘사하는 글이 아니라, 다음 한 걸음을 정리하는 도구.
  3. 결제 후 즉시 발급 – 기다림 없는 자기성찰. 검사 직후 자동 생성.
  4. 실행까지 자동 연결 – 3주 루틴 + 3개월 구조화 + 1년 흐름까지.
  5. 나를 위한 어조 – 5가지 톤 중 가장 잘 맞는 화법으로 자동 매칭.
마케터 활용 팁: 광고 헤드라인은 ‘Only One’과 ‘즉시 발급’ 두 후크를 A/B 테스트로 비교하는 것을 권장합니다. 특히 “당신만의 인생 설계도”는 30~40대 자기계발 타겟에서 가장 강한 후킹력이 예상됩니다.

Only One의 수학적 근거 — “유형 검사가 아닙니다”

왜 ‘5가지 유형’이 아니라 ‘Only One’인가

많은 검사가 ‘OOO형 인간’식으로 사람을 몇 가지 박스에 넣습니다. 인생포트폴리오는 정반대 철학으로 설계되었습니다. 사람을 분류하지 않습니다. 사람을 발견합니다.

기존 유형 검사인생포트폴리오 (Only One)
16가지 / 9가지 / 5가지 유형경우의 수 사실상 무한
같은 유형 = 같은 결과지같은 유형 개념 자체가 없음
“당신은 어떤 사람인지”“당신만의 방향과 첫 행동”
이해에서 끝남실행까지 자동 연결

경우의 수 계산 (보수적 추정)

항목경우의 수
76문항 × 5점 척도 답변 조합5⁷⁶ ≈ 3.7 × 10⁵³가지
× 어조(톤) 자동 매칭× 5
최종 결과 변형 수≈ 1.85 × 10⁵⁴가지
참고 — 전 세계 인구8 × 10⁹명
인류 전체가 검사 시 동일 리포트 확률사실상 0%
“5가지 톤”의 진짜 의미: 사용자를 5가지로 분류하는 것이 아닙니다. 각자에게 맞춤화된 결과를 전달하는 어조가 5가지인 것이며, 그 어조 안의 콘텐츠는 사람마다 모두 다릅니다. 비유하자면 요리사가 5명이지만 메뉴는 인구 수만큼 다른 구조입니다.

사용자 여정 (Funnel) – 한 번의 클릭으로 완결

End-to-End 자동화 흐름

# 단계 사용자가 하는 일 시스템이 자동 처리하는 일
1회원가입구글 계정 1초 로그인Firebase Auth 인증, 신규/기존 자동 분기

5리포트 확인웹/PDF로 즉시 열람jsPDF로 고품질 PDF 자동 변환

핵심 자랑거리: 1번부터 7번까지 사람의 수동 개입이 0회입니다. 100명이 결제해도, 1만 명이 결제해도 동일한 품질·속도로 자동 처리됩니다.

광고 소재로 쓰기 좋은 카피 예시 (Only One 후크)

  • “세상에 단 하나뿐인 당신의 인생 설계도, 15분이면 받습니다.”

  • “방향은 아는데 실행이 안 되는 분께. 첫 3주를 정리해 드립니다.”
  • “이해에서 끝나는 검사는 그만. 이번 주부터 움직이는 설계도.”
  • “결제하면 끝? 결제하면 시작입니다. 1년 흐름까지 자동 제공.”

차별화 포인트 – 경쟁사 대비 강점

1) Only One 자동화 엔진 (Engine v1.3)

  • 리포트 룰 엔진 + 프로그램 룰 엔진 2단 구조로 완전 자동화.
  • 사용자별 답변·강점·성장 포인트가 결합되어 사람마다 다른 결과가 자동 생성.
  • 외부 룰(JSON)만 갱신하면 콘텐츠 업그레이드 즉시 반영 → 운영 민첩성.

2) 모바일 100% 반응형 UX

  • 홈, 상품, 검사, 결제, 리포트, 마이페이지 등 전 페이지 반응형.
  • 아이폰 인앱 브라우저(인스타·카카오 등) 환경 전용 가이드 메시지 제공.
  • iOS Safari 전용 리다이렉트 로그인 흐름으로 팝업 차단 우회.

3) 똑똑한 회원 분기 UX (이탈 방지)

  • 신규 회원 → 자동 안내 토스트 후 결제 페이지.
  • 이미 결제한 회원 → 마이페이지로 자동 안내 (중복결제 방지).
  • 가입했지만 결제 전인 회원 → 결제 페이지로 친절히 유도.

→ 이탈 가능 지점에서 사용자 혼란을 제거, 전환률 상승 효과.

4) PDF 출력 고도화

  • jsPDF 2.5.1 + html2canvas 1.4.1 기반 고해상도 PDF 생성.
  • 한글 폰트 임베딩, 페이지 분할 시 글자·이미지 깨짐 방지.
  • 리포트·프로그램 모두 PDF로 저장 가능 → 오프라인 활용 가능.

적용된 보안 기술 상세

  • App Check (reCAPTCHA Enterprise): 봇·자동화 공격 차단.
  • Content Security Policy (CSP): XSS·악성 스크립트 주입 방어.
  • Anti-Clickjacking Guard: 외부 사이트의 iframe 임베드 차단.
  • Realtime DB 보안 룰: 본인 UID 외 접근 불가, 결제 플래그 변조 불가, 리포트 수정 횟수 3회로 제한.
  • 입력 새니타이즈: 회원 정보 저장 시 위험 문자 자동 제거.
마케팅 활용: 랜딩 페이지에 “Google 인증 · Payple 결제 · 4중 보안” 뱃지를 노출하면 결제 신뢰도 상승으로 직결됩니다. 특히 30~50대 여성 타겟에서 효과 큼.

운영 안정성 보장

  • 가입자 과부하 시 자동 확장 (Firebase 인프라 기반).
  • 결제 후 리포트 생성 실패 시 자동 재시도 + 사용자 안내.
  • 모든 변경 이력 GitHub로 관리 → 즉시 롤백 가능.

비즈니스 가치 — 부채꼴처럼 확장되는 수익 구조

핵심 수익 구조 한 줄

“진단 → 실행 점검 → 코칭 → 라이선스/IP화”로 이어지는 4단 사다리 구조로, 한 명의 고객이 단계를 따라 올라갈수록 객단가가 높아지고 수익성이 부채꼴처럼 확장됩니다.”

상품 라인업 — 부채꼴형 가치 사다리 (Value Ladder)

단계 상품 포지셔닝 고객이 받는 가치
1단
진입		인생포트폴리오 진단 + Only One 리포트저관여·고전환방향 한 줄 정리, 첫 행동 3가지
2단
실행		21일 실행 점검 패키지브리지 오퍼3주 루틴 정착 + 중간 점검
3단
심화		1:1 코칭 / 그룹 멘토링고객단가 상향3개월 흐름 동행, 실행 성과 가속
4단
확장		콘텐츠 IP화 · 라이선스 · 파트너십레버리지 수익고객의 살아낸 삶이 다시 양식이 되는 흐름

※ 단계가 올라갈수록 객단가는 커지고, 한 사람의 변화 데이터가 다음 고객에게 양식으로 재사용되며 수익이 누적·확장됩니다.

마케팅 시사점: 1단(진단) 광고 ROAS만 보지 말고, 1단 → 2단 전환율과 3단 LTV까지 함께 봐야 진짜 그림이 나옵니다. 퍼널 전체에서 한 사람의 가치는 처음 진단가의 수십 배로 확장될 수 있는 구조입니다.

운영의 강점 (수치 대신 구조로 보기)

  • 고정비 부담 최소화 인프라: 무료 한도 인프라(Firebase Hosting·Firebase Spark·App Check) 위에 구축되어, 트래픽이 늘어도 비용 곡선이 완만합니다.
  • 자동화에 의한 한계비용 감소: 결제 → 검사 → 리포트 → 실행 프로그램이 사람 손 없이 자동 처리되어, 고객이 늘어도 운영 인력이 비례해서 늘지 않습니다.
  • 지출 통제 장치: Google Cloud 예산 알림(월 10,000원)으로 예상치 못한 과금을 사전 감지합니다.
  • 확장 옵션 완비: 카카오페이·네이버페이·카카오 로그인 확장 설계 완료 → 매출 성장 시 추가 채널 즉시 가동 가능.

※ 본 자료는 운영 구조의 강점을 정리한 것이며, 구체적인 비용·수익 수치는 내부 재무자료를 기준으로 합니다.

FAQ – 광고/콘텐츠 제작에 인용 가능

  • Q. 다른 검사와 무엇이 다른가요?
    A. 유형으로 분류하지 않습니다. 76문항이 만드는 약 10⁵³가지 조합으로, 사람마다 단 하나뿐인 결과를 만듭니다.
  • Q. 리포트는 언제 받을 수 있나요?
    A. 결제 후 검사를 마치면 즉시 자동 생성됩니다. 평균 검사 시간은 15~20분입니다.
  • Q. 리포트 외에 더 받는 게 있나요?
    A. 네, 3주·3개월·1년 단위의 맞춤 실행 프로그램을 추가로 즉시 생성해 드립니다.
  • Q. 결제는 안전한가요?
    A. 정식 PG사 Payple과 Google 인증을 사용하며, 4중 보안이 적용되어 있습니다.
  • Q. 리포트는 다시 볼 수 있나요?
    A. 회원이신 동안 마이페이지에서 언제든 다시 보실 수 있고, PDF로도 받을 수 있습니다. (자세한 보유 기간은 개인정보처리방침을 따릅니다.)
광고 카피 작성 시 주의 (컴플라이언스): “영구 보관 / 평생 보관 / 영구 저장” 같은 표현은 사용 금지입니다. 개인정보 보호법(제21조 파기 의무) 및 표시·광고법(거짓·과장 광고 금지)에 저촉될 수 있으며, 당사 개인정보처리방침과도 충돌합니다. 권장 표현: “회원 이용 기간 동안 마이페이지에서 다시 열람”.
