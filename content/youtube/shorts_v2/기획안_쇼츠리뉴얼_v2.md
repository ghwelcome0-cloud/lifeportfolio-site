# 인생포트폴리오 쇼츠 리뉴얼 기획안 v2 — "칠판 강의" 시리즈

> **검토용 문서.** 제작 착수 전 컨셉·구성·제작 방식을 확정받기 위한 기획안입니다.
> 원본: `서비스 메뉴얼 v1.1` (24p, 8개 장) · 컨셉: 강사가 칠판에 분필로 적어내려가는 강의

---

## 0. 왜 다시 만드는가 — 기존 18개 진단 (데이터 기반)

| 문제 | 진단 결과 (실측) | 원인 |
|------|------------------|------|
| **분량/퀄리티 부족** | 18개 중 14개가 **7~13초** (평균 ~12초) | 롱폼을 기계적으로 18분할 → 각 조각이 완결성·후킹 없음 |
| **에버그린 효과 없음** | 후킹·루프·CTA 구조 부재 | "정보 나열"만 있고 "한 편이 하나의 완결된 깨달음"이 아님 |
| **7초 후 음성 작아짐** | 오디오 RMS **-41 ~ -44dB** (정상은 -16~-20dB) | **라우드니스 정규화 미적용** + 구간별 볼륨 불균일 |

**핵심 결론:** 18분할이 문제의 근원. → **개수가 아니라 "완결된 한 편"** 으로 재설계한다.

## 1. 시장조사 결론 → 적용할 5대 원칙 (출처: opus.pro, miraflow 2025/26)

| # | 원칙 | 벤치마크 수치 | 우리 적용 |
|---|------|---------------|-----------|
| 1 | **길이** | 교육 쇼츠는 **25~45초**가 최적 (35~45s 페이오프 명확 시) | 목표 **30~42초** (기존 12초 → 3배) |
| 2 | **3초 후킹** | 첫 3초 이탈 50~60% → **인트로 리텐션 70%+** 목표 | 0~3초에 **칠판에 핵심 한 줄**이 빠르게 써짐 + 질문형 후크 |
| 3 | **평균 리텐션** | 60% 최저, **70%+ 우수, 80%+ 바이럴** | 컷 전환 2~4초 1회 + 자막 word-by-word |
| 4 | **루프** | 엔딩이 시작으로 자연 연결 → 재생 유도 | 마지막 컷 = 첫 칠판 글자로 회귀 (페이드) |
| 5 | **자막** | 번인 word-by-word 자막 → 리텐션 +15~25% | 칠판 손글씨 자막 + 하단 강조 자막 이중 |

> **정보 밀도 규칙:** 한 편당 **핵심 메시지 1개 + 근거 1~2개**. 절대 더 넣지 않는다(3~5스텝 한도).

## 2. 컨셉 — "The North Star 칠판 강의실"

**비주얼 컨셉:** 어두운 강의실, 짙은 녹색(혹은 흑색) 칠판. 보이지 않는 강사(손/분필만 등장 또는 손 없이 글씨가 써짐)가 **핵심 개념을 분필로 또박또박 적어내려간다.** 슬라이드가 잘 짜여 있으므로 **매뉴얼 8장 = 강의 8교시**로 매핑.

**무드보드 키워드:**
- 따뜻한 텅스텐 조명, 분필 가루 입자, 손글씨가 "사각사각" 써지는 ASMR감
- 컬러: 칠판 딥그린 `#1f3a2e` / 분필 화이트 `#f5f1e8` / 강조 옐로우 `#f0c040` (North Star)
- 폰트(자막): 손글씨체(나눔손글씨/Gaegu) + 가독성 산세리프 병기
- 브랜딩: 우하단 작게 "인생포트폴리오 · The North Star" 워터마크

**왜 칠판인가 (시장조사 근거):** 화이트보드/칠판 애니메이션은 **인지 부하를 낮추고 기억·몰입·완주율을 높이는** 것이 학술적으로 입증됨(PMC 연구). 손글씨가 "지금 그려지는 중"이라는 진행감이 스크롤을 멈추게 함.

## 4. 한 편의 내부 구조 (모든 편 공통 템플릿)

```
[0-3s]  HOOK    칠판에 핵심 질문/도발 한 줄이 빠르게 써짐  ← 70% 리텐션 관문
                예) "MBTI를 안다고 '나'를 아는 걸까?"
[3-10s] 문제제기  공감되는 현실/통증 (분필로 키워드 1~2개 추가)
[10-30s]전개     핵심 개념을 분필로 단계적으로 적으며 설명 (컷/줌 2~4초마다)
                + 데이터 수치는 크게 강조 (89% / 75.3% / 76문항)
[30-38s]정리     한 문장으로 압축 → 칠판 중앙에 밑줄
[38-42s]루프/CTA "당신의 Only One은?" → 첫 화면으로 페이드(루프)
                + 마지막 0.5초 브랜드 워터마크
```

**자막 정책:** 칠판 손글씨(메인) + 하단 번인 자막(word-by-word 강조). 무음으로 봐도 이해되도록.

## 5. 제작 방식 (오디오 문제 완전 해결 포함)

### 5-1. 비주얼 제작
- **칠판 배경 + 손글씨 애니메이션**: AI 이미지 생성(칠판 강의실 무드) + 텍스트 손글씨 "써지는" 효과를 ffmpeg/모션으로 합성. (필요 시 짧은 AI 영상클립으로 분필 손 모션 추가)
- **컷 전환**: 2~4초마다 줌인/줌아웃 또는 칠판 영역 이동으로 시선 리셋.

### 5-2. 오디오 — ★ 기존 결함 완전 해결
| 항목 | 기존 | 개선 |
|------|------|------|
| 볼륨 | RMS -41~-44dB (너무 작음) | **EBU R128 라우드니스 정규화 -16 LUFS** 통일 |
| 구간 불균일 | 7초 후 작아짐 | **전 구간 동일 음량** (loudnorm 1-pass + limiter) |
| 내레이션 | — | TTS(차분한 강사 톤) + 분필 사각사각 ASMR + 잔잔한 BGM(-26dB 사이드체인) |

**오디오 파이프라인:** `TTS → loudnorm(I=-16:TP=-1.5:LRA=11) → BGM 믹스(덕킹) → 최종 -14 LUFS 마스터`

### 5-3. 기술 스택
- TTS: 차분하고 신뢰감 있는 한국어 강사 보이스 (audio_generation)
- 배경: image_generation (칠판 강의실, 일관된 톤)
- 합성: ffmpeg (손글씨 reveal, 자막 번인, 오디오 정규화, 9:16 1080×1920)
- 포맷: 1080×1920, H.264, 30fps, AAC -14 LUFS
