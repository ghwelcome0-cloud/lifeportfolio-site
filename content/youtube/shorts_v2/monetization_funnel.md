# 인생포트폴리오 수익화 퍼널 설계서 v1.0
### 쇼츠(에버그린·당김) → 롱폼 → 홈페이지 → 9,900원 결제

> 작성: 마케팅·수익 전문가 관점 / 대상: 총괄(사장님)
> 핵심 KPI: **쇼츠 1만뷰당 결제 전환 수** (= 1만뷰 → 클릭률 → LP방문률 → 결제율)
> 제품: ONLY ONE 진단 리포트(76문항·15분) **9,900원** = 진입 상품

---

## 0. 한 줄 전략

> **쇼츠는 "팔지 않는다". 쇼츠는 비교에 지친 사람의 마음에 '나도 저 질문이 있었다'는 인식을 심는다.**
> 파는 일은 롱폼과 홈페이지가 한다. 쇼츠의 유일한 임무는 **'저장·구독·다음 영상'** 으로 시청자를 묶어두는 것.

돈을 버는 구조는 단순합니다. **싸게 사서 비싸게 파는 게 아니라, 신뢰를 쌓아 결제 저항을 0에 가깝게 만드는 것.**
9,900원은 "지를까 말까"가 아니라 "안 할 이유가 없는" 가격이어야 합니다. 그 심리 상태를 만드는 게 퍼널의 전부입니다.

---

## 1. 퍼널 4단계 구조 (AARRR 기반)

```
[1] 발견(Awareness)      쇼츠 12편 에버그린 — 매일 YT/IG 교차 발행
        │  목표: 3초 후킹 → 끝까지 시청 → "저장/팔로우"
        ▼
[2] 관심·신뢰(Interest)  롱폼 8분 21초 — 쇼츠 더보기/고정댓글/프로필 링크
        │  목표: 의미경제·번아웃 데이터·강영우·달란트·ONLY ONE 논리로 "납득"
        ▼
[3] 욕구·검증(Desire)    홈페이지 lifeportfolio.co.kr — 랜딩
        │  목표: "나도 80억분의 1" → 76문항 미리보기 → 결제 버튼
        ▼
[4] 전환(Action)         9,900원 결제 → 15분 진단 → ONLY ONE 리포트
        │  목표: 결제 완료 + 리포트 만족 → 후기/공유 → [1]로 재유입 (루프)
        ▼
[5] 재방문·확산(Referral) 리포트 공유카드 / 재진단 / 상위 상품(미래 LTV)
```

### 각 단계 전환 가정(보수적 벤치마크)
| 단계 | 전환 지표 | 보수 | 현실목표 | 공격 |
|---|---|---|---|---|
| 쇼츠 → 프로필/링크 클릭 | CTR | 0.8% | 1.5% | 3% |
| 클릭 → 홈페이지 도달 | 도달률 | 60% | 75% | 85% |
| 홈페이지 → 결제 시작 | LP전환 | 3% | 6% | 10% |
| 결제 시작 → 완료 | 결제완료 | 70% | 85% | 92% |

**역산 예시(현실목표):** 쇼츠 누적 100만뷰 →(1.5%) 1.5만 클릭 →(75%) 1.1만 LP방문 →(6%) 675 결제시작 →(85%) **약 574건 결제 = 약 568만원** (9,900원 기준). 광고비 0원(오가닉)이면 전액 마진에 가깝습니다.

> 핵심 레버는 **LP전환율(3→6→10%)**. 여기가 1%p 오르면 매출이 직접 17~30% 움직입니다. 쇼츠 조회수를 늘리는 것보다, **홈페이지 랜딩을 다듬는 게 ROI가 훨씬 높습니다.**

## 2. 단계별 상세 설계

### [1] 쇼츠 — 발견 엔진 (에버그린 + 당김)

**역할 분담 (사장님 확정 사항 반영)**
- **에버그린**: 1년 뒤에 봐도 유효한 본질 메시지 (의미경제·고유성·ONLY ONE) → 검색·추천 유입의 꾸준한 베이스
- **당김(Pulling)**: 댓글 질문/시의성 후킹으로 "지금 클릭"하게 만드는 영상 → 알고리즘 초기 부스트
- **발행**: 매일 1편, **YouTube ↔ Instagram 교차** (같은 영상 동시 X, 하루 단위 번갈아 → 두 플랫폼 알고리즘 모두에 매일 신호)

**쇼츠 안에서 절대 하지 말 것**
- ❌ "9,900원" 가격을 직접 외치기 (쇼츠에서 가격 노출 = 광고로 인식 → 도달 급감)
- ❌ 외부링크를 영상 중간에 강요 (이탈)
- ✅ 대신 → **"더 깊은 이야기는 채널에서"(고정 CTA)** + 끝맺음 루프로 다음 영상 유도

**쇼츠 → 다음 단계 연결 장치 (3중)**
1. **고정 댓글**: "👉 전체 이야기(8분) + 무료 자가진단 미리보기: [링크]" (첫 댓글 고정)
2. **프로필 링크**: YT 채널 배너/IG 바이오 → `lifeportfolio.co.kr/start`
3. **영상 엔드카드**: 마지막 0.5초 "인생포트폴리오 · 채널에서 계속" (현재 EP1 적용됨)

**12편 발행 순서 전략 (후킹 강도 순 배치)**
- 1주차(강한 후킹): EP1(효율의 시대), EP4(번아웃 75.3% 데이터), EP10(AI가 못 흉내내는 것)
- 2주차(논리·납득): EP2(의미경제), EP6(ONLY ONE 진단), EP7(80억분의 1)
- 3주차(공감·확산): EP5(세대별 고민), EP8(발견·살아냄·남김), EP12(순환)
- 분산: 스토리(강영우/달란트)·신뢰(300회 개선)는 사이사이 배치 → 신뢰 누적

### [2] 롱폼 — 신뢰·납득 엔진 (8분 21초, 이미 보유)

쇼츠가 "질문"을 던졌다면, 롱폼은 "그 질문에 답할 자격이 우리에게 있다"를 증명합니다.
이미 녹음된 롱폼 내레이션(501초)이 **그대로 설계서**입니다 — 의미경제→번아웃 데이터→강영우→달란트→ONLY ONE→3동사 순서가 곧 설득 논리.

**롱폼의 결제 전환 장치**
- 6:10 지점(달란트~ONLY ONE) 이후 → 화면 하단 고정 "👇 9,900원 15분 진단" 자막 배너
- 영상 설명란 최상단: `▶ 지금 시작: lifeportfolio.co.kr` (UTM 태깅: `?utm_source=youtube&utm_medium=longform&utm_campaign=ep_main`)
- 고정 댓글에 **타임스탬프 챕터** (시청 유지율↑ = 추천↑)
- 카드/엔드스크린: 홈페이지 + 쇼츠 재생목록

> 롱폼은 "한 번 보면 결제하고 싶어지는" 영상이어야 합니다. 가장 강력한 전환 멘트는 이미 내레이션 470초에 있습니다: **"첫걸음은 9,900원 15분."** — 이 부분을 영상에서 시각적으로 강조(자막+버튼)하세요.

---

### [3] 홈페이지 랜딩 (lifeportfolio.co.kr) — 전환 엔진 ★최우선 최적화

**랜딩 페이지 골든 구조 (스크롤 순서)**
1. **히어로 (3초 안에 승부)**
   - 카피: *"비교하지 않아도 되는, 단 하나뿐인 인생 설계도"* (= EP1 엔딩과 동일 → 쇼츠에서 온 사람이 "아 그거!" 인식)
   - 서브: "76문항 15분 · 80억분의 1 ONLY ONE 진단"
   - CTA 버튼: **[9,900원으로 시작하기]** (스크롤 없이 첫 화면에 노출)
2. **공감 (당신의 문제)**: "AI에게 물으면 답은 3초. 그런데 나는 왜 뒤처진 것 같을까?" (쇼츠 메시지 재현)
3. **반전 (의미경제)**: "이제 '무엇을'이 아니라 '왜'가 일이 되는 시대"
4. **증거 (신뢰)**: 번아웃 75.3% / ChatGPT 2천만 / 300회+ 개선 / 응답 비공개·삭제 가능
5. **차별점 (ONLY ONE)**: "MBTI는 다리, 그 다음 빈칸을 채웁니다" + 리포트 샘플 이미지
6. **가격 정당화**: "커피 두 잔 값으로 15분 만에 평생의 좌표" / 환불·개인정보 안심 배지
7. **최종 CTA + FAQ**: 결제 버튼 재노출 + "15분이면 되나요? / 결과는 어떻게 받나요?"

**전환율 끌어올리는 5대 장치**
- ⏱ **무료 맛보기**: 76문항 중 3~5문항 무료 체험 → "여기까지 했는데 결과 안 볼래?" (sunk cost)
- 🔒 **결제 저항 제거**: 9,900원 단건 = 구독 아님 명시 / 카카오페이·네이버페이 원클릭
- 📊 **결과 미리보기**: ONLY ONE 리포트 한 페이지 블러 처리 노출 (호기심)
- 💬 **사회적 증거**: 진단 완료자 후기 3~5개 (실명 X, 한 줄 인용)
- 🎯 **이탈 방지**: 나가려 할 때 "잠깐, 무료 진단 미리보기부터" 모달

> **여기가 매출의 70%를 결정합니다.** 쇼츠/롱폼은 트래픽을 만들 뿐, 돈은 이 페이지에서 결정됩니다. A/B 테스트 1순위 = 히어로 카피 & CTA 버튼 색/문구.

---

### [4] 결제 → 진단 → 리포트 (전환 완료 + 만족)

- **결제 단계 최소화**: 클릭 → 결제 → 바로 진단 시작 (회원가입을 진단 후로 미루기 = 이탈↓)
- **15분 진단 중 이탈 방지**: 진행률 바 + "거의 다 왔어요" 마이크로카피
- **리포트 수령 = 다음 루프의 시작**:
  - 리포트에 **공유용 ONLY ONE 카드**(1장 요약 이미지) → SNS 공유 시 자동 바이럴
  - "내 결과 친구와 비교" (역설적이지만 = 재유입)

---

### [5] 재방문·확산 (LTV 확장 — 향후 로드맵)

9,900원은 **진입 상품(미끼)**. 진짜 수익은 LTV에서 나옵니다.
- **재진단/시즌 리포트**: 6개월 후 "당신은 얼마나 살아냈나요?" 리마인드
- **상위 상품**: 심화 코칭/PDF 인생설계서/구독형 "남김" 아카이브 (제작규칙서 철학의 '남김' 단계 = 자연스러운 업셀)
- **이메일/카톡 수집**: 결제 시 동의 → 신규 쇼츠/롱폼 알림 → 채널 재방문

## 3. 추적·측정 체계 (UTM & KPI 대시보드)

**UTM 규칙 (모든 링크에 태깅)**
```
?utm_source={youtube|instagram}
&utm_medium={shorts|longform|bio}
&utm_campaign={ep1|ep2|...|longform_main}
```
→ 어느 쇼츠가 결제까지 끌었는지 역추적 가능 → 잘 되는 EP에 제작 리소스 집중

**주간 점검 KPI (5개만)**
1. 쇼츠 평균 시청유지율 (목표 60%+) — 후킹/길이 진단
2. 프로필·링크 CTR (목표 1.5%+)
3. **LP 결제전환율 (목표 6%+)** ★최중요
4. 결제 완료율 (목표 85%+)
5. 1만뷰당 결제수 (북극성 지표)

## 5. 즉시 의사결정 필요 항목 (사장님 컨펌 요청)

1. **무료 맛보기 도입 여부** — 76문항 중 3~5문항 무료 체험을 랜딩에 넣을지? (전환율 ↑↑, 개발 필요)
2. **가격 표시 위치** — 쇼츠 X / 롱폼 후반 O / 홈페이지 히어로 O — 이대로 확정?
3. **공유 카드(ONLY ONE 1장 요약)** — 리포트에 SNS 공유용 이미지 자동 생성 기능 추가?
4. **상위 상품(LTV)** — 9,900원 다음 단계 상품을 지금 설계할지, EP 양산 후로 미룰지?

---

> **요약:** 쇼츠로 '질문'을 심고 → 롱폼으로 '신뢰'를 주고 → 홈페이지에서 '확신'을 만들어 → 9,900원의 결제 저항을 0으로 만든다.
> 가장 큰 매출 레버는 **홈페이지 랜딩 전환율**. 쇼츠 조회수보다 랜딩 최적화에 먼저 투자하는 것이 ROI 최고.
