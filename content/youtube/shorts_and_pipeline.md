# 숏츠 풀링 전략 + 제작 파이프라인

## 1. 숏츠 풀링 전략 (에버그린 18편)

키 콘텐츠(10분) 1편 = 숏츠 18편의 저수지. 각 SCENE이 독립 메시지를 갖도록 설계했으므로, 영상 편집본을 그대로 9:16으로 잘라 자막만 키우면 숏츠가 됩니다.

### 발행 캘린더 (예시 — 주 3회 × 6주)
| 주차 | 월 | 수 | 금 | 테마 |
|---|---|---|---|---|
| 1주 | #1 불안의 역설 | #3 번아웃 75.3% | #7 강영우 실화 | 후킹·공감 |
| 2주 | #2 SaaS→의미 | #4 MBTI는 다리 | #8 다섯과 둘 | 통찰 |
| 3주 | #5 4시기 질문 | #9 기준=충실함 | #10 15분 좌표 | 해법 진입 |
| 4주 | #6 내 얘기? | #11 80억 장 | #12 세 동사 | 차별성 |
| 5주 | #13 AI 못 흉내 | #14 운영 중 서비스 | #16 맡겨진 것 | 신뢰 |
| 6주 | #15 데이터는 당신 것 | #17 9,900원 CTA | #18 구독 유도 | 전환 |

### 숏츠 제작 원칙
- **첫 1.5초 = 자막 훅** (질문형). 예: "답이 빨라질수록 왜 더 불안할까?"
- **세로 9:16**, 핵심 문장 큰 자막(골드), 하단 고정 CTA "프로필 링크 → 15분 진단"
- 각 숏츠 끝 1초: North Star 로고 + "인생포트폴리오" → 채널 인지 누적
- 키 콘텐츠 본편으로 유도하는 고정 댓글: "🔗 전체 이야기는 본 영상에서"

### 풀링(funnel) 동선
숏츠(인지) → 본편 10분(설득) → 더보기 링크(전환) → 9,900원 진단(매출) → 21일 동행(₩49,000 업셀)

## 2. 제작 파이프라인 (다음 단계)

### STEP 1 ✅ 대본 (완료)
- `key_content_script.md` — 연출 대본(타임코드·화면·숏츠 분할)
- `tts_narration.md` — TTS 낭독 전용 원고(BLOCK 1~9)

### STEP 2 🎙️ 음성 녹음 (총괄님)
- MiniMax Speech-2.8-HD / Youngsik Kim 보이스, Speed 0.9
- BLOCK 1~9 각각 생성 → `block1.mp3` ~ `block9.mp3`
- 완성 파일을 업로드해 주시면 STEP 3로 진행

### STEP 3 🔧 음성 통합 (제가 처리)
- 9개 블록을 하나의 내레이션으로 병합, 블록 사이 간격(0.8~1.2초) 정리
- 전체 길이 측정 → 영상 타임라인 확정

### STEP 4 🎬 비주얼 제작 (제가 처리)
- 가이드북 비주얼 시스템(다크네이비+골드, North Star)을 모션 키프레임으로
- SCENE별 키네틱 타이포 + 추상 모션그래픽 생성 (image_generation → video_generation)
- 각 SCENE 키프레임 이미지를 먼저 만들고, 5~10초 클립으로 영상화 후 연결

### STEP 5 🎵 음악·SFX (제가 처리)
- 다크/시네마틱 앰비언트 BGM (audio_generation, 장면별 분위기)
- North Star 점멸·전환 SFX

### STEP 6 🪡 싱크·자막·마스터링 (제가 처리)
- 내레이션 + 비주얼 + 음악 싱크
- 한국어 자막(가독성), CTA 카드, 채널 로고/구독 버튼
- 16:9 본편 + 9:16 숏츠 18편 컷 추출
