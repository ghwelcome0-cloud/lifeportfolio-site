## 📌 적용 가이드

1. **채널 설명**: YouTube Studio → 사용자 지정 → 기본 정보 → 채널 설명란에 한국어 버전 입력 (영어권 노출 강화 시 영어 버전을 하단에 추가하거나 별도 언어 설정에 입력)
2. **다국어 설정**: YouTube Studio → 설정 → 채널 → 고급 설정에서 언어 추가 시, 각 언어별 채널 설명을 입력 가능
3. **채널 키워드**: YouTube Studio → 설정 → 채널 → 기본 정보 → 키워드란에 쉼표로 구분하여 입력
4. **첫 2~3줄 원칙**: 검색·미리보기에 노출되는 핵심 구간이므로 "단 하나뿐인 포트폴리오 + 인생 자산화 + 발견·살아냄·남김"이 앞에 배치됨
