# 인생포트폴리오 — YouTube 썸네일 템플릿 (재사용 규격)

> **Brand**: 인생포트폴리오 (Life Portfolio) — 따뜻하고 신뢰감 있는 셀프코칭·커리어 자기진단 채널
> **Brand feeling**: calm confidence(차분한 자신감), clarity(명료함), growth(성장)
> **Spec**: 1280×720px (16:9) · 채널 전 영상 공통 사용

## 🎨 1. 컬러 시스템

| 역할 | 색상 | HEX | 사용처 |
|---|---|---|---|
| 베이스 | 딥 네이비 | `#0B1A2F` | 배경 전체 |
| 베이스 보조 | 네이비 그라데이션 | `#12233D` | 배경 하단/비네팅 |
| **메인 강조** | **에메랄드 그린** | `#2FbF8F` / `#10B981` | 핵심 단어, 포인트 라인, 성장 모티프 |
| 서브 액센트 | 골드(소량) | `#C8A24A` | North Star 점광 1개만 (영상 연속성) |

> **규칙**: 한 썸네일에 강조색은 **에메랄드(메인) + 골드(점 1개)** 까지만. 그 외 색 금지(clutter 방지).

## ✍️ 2. 타이포그래피 규칙

| 항목 | 규격 |
|---|---|
| 한글 폰트 | **나눔스퀘어 ExtraBold / Black** (또는 Pretendard Black) — 고딕, 굵게 |
| 영문/숫자 | 동일 패밀리 또는 Montserrat Black |
| 줄 수 | **최대 2줄** (3~7 단어) |
| 1줄차 | 화이트, 중간 크기 — 상황/질문 도입 |
| 2줄차 | **에메랄드 그린, 가장 크게, 최강조** — 핵심 메시지 |
| 외곽선 | 2~4px 딥네이비 stroke + 약한 그림자 (모바일 가독성) |
| 정렬 | 좌측 2/3 텍스트 존 (좌측 정렬) |

## 🧩 3. 레이아웃 틀 (1280×720)

```
┌─────────────────────────────────────────────┐
│  [상단 24px 안전여백 — 제목/시간 안 가림]        │
│                                               │
│  ┌──────────────── 텍스트 존(좌 65%) ───────┐  │  ◀ 우측 35%: 비주얼 존
│  │ 1줄차 (화이트, 중)                        │  │     (실루엣 / 모티프 /
│  │ ★2줄차 (에메랄드, 대·최강조)             │  │      North Star 골드점 1개)
│  │ ── 에메랄드 언더라인 (성장 모티프)        │  │
│  └──────────────────────────────────────────┘  │
│                                               │
│  [로고: "인생포트폴리오" 좌하단, 작게, 에메랄드] │
│  [하단 우측 56px — 영상시간 표시 영역 비움]      │
└─────────────────────────────────────────────┘
```

**고정 규칙**
1. **1 focal point**: 비주얼 존엔 요소 1개만(실루엣 OR 모티프). 70:30 = 텍스트:비주얼.
2. **안전 여백**: 상단 24px, 하단 우측 영상시간 영역(약 120×40px) 텍스트 금지.
3. **로고 고정 위치**: 좌하단, 작게, 에메랄드 그린.
4. **성장 모티프**: 우상향 라인 / 좌표 상승 / 작은 별 등 'growth'를 은유하는 요소 1개 허용.
5. **clutter 금지**: 텍스트 2줄, 강조색 2개, 비주얼 1개 — 이 상한을 넘지 않는다.

## 🔁 4. 재사용 워크플로 (새 영상마다)

1. 배경: 딥네이비 + 미세 그라데이션/노이즈 (고정)
2. 비주얼 존: 영상 주제에 맞는 모티프 1개 교체
3. 1줄차/2줄차 카피만 교체 (2줄차는 항상 에메랄드 최강조)
4. 로고·언더라인·여백은 고정
5. 출력 1280×720 PNG + 2K 원본 보관
6. gpt-image-2로 한글 렌더링 → understand_images로 전수 검증

---

## 📐 5. 키 콘텐츠 적용 예 (제목 A)

- 1줄차(화이트): `AI는 3초면 답하는데`
- 2줄차(에메랄드, 최강조): `나는 왜 더 불안할까?`
- 비주얼 존: 별(골드 점 1개)을 올려다보는 고독한 실루엣 + 에메랄드 우상향 빛
- 로고: 좌하단 `인생포트폴리오`
