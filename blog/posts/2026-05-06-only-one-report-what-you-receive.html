한 줄 사명 한 권의 리포트 — 결제 후 받는 것 전부 보여드립니다

9,900원(해외 $8.99)을 결제하면 정확히 무엇을 받는가. ‘이렇게 생긴 PDF 한 장’이 아니라, 두 권의 문서가 어떻게 이어지는지를 구성 단위로 보여드리는 글입니다.

2026년 5월 6일 · 6분 읽기 · 상품 안내

결제하면 받는 것 — 한눈에

  • ① 인생포트폴리오 리포트 PDF · 11개 영역, 표지 포함 PDF 약 3~4페이지(A4 기준) · PC/모바일 마이페이지에서는 스크롤 한 페이지로 부담 없이 쭉 보실 수 있습니다
  • ② 맞춤 실행 프로그램 PDF · 3주 / 3개월 / 1년의 3중 시점
  • ③ 마이페이지 보관 · 회원 계정이 유지되는 동안 언제든 다시 열기·재다운로드(발급받은 언어 그대로)
  • ④ 검사 즉시 자동 발급 · 결제 → 76문항 → 결과 생성까지 평균 15분

① 리포트 PDF — 페이지별로 무엇이 들어 있는가

아래는 한 권의 리포트가 가지는 고정된 구성과 페이지 흐름입니다. 단어와 백분율은 응답에 따라 다르게 나오지만, 타이틀·순서·역할은 동일합니다.

표지 — 사명 한 줄 + 비전 한 줄

  • 이름과 제출일 (예: 〈···님의 인생포트폴리오 · 제출일 2026-05-06〉)
  • 한 줄 유형 정의 — 예: “사랑·자유·의미 추구 중심의 공감형 연결자”
  • 그 아래 한 문단의 ‘사명(Mission) 제안 문장’ 과 ‘비전(Vision) 제안 문장’

표지가 끝나면 “이 분이 어떤 자리에서 가장 자기다운가”가 한 화면에 보입니다.

실행 프로파일 카드 — 표 한 장

리포트 안에서 가장 많이 다시 펴게 되는 페이지입니다. 표 형식으로 6줄이 들어갑니다.

  • 실행 유형 (예: 공감·관계형 실행가)
  • 실행 스타일 (예: 감정 중심형)
  • 추진력 요인 (예: 사랑, 자유, 의미 추구)
  • 몰입 환경 (예: 자연 속 장소 / 몰입 시간과 휴식 시간을 분리한 하루)
  • 잘 맞는 활동 / 추천 도구·전략

성장 가이드맵 요약 — TOP3 강점 / TOP2 성장 포인트

  • TOP3 강점 — 한 문장으로 풀어 쓴 3개의 강점 (예: “흩어진 점들을 잇는 통찰이 자연스럽다” / “새로운 가능성을 가장 먼저 발견한다” / “자기 의미를 따라 끝까지 밀고 간다”) ※ 강점은 단어 나열이 아닌 문장형으로 표기됩니다
  • TOP2 성장 포인트 — 의도적으로 보완할 영역 2가지 (예: 갈등 상황에서의 단단함, 의견 단호한 표현)

리포트는 강점만 늘어놓지 않습니다. 성장 포인트를 같은 페이지에 둡니다. 이 한 장 때문에 1주차 행동의 방향이 잡힙니다.

진로·경력·교육 큐레이션 — 카드 3종

  • 추천 진로/직업 3가지
  • 추천 교육/훈련 3가지
  • 추천 확장 방향 3가지 (예: 경제 / 교육 / 종교 영역의 전문성 확장)

활용 예시 및 다음 단계 — ‘바로 실행할 첫 행동 3가지’

전체 리포트에서 가장 짧지만 가장 자주 인용되는 부분입니다. 결과를 행동으로 옮기는 첫 다리.

  • 1. (예) 주 1회 감사 메시지를 3명에게 보내기
  • 2. (예) 월 1회 1:1 미팅 정례화
  • 3. (예) 감정 일기 5분 시작하기

이 세 줄은 그대로 1주차 캘린더에 옮겨 적습니다.

5축 인사이트 — 응답 강도(%) 포함

리포트 본문은 5개의 축으로 사람을 입체적으로 봅니다. 각 축에는 ‘본질 한 줄 / 한 마디 자기 정의 / 키워드 4~5개’가 함께 옵니다.

  • 자기이해 (Self-Understanding) — 응답 강도 %
  • 자기표현 (Self-Expression) — 응답 강도 %
  • 자기설계 (Self-Design) — 응답 강도 %
  • 자기실행 (Self-Execution) — 응답 강도 %
  • 요약하자면 — 사명·비전 정교화 / 실행 전략 설계 / 진로·교육 설계 3줄

② 맞춤 실행 프로그램 PDF — 3주 / 3개월 / 1년

리포트가 “나는 누구인가”에 답한다면, 실행 프로그램은 “그래서 다음 주에 무엇을 하는가”에 답합니다. 같은 사람의 응답 데이터로 두 번 가공된 두 번째 PDF입니다.

분기 테마 한 문장

예: “이미 사람을 향하는 따뜻함이 충분합니다. 이번 분기는 그 따뜻함을 듣기 → 표현 → 관계 자산으로 구조화하는 시간입니다.”

분기 테마는 리포트의 5축에서 가장 약한 축을 ‘작은 루틴’으로 보강하는 방향으로 잡힙니다.

3주 실행 루틴 — 주차별로 카드 한 장씩

  • 1주차 — 마음을 듣는 루틴 만들기 (실행 안내 / 실행 방법 3가지 / 실행 효과 4가지)
  • 2주차 — 감사와 표현으로 관계 데우기
  • 3주차 — 관계를 자산으로 정리하기

각 주차 카드 안에는 정확한 실행 방법이 3개씩 들어 있습니다. 예 — 1주차: “매일 아침 ‘오늘 만날 사람 1명’의 컨디션을 떠올리고 한 줄 메모” “대화 중 상대의 핵심 감정 단어를 1개씩 받아 적기” “하루 끝, 마음이 머문 한 장면을 3줄로 기록.”

3개월 구조화 목표 — ‘실제로 자리 잡혀야 하는 3가지 결과’

  • 감사·공감 루틴 정착 (확인 기준: 주 3회 메시지 + 월 1회 깊은 대화)
  • 관계 자산 시각화 (확인 기준: 신뢰 네트워크 1장 작성 — 15명 내외)
  • 감정 표현 안전지대 확장 (확인 기준: 느낌 표현 일기 10건 이상 누적)

3개월은 ‘했다/안 했다’가 아니라 확인 기준까지 함께 옵니다.

1년 확장 방향 — 비전 한 문장 + 마일스톤 3개

예: “1년 후에는 ‘새로운 가능성을 발견하고 실제로 실현해 보여 주는 사람’으로 자리잡게 됩니다. 자유·사랑·의미가 한 작품 안에서 만나는 자기만의 비전 구조를 갖추게 됩니다.”

그리고 그 비전을 검증할 마일스톤 3가지가 함께 옵니다 (예: 검증된 가능성 3건 누적 / 의미형 결과물 1건 외부 공개 / “이 사람의 시선을 빌리면 새로운 길이 보인다”는 평이 자리잡음).

실행 모듈 카드 (TOP3) — 도구·템플릿까지

  • 강점 활용 모듈 — 예: 감사 메시지 루틴 (추천 도구: 메시지 템플릿·감정 일기 노트·관계 캘린더)
  • 보완 훈련 모듈 — 예: 감정 일기 & 표현 연습 (추천 도구: 느낌 단어 카드·1:1 대화 가이드)
  • 핵심 전략 모듈 — 예: 신뢰 네트워크 점검 세션 (추천 도구: 신뢰 네트워크 맵·분기 점검 시트)

③ 마이페이지 — 다시 보기 · 재다운로드

  • 로그인 후 마이페이지에서 리포트와 실행 프로그램을 언제든 다시 열 수 있습니다.
  • 발급 시점에 선택하신 언어로 보관되며, PC·모바일 어디서든 동일하게 열립니다. PDF 다운로드도 그대로 가능합니다.
  • 회원 계정이 유지되는 동안 보관되며, 본인이 마이페이지에서 개별 삭제하거나 회원 탈퇴하시기 전까지 다시 열 수 있습니다(자세한 사항은 이용약관 제12조, 개인정보처리방침 제3조 참고).

왜 ‘리포트 + 실행 프로그램’ 두 권인가

리포트만 받으면 가장 흔한 결말은 “좋은 글” 한 편을 읽고 잊는 것입니다. 실행 프로그램이 같이 와야 1주차에 캘린더가 바뀌고, 3개월이면 ‘확인 기준’이 따라옵니다. 이 두 권이 한 세트라는 점이 본 상품의 핵심입니다.

리포트는 “나는 누구인가”에 답하고, 실행 프로그램은 “그래서 다음 주에 무엇을 하는가”에 답합니다. 두 줄이 만나는 자리에서 변화가 시작됩니다.

자주 묻는 질문

  • Q. 결제 후 얼마나 걸리나요? — 평균 15분(76문항 응답 시간 포함). 리포트와 실행 프로그램은 검사 완료 즉시 자동 발급됩니다.
  • Q. 본인이 언제든 다시 받을 수 있나요? — 회원 계정이 유지되는 동안 마이페이지에서 발급받은 언어 그대로 재다운로드하실 수 있습니다.
  • Q. 환불은 어떻게 되나요? — 검사 시작 전 100% 환불. 시작 후에는 발급된 결과물 특성상 환불이 제한됩니다 (자세한 내용은 개인정보 / 환불 정책).
  • Q. 회사·기관 단위로도 가능한가요? — 가능합니다. faise@lifeportfolio.co.kr로 인원·일정 알려주세요.
